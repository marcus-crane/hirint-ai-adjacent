## 职位描述

1. 负责后端系统的开发和维护，包括 API 接口、数据库、缓存等方面的设计和实现；

2. 参与项目的需求讨论和架构设计，为项目提供技术支持；

3. 使用 Java/Python/Go等技术语言进行开发，保证代码质量和性能；

4. 支持和优化系统的性能和稳定性，处理高并发的请求；

5. 参与项目的技术选型、架构设计和实施；

6. 与算法、产品、运维团队紧密合作，共同打造高质量产品。

## 任职要求

1. 本科及以上学历，计算机相关专业背景

2. 5年以上后端开发经验，2年以上分布式后端系统开发经验。

3. 精通 Python/Go/Java（至少一种），有大规模系统实践。

4. 熟悉数据库设计和优化，有数据库调优经验。

5. 熟悉大模型服务平台，有推理服务/训练服务/模型管理等平台开发经验。

6. 自驱力强，优秀的问题解决能力和团队协作精神。

【加分项】

1. 有大模型服务平台实际开发经验，熟悉大模型原理和应用
