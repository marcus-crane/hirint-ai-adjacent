## 职位描述

1. 产品战略与规划

- 深入调研行业场景与用户痛点，识别 Agent 技术可落地的高价值业务场景

- 制定 Agent 产品路线图，定义短期迭代与长期突破方向，确保产品愿景与公司战略一致

- 量化产品目标与核心指标（如任务成功率、响应时长、用户满意度等），建立数据驱动的评估体系

2. 产品设计与研发

- 主导 Agent 产品的功能设计，包括任务理解、规划、执行、记忆、多轮对话等核心模块

- 设计并优化 RAG 知识库框架，定义数据采集、清洗、向量化与检索逻辑

- 规划工具调用（Function Calling）架构，设计 API 集成、参数映射与结果处理机制

- 协同算法团队确定提示工程、模型微调与强化学习策略，提升 Agent 的智能水平

3. 跨部门协作与落地

- 与研发团队共同制定技术实现方案，把控开发质量和进度

- 与设计团队协作，优化 UI/UX 交互流程，提升用户体验

- 配合业务方梳理流程痛点，定制 Agent 解决方案，推动落地验证与规模复制

- 协调运营与市场团队，制定产品推广策略与商业化模式

4. 运营优化与迭代

- 建立 Agent 产品监控体系，追踪关键指标并定位问题

- 设计 A/B 测试方案，科学评估不同策略与版本的效果差异

- 收集用户反馈并快速响应，持续迭代产品功能与体验

- 优化资源利用效率，控制算力成本，提升产品可持续性

5. 行业洞察与创新

- 持续跟踪 Agent 与大模型技术前沿，挖掘创新应用场景

- 参与行业生态建设，与开源社区、技术峰会等保持互动，确保产品技术领先性

## 任职要求

必备技能

- 教育背景：本科及以上学历，计算机科学、人工智能、产品设计等相关专业；985/211 或知名院校优先 9

- 产品经验：3 年以上产品管理经验，有 AI 产品或智能客服、自动化平台相关经验者优先 7

- 技术理解：深入理解 LLM 工作原理，熟悉提示工程、模型微调、RAG 检索、多模态融合等技术；不必精通代码，但需能与算法团队高效沟通 3 8

- 业务分析：擅长流程建模与拆解，能将复杂业务逻辑转化为 Agent 可执行的任务规划 5

- 数据驱动：熟练运用数据分析工具，能通过指标设计、埋点规划与 A/B 测试指导产品决策 2

优先技能

- 具备 Agent 或工作流相关产品（如 Zapier、Make、Coze 等）实践经验

- 熟悉主流 LLM（GPT、Claude、Gemini 等）特性与限制，有本地部署或私有化模型调优经验

- 具备多模态产品（文本、语音、图像处理）设计经验

- 了解低代码 / 无代码平台，能降低 Agent 开发与配置门槛
