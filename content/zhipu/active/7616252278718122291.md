## 职位描述

1. 负责基座模型在Agentic Tool Using / General Agent 场景的效果优化；

2. 负责基座模型的工具调用能力的统一评测框架与优化；

3. 跟进 LLM Tool Using前沿技术，持续优化基座模型调用工具解决复杂任务的能力；

## 任职要求

1. 本科及以上学历，计算机、软件工程、人工智能等相关专业；

2. 具有数据构建和模型SFT经验（重要），具备优秀的代码能力和基础算法功底，有较为丰富的工程经验（重要）；

3. 具有优秀的逻辑思维能力，关注工具调用前沿进展，对提升模型工具调用能力充满热情；

4. 具有良好的团队合作精神，善于分析和解决问题，有较强的沟通能力和学习能力；

加分项：

1. 有Agentic Tool Using 项目经验者优先；

2. 在大模型 / 工具调用 领域有研究论文者优先；

3. 熟悉大模型预训练 / SFT / RL 者优先
