## 职位描述

1. 负责 MiniMax Agent / AI App 相关产品的服务端研发；

2. 参与 Agent 系统架构、任务调度、上下文管理、工具调用、数据链路等能力建设；

3. 与算法、产品、前端、客户端团队协作，推动大模型能力稳定、高效地服务用户；

4. 持续提升系统性能、稳定性、可扩展性和研发效率。

## 任职要求

1. 具备扎实的计算机基础和服务端工程能力，理解高并发、分布式、存储、缓存等常见问题；

2. 有复杂系统设计能力，关注技术方案的长期质量；

3. 对 LLM / Agent / AI App 有兴趣，有独立开发、开源项目、创业项目或 AI 工具实践者优先。
