## 职位描述

1. 负责 MiniMax Agent 相关 Web / 桌面端产品的前端研发；

2. 参与 AI Agent 交互体验、工作流界面、多模态内容展示等核心模块建设；

3. 与产品、设计、服务端、算法团队协作，推动复杂 AI 能力产品化；

4. 持续优化前端性能、工程架构和研发效率。

## 任职要求

1. 熟悉前端技术栈，有扎实的前端基础；

2. 具备良好的工程能力，关注代码质量、组件设计、性能和用户体验；

3. 学习能力强，能快速理解业务和技术方案，主动推进落地。

4. 对 AI Agent、AI 工具、AI 应用有兴趣，有独立开发、开源项目、创业项目或 AI 工具实践者优先。
