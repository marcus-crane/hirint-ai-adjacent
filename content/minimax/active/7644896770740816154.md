## 职位描述

岗位背景

我们在为大模型做「Web 应用开发」方向的后训练(SFT / RL / Agent 评测),其中 Web 3D 是单独拎出来的一条线 —— 涵盖着色器语言、图形API、上层工具库、资产,这一类任务对模型既考视觉审美又考数学和图形学,造数据和搭评测都比常规前端难一个量级。

模型能力的上限,本质上由训练数据和评测体系的质量决定,而 Web 3D 这种"代码能跑只是开始、渲染对不对、性能好不好才是关键"的领域,造数据和搭评测的工程难度并不比训模型低。

我们需要一位真正懂 Web 3D 与图形的工程师,把数据合成、渲染、评测这条链路做扎实。

- 着色语言:GLSL ES 3.0、WGSL、TSL(Three.js Shader Language)

- 图形API:WebGL 1/2、WebGPU、Canvas2D

- 工具库:Three.js、React Three Fiber (R3F)、Babylon.js、PlayCanvas

- 资产:GLTF / GLB / Draco / KTX2/Basis / USDZ / 3D-Tiles / point cloud / Gaussian Splat 及程序化生成

你将负责

1. 训练数据构造:为 Web 3D 方向设计任务维度与配方,产出可用于 SFT / RL 的高质量数据,包括纯 Shader 题、Three.js 场景题、ShaderToy 风格题、3D 交互组件题、动画 / 物理仿真题、多轮 Agent 任务等;

2. 自动化 / Agent / LLM 评测体系:设计与维护 Web 3D 代码 / 渲染产物的自动化评测 —— 包括但不限于 Shader 编译验证、Three.js / R3F 场景启动验证、画面截帧 + 视觉回归、SSIM / PSNR / LPIPS 指标、帧率与 GPU 占用 sanity、Agent 端到端评测 —— 让"模型生成的 3D 场景或 Shader 是不是真的渲对、跑得动"可被自动打分;

3. 渲染沙盒和评测集群:建设并维护 高并发、高可用、多样化 的 Web 3D 渲染沙盒(headless Chrome + GPU 加速、SwiftShader / ANGLE / 真实 GPU 渲染节点),覆盖不同浏览器内核 / 显卡驱动 / WebGPU 实现版本,保证评测与训练场景的真实性与稳定性;扩展 Agent 在 3D 场景里的能力边界 —— 抓帧、调材质、调摄像机、读 scene graph。

## 任职要求

硬性要求

- 认可并深度拥抱 AI:这是岗位的前提 —— 本岗位的产出对象本身就是 AI,对 AI 抱观望或抵触态度的同学请绕道。要求把 AI 编程工具当成日常生产力的一部分,熟练使用 Claude Code / Cursor / Codex 等至少一款,持续关注前沿 Agent 框架与 harness(Claude Code、mini-swe-agent、OpenHands、SWE-agent 等),知道它们各自的设计取舍;了解 skill / plugin / sub-agent 等机制的原理与作用,知道在什么场景下该用哪种;

- 真正理解大模型与 Agent 的工作原理(这一条甚至比"会写 Three.js"更重要):能清晰区分 SFT 数据 / RL reward / 评测集的差别,理解 prompt 设计 / tool calling / 多轮对话 / context engineering 这些概念到底在解决什么问题。不要求训过模型,但概念上要扎实 —— 你来这里不是写炫酷场景的,是给模型造能力的,概念错了方向就错了;

- 1 年以上 Web 3D / 图形相关工程经验,Three.js / Babylon.js / R3F 至少一栈真的写过,能用 GLSL 或 WGSL 写 Shader;

- 有图形学基础 —— 知道渲染管线、坐标系、矩阵变换、PBR / 光照模型、UV / 法线 / 切线、深度 / Stencil / Blend 这些概念到底在解决什么问题;

- TypeScript 熟练,能读懂 Three.js / R3F 的源码;Python 必备(造数据 / 评测代码很多 Python);

- 熟悉 Docker / Linux,能在 headless 浏览器里把 WebGL / WebGPU 渲染跑起来;

- 会用 Playwright / Puppeteer 写浏览器自动化,能截帧、能注入脚本读 scene graph。

- 在 Agent / MCP 生态里有过 深度参与(魔改过 harness、写过 MCP server、给开源 Agent 项目提过 PR 等);

- ShaderToy / 程序化生成 / Ray Marching / SDF / 流体 / 粒子系统 实战经验;

- 3D 模型管线(gltf / glb / Draco / Meshopt / Blender 脚本)、骨骼动画、IK 经验;

- WebGPU / WGSL 实战、Compute Shader、GPU 性能 profiling 经验;

- WebXR / VR / AR、3D 物理(Cannon / Rapier / Ammo / Havok)经验;

- Rust 图形(wgpu / Bevy)或原生图形栈(Vulkan / Metal / DirectX)经验;

- 了解 K8s / QEMU / 虚拟化 等基础设施技术,有 GPU 渲染集群 / 远程渲染节点 搭建经验;

- Go 工程经验。
