## 职位描述

1. 作为业务的财务搭档，深入业务、参与经营会议，输出财务视角的判断与建议

2. 负责所辖业务线月度 P&L 编制、滚动预测与差异分析，驱动业务完成预算目标

3. 搭建业务经济模型，持续监控关键指标健康度

4. 主导业务预算编制与全过程管控，识别资源浪费与降本机会

5. 对业务新项目、新策略、新产品进行事前财务评估（投入产出 / ROI / NPV / 盈亏平衡），出具立项财务意见

6. 与业务共建关键业务规则（定价、返点、考核口径、激励机制等），确保业务动作与财务结果对齐

7. 向管理层定期输出经营分析报告，揭示风险、捕捉机会、推动改善

## 任职要求

1. 8年以上财务相关工作经验，其中至少 3 年财务BP或业务财务经验

2. 熟悉一个业务方向的财务逻辑：toB/SaaS 行业 / toC 互联网 / 职能中后台 / 科技/IDC/云计算，有完整业务P&L 闭环的财务支持经验

3. 拥有战略思维、快速学习能力、对AI行业有热忱，能运用 Python 或 SQL 进行数据提取与分析者优先

4. 业务好奇心：愿意下到业务一线，理解产品、用户、竞争，而非只对数字

5. 结构化思考：面对复杂问题能快速拆解、建模、给出可落地的结论

6. 沟通影响力：敢于表达专业判断，也能换位推动共识，结果导向
