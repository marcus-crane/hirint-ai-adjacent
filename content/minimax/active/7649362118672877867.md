## 职位描述

1.平台前端建设：负责计算平台前端开发,覆盖模型训练、评测、调度、可视化、监控等多个核心业务场景,持续打磨平台性能、稳定性与易用性;

2.跨团队协作：与算法、后端团队深度共建,从用户视角出发,持续提升计算平台的使用效率与研发体验;

3.AI Coding 工程化探索：深度参与 AI Coding 工程化链路(Skill / Agent / Plugin / Workflow)的研究与落地,将 AI辅助开发能力真正融入日常迭代,推动团队研发范式升级。

## 任职要求

1.基础能力：1–3 年前端开发经验,精通 JavaScript / TypeScript,具备良好的代码规范意识与工程素养;

2.框架与生态：熟练掌握 React / Vue,深入理解其核心原理与生态体系(状态管理 Redux Toolkit / Pinia、路由方案 Next.js App Router、组件化设计、Hooks 心智模型);有 Ant Design + Tailwind CSS 工程实践经验者优先;

3.工程化能力：理解现代前端工程化思想,熟悉 Webpack / Vite 等构建工具,了解 CI/CD 持续集成流程(GitLabPipeline / 自动化部署 / Tag-based Release);

4.AI 编程实践：熟练使用 Claude Code / Codex 等 AI 编程工具,理解 Skill / Plugin / Agent / Workflow等机制的原理与适用场景,能让 AI 跑通完整研发流程,而非仅停留在"让 AI 写代码"层面,在实际项目中带来可量化的效率提升;

5.软素质:沟通协作能力强,主动性高,具备持续学习意愿与强责任心,能在快速变化的业务中保持稳定输出。

加分项

1.接触过 ML 训练 / 评测 / 实验追踪类产品（MLflow / wandb等）

2.关注 AI 辅助开发 前沿（subagent / 工具调用 / 上下文管理 / MCP），有自己的实践沉淀
