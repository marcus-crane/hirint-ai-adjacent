## 职位描述

负责 MiniMax 开放平台核心前端产品的设计与实现，具体工作包括：

- 负责 API 控制台、开发者文档站、Playground、用量与账单等核心页面的开发与体验优化

- 实现流式对话、模型调试、参数调优等 AI 场景的前端交互，让开发者在浏览器里就能快速验证模型能力

- 与服务端协作落地计费、账号、权限等商业化能力的前端呈现，确保复杂业务逻辑的交付质量

- 参与前端基础设施建设，包括组件库、工程规范、性能监控，提升团队整体研发效率

## 任职要求

- 2-5 年前端开发经验，熟悉 React/Vue + TypeScript，有完整项目交付经验

- 关注用户体验，能主动发现并推动体验问题的解决，而不只是还原设计稿

- 能独立负责模块，从需求理解到方案设计到上线交付端到端闭环

- 了解 AI / LLM 基本概念，对 API 调用、流式响应（SSE）等场景有基本认知

加分项

- 有开发者工具、ToB 控制台或文档站建设经验

- 活跃于开发者社区，有开源贡献或技术写作习惯
