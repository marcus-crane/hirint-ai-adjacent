## 职位描述

团队介绍

我们正在构建企业级 AI Agent 评测与管理平台，覆盖模型管理、Agent 调试、评测任务调度、Pipeline 编排、沙盒执行等完整链路。平台服务于内部多个 AI 团队，日均承载大量 LLM 调用与 Agent 评测任务，直接影响模型评测、Agent 研发与迭代效率。

在这里，你将深度参与 AI Agent 前沿技术的工程化落地，接触模型训练、Agent 研发、数据标注、评测 Benchmark 等多个方向。平台技术栈现代、架构演进快，既有高并发调度、LLM 网关、容器沙盒等后端核心系统，也有可视化 DAG 编排、评测数据管理等产品化场景，具备充分的系统设计与技术发挥空间。

工作职责

1. 参与 AI Agent 评测平台核心功能开发，包括评测任务调度、Pipeline 编排引擎、数据集管理等模块

2.设计并实现 LLM 网关层，负责模型路由、Token 计量、多供应商模型接入与调用治理

3.开发和维护 Agent 沙盒执行环境，包括容器调度、工具执行、MCP 服务集成等能力

4.参与前端可视化编排界面开发，支持基于 DAG 的评测流水线配置与管理

5.持续优化平台稳定性与性能，支撑高并发评测任务的调度、执行与监控

## 任职要求

必须具备

1.一年以上后端开发经验，熟练掌握 Go 语言

2.熟悉 Web 框架开发，理解 RPC / HTTP 接口设计与代码生成

3.熟悉 MySQL、Redis，具备数据库设计与性能优化经验

4.熟悉 Docker / Kubernetes 等容器化部署技术

5.具备良好的系统设计能力，能够独立负责模块方案设计与落地

加分项

1.有 LLM / Agent 应用开发经验，了解 prompt engineering、function calling、MCP 协议

2.熟悉 Vue 3 或 React，能够参与前后端协同开发

3.有评测系统、Benchmark 平台或任务调度系统建设经验

4.有 Python 开发经验，了解数据处理流水线

5.熟悉 CI/CD 流水线搭建，有 DevOps 实践经验
