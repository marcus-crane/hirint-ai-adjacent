## 职位描述

1. 构建超大规模（等效100T token/day）吞吐的MaaS平台，解决相应规模下的技术问题。

2. 主导标准 API/SDK 体系建设与高并发服务网关设计，沉淀多租户隔离、鉴权、计费与智能流量控制等系统级最佳实践。

3. 从MaaS视角统筹模型全生命周期管理，结合云原生架构，保障SLA，最大化模型吞吐。

4. 关注模型效果，协同算法团队，基于真实业务表现提升模型效果。

5. 建立团队技术标准与后端工程品位，完善全链路可观测性与容灾体系，提升整体复杂问题解决能力与线上稳定性。

## 任职要求

1. 深入理解分布式系统原理，具备复杂云服务、微服务架构或大规模中台系统从 0 到 1 的设计与演进经验；

2. 优秀的编程能力与代码品位，熟练掌握 Go / Python / C++ 至少一门语言，具备扎实的系统性能调优功底；

3. 熟悉云原生与容器化生态；

4. 具备极强的业务抽象与产品化能力，能够将底层算法能力转化为标准化、极致开发者体验的服务接口；

5. 具备严苛的系统可靠性（SRE）意识，精通高并发场景下的容灾、限流、降级等工程体系；

6. 具备优秀的沟通协作能力与团队管理经验，能够高效推动跨团队（算法、Infra、业务）复杂项目落地，对技术有长期追求。
