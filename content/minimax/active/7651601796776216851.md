## 职位描述

- 负责开放平台前端团队的整体管理与梯队建设，制定技术方向，推动工程规范落地，培养核心骨干，打造高效、自驱的前端团队。

- 主导开放平台面向开发者的全链路前端架构设计与技术演进，覆盖 API 控制台、开发者文档站、Playground、SDK 示例、用量与账单可视化等核心产品。

- 承接平台商业化、计费、账号、网关等后端能力的前端落地，与服务端、产品、设计团队紧密协作，确保复杂业务逻辑在前端的高质量交付。

- 深入开发者使用场景，从开发者视角定义和优化产品体验，建立体验度量体系（如接入转化率、文档跳出率、首次调用成功率），用数据驱动体验持续改进。

- 关注开发者社区生态，理解主流 AI 开发者工具链和开源社区动态，将社区最佳实践融入平台产品设计，提升平台在开发者群体中的口碑和竞争力。

## 任职要求

- 5 年以上前端开发经验，2 年以上前端团队管理经验，有完整的团队搭建、方向制定和梯队培养经历。

- 主导过 ToB 平台或开发者平台的前端架构设计，理解控制台、文档站、Playground 等开发者工具产品的设计范式和体验标准。

- 具备全局业务视角，能从商业目标和用户价值出发拆解前端技术方案，而非仅关注实现层面。能独立与产品、服务端、设计团队对话并推进决策。

- 对用户体验有深度追求，有建立体验度量和数据驱动优化的实际经验，能定义"什么是好的开发者体验"并推动落地。

- 熟悉现代前端技术栈（React/Vue/Next.js），熟悉前端工程化、性能优化、可观测性建设。

- 了解 AI / LLM 应用开发生态，理解 API 调用、流式响应、Token 计费等开发者核心使用场景。

加分项

- 有 AI 开发者平台、云厂商控制台或 API 平台的前端建设经验，深度理解开发者用户的需求和决策链路。

- 活跃于开发者社区，有技术博客、开源项目贡献或社区运营经验，能从社区视角审视产品体验。

- 有 Developer Experience (DX) 体系建设经验，主导过 SDK、CLI、文档系统或 Playground 的从 0 到 1 建设。
