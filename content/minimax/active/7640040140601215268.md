## 职位描述

岗位职责

1. 安全标准体系建设：建立和完善跨产品线统一的内容安全分级分类标准，定义安全边界、风险等级和处置规范，实现策略的标准化和可复用

2. 安全评估框架设计：设计和维护产品安全评估框架，包括新功能/新市场上线的安全评估准入 checklist，确保安全前置介入产品迭代流程

3. 策略模块化推进：推动安全策略的模块化建设，按"基础规则层+ 地区适配层 + 场景叠加层"三层架构，实现新市场/新场景的策略快速配置

4. AIGC 安全标准制定：针对 AI 生成内容建立专项安全标准，包括 AIGC 内容安全红线定义、AI 生成内容标识规范、Prompt injection 防护评估标准等

## 任职要求

任职要求

1. 本科及以上学历，3-5 年互联网内容安全标准相关经验

2. 具备安全策略设计和标准制定能力，有从 0 到 1 搭建安全标准/评估体系的经验

3. 熟悉国内外主要内容安全法律法规，能够将法规要求翻译为可落地的安全策略

4. 了解 AI/大模型/AIGC 内容安全的技术原理和风险面，熟悉 Prompt injection、Jailbreak 等安全风险

加分项

- 有海外市场（欧洲/东南亚/北美）的内容安全合规经验

- 有 Agent/LLM 应用安全评估经验

- 有多语言/多国家安全策略适配经验
