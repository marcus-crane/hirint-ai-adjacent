## 职位描述

1. 负责 MiniMax Agent 相关产品的客户端研发，包括移动端、桌面端等方向；

2. 参与 AI Native 产品体验设计与工程实现，打造流畅、稳定、高质量的用户体验；

3. 与产品、设计、服务端、算法团队协作，推动 Agent 能力在客户端落地；

4. 关注性能、稳定性、交互体验和工程效率，持续优化客户端架构。

## 任职要求

1. 具备扎实的代码能力和客户端工程经验，理解性能优化、稳定性治理等基础问题；

2. 对 AI Agent / AIGC 产品感兴趣，愿意主动体验和研究新产品；

3. 自驱力强，沟通顺畅，能在快速迭代环境中推进问题解决。

4. 有独立开发、开源项目、创业项目或 AI 工具实践者优先；
