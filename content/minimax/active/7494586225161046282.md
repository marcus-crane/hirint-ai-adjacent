## 职位描述

我们是 MiniMax 产品研发团队。在这里，不只是做 Feature，而是在构建大模型与世界连接的新界面（Interface）、打造 SOTA 模型。

我们正在做的

- 多模态 Agent 矩阵： 打造 MiniMax Agent、海螺视频、星野/Talkie 等现象级产品，探索 Agent 在多场景下的能力极限。

- 深度参与 LLM Post-Training：我们不仅使用模型，更与算法团队并肩，通过优化平台与训练链路、参与BenchMark、数据、Reward 设计，共同打造 SOTA 模型。

我们的基因

- AI First： 我们坚持用 AI 重新定义开发范式与组织协同，拒绝平庸的重复。

- 最佳体验： 致力于通过多模态技术，为全球用户带来最具效率与情感价值的 AI 体验。

如果你对最先进的模型、最新的 Agent 范式充满好奇，欢迎加入我们，一起用 AI First 改变世界！

工作内容

- 负责各AI Native / AI Agent产品的服务端开发

- 负责 LLM Post-Traning 平台、环境开发

## 任职要求

基础要求

- 出色的代码能力：语言不限，Golang/Java/Python/Rust/C++等均可

- 出色的工程能力：理解所在技术栈底层原理，业务层、Infra 层开发经验均可

- 复杂系统设计能力、优秀的技术品味

- 高效的沟通能力、强大的自驱力

加分项

- 对主流开源项目（分布式存储/容器化/服务治理 ...）有深入了解，并有贡献

- 对 AI App / Agent 有浓厚兴趣，有个人产品或开源项目

- 熟悉 LLM 相关技术栈（如LangChain、Dify、MCP、vllm、ollama, etc. ），关注社区技术进展并有相应实践

- 熟悉 LLM 模型原理，有模型部署、推理优化、PostTrain实践经验

- ACM/ICPC、NOI/IOI、Top Coder、Kaggle 等比赛获奖
