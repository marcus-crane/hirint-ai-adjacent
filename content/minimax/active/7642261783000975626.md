## 职位描述

1. 负责 MiniMax相关业务方向的整体规划与推进，覆盖 ToC 和 ToB 场景；

2. 结合大模型、Agent、工具调用、多模态等技术能力，判断产品机会和业务落地方向；

3. 拆解用户需求和业务目标，定义产品路线、关键功能、核心指标和迭代节奏；

4. 与研发、算法、产品、设计、运营、市场等团队协作，推动AI产品从 0 到 1、从 1 到 N 落地；

5. 关注用户反馈、数据表现和行业趋势，持续优化产品体验、商业模式和业务增长；

6. 搭建和管理跨职能团队，推动团队高效协作，对业务结果和产品质量负责。

## 任职要求

1. 有 AI / Agent / AIGC / SaaS / 企业服务 / C端产品等相关经验；

2. 既懂技术又懂业务，能理解大模型、Agent 工作流、工具调用、RAG、多轮对话等基础概念，并能转化为产品和业务方案；

3. 有较强的业务判断能力，能判断不同场景的用户价值、商业价值和落地路径；

4. 有从 0 到 1 产品或业务搭建经验，能独立拆目标、定策略、推资源、拿结果；

5. 有跨团队协作和团队管理经验，能推动研发、算法、产品、运营等多角色高效配合；

6. 对 AI 产品有强兴趣和高敏感度，持续关注海外及国内AI产品；

7. 有技术背景、研发管理经验、AI创业经验、ToB商业化经验或成功 C端产品经验者优先。
