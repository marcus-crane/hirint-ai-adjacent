## 职位描述

为什么加入我们

1. 用 AI 重新定义 IT ：不是传统的工单驱动模式——你要用 AI Agent、自动化工作流替代一切重复劳动，把 IT 做成"无感服务"

2. 从零搭建安全体系：公司处于安全建设的起点，你将有机会主导设计整套安全规范和基础设施，而不是在别人的框架里修修补补

3. 离 AGI 最近的 IT实践：公司本身就在做最前沿的多模态大模型，你能第一时间拿到最新模型能力，探索AI 在 IT 场景的极限

4. 扁平文化，成长快：团队精干，没有层层汇报，你的方案能快速落地、快速验证

你将做什么

1. 安全体系建设（从0到1）

- 主导建立公司信息安全规范、制度和技术方案，覆盖网络安全、数据安全、终端安全、权限管理等领域

- 设计和维护沙盒环境安全方案，特别是模型安全相关的隔离与防护机制

- 推动安全意识培训和安全审计流程的落地

2. AI 驱动的 IT 自动化

- 用 AI 工具（Claude Code、MCP Server、Agent 工作流等）重构 IT运维流程，大幅减少人工介入

- 将账号开通、权限申请、设备管理、故障排查等高频 IT 事务自动化，让 AI成为一线响应者

- 持续评估和引入 AI 工具，提升整个 IT 团队的工具链效能

3. 数据与资产自动化管理

- 搭建 IT 资产、账号、许可证等数据的自动化管理体系，替代手工台账和人肉巡检

-打通各系统数据源（飞书、OA、LDAP、云平台等），实现数据自动采集、同步和异常告警

- 用数据驱动 IT 决策，输出可视化报表和成本分析

4. 企业 IT 基础设施运营

- 维护和优化网络、身份认证（SSO/LDAP）、办公协同、终端管理等基础设施

- 以 AI-first 思维重新设计 IT 服务流程，而非简单搬运传统方案

## 任职要求

1. 1-5 年 IT 工程/运维/安全相关经验，计算机相关专业本科及以上

2. 扎实的工程能力，熟悉至少一门编程语言（Python / Go / Shell /TypeScript），能独立开发自动化工具和脚本

3. 日常深度使用 AI 编程工具（Claude Code / Cursor / Copilot），已经用 AI改变了自己的工作方式，而不是"试过"

4. 熟悉企业 IT 基础技术栈中的至少两项：网络（防火墙/VPN/DNS）、身份认证（LDAP/SSO/OAuth）、终端管理（MDM/EDR）、云平台（AWS/GCP/私有云）

5. 对信息安全有基本认知，了解常见安全框架（ISO 27001 / 等保）或有安全体系建设经验者优先

6. 强自驱力，能把"IT 自动化"当产品来做——主动发现问题、设计方案、推动落地

加分项

1. 有 AI Agent / MCP / LLM 应用开发经验，做过 AI 自动化的实际项目

2. 有企业安全体系从0到1搭建的经验

3. 熟悉 Docker / K8s / CI/CD 等云原生技术栈

4. 在技术社区有 AI Native 工作方式的分享或开源项目

5. 对"用 AI 替代一切不值得人做的事"有执念
