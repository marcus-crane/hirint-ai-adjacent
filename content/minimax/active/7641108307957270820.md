## 职位描述

关于我们

我们是 MiniMax 产品研发团队 Talkie / 星野 服务端方向。Talkie（海外）与星野（国内）是面向全球数千万用户的 AI

角色陪伴产品，让每个人都能与心目中的角色对话、共创内容、建立情感连接。

我们正在做的

- 全球化 AI 陪伴产品：Talkie 服务百余个国家与地区的海外用户，星野深耕国内 Z 世代，两端共同构成业内规模与活跃度领先的 AI角色社区。

- 下一代多模态对话体验：在文本对话基础上，持续打磨实时语音、AI 视频通话、图像生成、角色直播等多模态形态，重新定义"人与AI 的相处方式"。

- 海量 UGC 角色生态：支撑用户自由创建、训练、分发 AI 角色，从角色编辑器、记忆系统、Persona引擎到内容审核与分发，构建完整的角色经济。

- 深度参与 LLM Post-Training：与算法团队并肩，围绕角色扮演、长对话记忆、情感表达等场景共建数据、Reward 与Benchmark，让模型为产品体验服务。

我们的基因

- AI First：用 AI 重新定义开发范式与协同方式，拒绝平庸的重复。

- 极致体验：在数千万 DAU、超高并发与跨地域延迟约束下，依然为全球用户提供最具效率与情感价值的 AI 体验。

- 小步快跑：快速假设、快速验证、快速上线，让每一次迭代都直接被海量用户感知到。

工作内容

- 负责 Talkie / 星野 服务端核心系统的设计与开发，包括但不限于：

- 对话与 Agent 引擎：会话编排、上下文管理、角色记忆、工具调用、多模态消息流转

- 角色与内容平台：角色创建、Persona/Prompt 工程化管理、UGC 内容生命周期、审核与分发

- 实时通信与多模态：长连接推送、语音/视频通话信令与媒体链路、流式生成与端到端延迟优化

- 高并发基础服务：海量用户的存储、缓存、消息队列、限流与容灾架构

- 与算法、产品、客户端紧密协作，把最前沿的模型能力（角色扮演、长记忆、Agent、多模态生成）以稳定、高性价比的方式落地到产

品中。

- 参与 LLM Post-Training 链路建设：从产品反馈数据回流、Reward 信号设计到效果评估，与算法共同打磨更适合陪伴场景的 SOTA模型。

## 任职要求

基础要求

- 出色的代码能力，语言不限：Golang / Java / Python / Rust / C++ 等均可，团队主力栈为 Golang。

- 出色的工程能力：理解所在技术栈底层原理，业务层、Infra 层开发经验均可。

- 具备复杂系统设计能力，能够在高并发、海量数据、跨地域部署场景下做出合理的架构权衡。

- 优秀的技术品味、高效的沟通能力与强大的自驱力。

加分项

- 有大规模 C 端产品（IM / 社交 / 内容社区 / 直播 / 在线游戏等）服务端经验，熟悉长连接、实时通信、消息系统、Feed/推荐链路。

- 有海外业务经验，了解多地域部署、跨地域数据一致性、合规与本地化挑战。

- 对 AI App / Agent 有浓厚兴趣，有个人产品、Side Project 或开源贡献。

- 熟悉 LLM 相关技术栈（LangChain / Dify / MCP / vLLM / Ollama 等），关注社区进展并有实际落地经验。

- 熟悉 LLM 模型原理，有模型部署、推理优化或 Post-Training 实践经验。

- 对主流开源项目（分布式存储 / 容器化 / 服务治理等）有深入了解并有贡献。

- ACM/ICPC、NOI/IOI、Top Coder、Kaggle 等比赛获奖。
