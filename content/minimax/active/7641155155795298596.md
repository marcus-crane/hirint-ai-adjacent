## 职位描述

1. 负责 MiniMax Code 相关产品的质量保障和测试开发工作（PC Web APP等）；

2. 设计研发测试AI/Agent类产品的TestAgent，提高交付质量和效率。

3. 建设稳定性、性能、兼容性、回归测试等质量体系；

4. 与研发、产品、算法团队协作，推动问题发现、定位和闭环。

5.时刻关注AI、LLM领域前沿技术和开源社区，确保测试场景/方案与业界最新动态一致

## 任职要求

1.本科及以上学历，计算机相关专业，2年以上互联网测试开发经验；

2. 熟悉测试流程和测试方法，有 PC/ Web / App / 服务端等相关测试经验；

3. 具备一定代码能力，熟悉 Python / Java / Go 等任一语言；

4. 有自动化测试、接口测试、CI/CD、性能测试经验者优先；

5. 责任心强，关注产品质量，对 AI 产品有兴趣

6.热衷于开源技术，熟悉常见AI/Agent框架，有AI类产品、Agent测试经验者优先
