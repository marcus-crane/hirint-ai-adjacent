## 职位描述

1. 负责 MiniMax Agent 相关产品的质量保障和测试开发工作；

2. 参与 Web、客户端、服务端等多端产品的测试方案设计与自动化建设；

3. 建设稳定性、性能、兼容性、回归测试等质量体系；

1. 熟悉测试流程和测试方法，有 Web / App / 服务端测试经验；

2. 具备一定代码能力，熟悉 Python / Java / JavaScript 等任一语言；

3. 有自动化测试、接口测试、CI/CD、性能测试经验者优先；

4. 责任心强，关注产品质量，对 AI 产品有兴趣。
