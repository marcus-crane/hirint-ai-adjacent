## 职位描述

职位亮点

1.你将参与 AI 基础设施核心平台的建设，把复杂的 GPU 集群、节点与资产管理能力，转化为高效、易用的产品体验。

2.这里不只是常规的页面开发：你会面对大规模数据交互、可视化、网络拓扑、在线代码编辑器等有挑战性的前端场景，也有机会深入探索 AI Coding 与 Agent 驱动的研发方式。

你将负责

1.参与 AI 基础设施平台的设计与开发，覆盖 GPU 集群管理、节点运维、资产管理、工单系统等核心场景；

2.负责复杂表格与表单、数据可视化仪表盘、网络拓扑图、在线代码编辑器等功能的设计与实现；

3.持续优化大规模数据场景下的页面性能、交互体验和系统稳定性；

4.与后端、SRE、产品等团队紧密协作，将基础设施运维流程产品化、自动化，提升整体运维效率；

5.探索 AI Coding、Agent、Skill、Plugin 等新型研发方式，并将其应用到真实工程实践中；

6.参与前端工程体系和通用组件建设，持续提升团队的研发质量与交付效率。

## 任职要求

1.具备 1–3 年前端开发经验，有扎实的 JavaScript / TypeScript 基础和良好的工程习惯；

2.熟练掌握 React 或 Vue 中至少一种技术栈，熟悉状态管理、路由、组件设计等常用生态；

3.熟悉前端工程化体系，使用过 Vite、Webpack 等构建工具，对 CI/CD 和自动化交付有基本理解；

4.关注性能与用户体验，能够独立分析并解决复杂交互或大规模数据场景中的问题；

5.具备良好的沟通协作能力，主动负责，愿意深入理解业务并推动问题落地；

6.对 AI 编程工具和新型研发范式保持好奇，愿意持续学习和实践。
