## 职位描述

1. 负责 AI 基础架构团队的前端开发,覆盖 GPU 集群管理、节点运维、资产管理、工单系统等多个业务平台;

2. 实现大规模表格/表单、数据可视化仪表盘、网络拓扑图、在线代码编辑器等交互场景，持续优化平台性能与易用性;

3. 与后端、SRE等团队深度协作,将基础设施运维流程产品化、平台化,持续提升运维效率与体验。

4. 学习 & 探索AI coding 链路

## 任职要求

1. 1–3 年前端开发经验,精通 JavaScript / TypeScript,具备良好的代码规范与工程素养;

2. 熟练掌握 React / Vue,理解其核心原理与生态(状态管理、路由、组件设计等);

3. 理解前端工程化思想,熟悉 Webpack / Vite 等构建工具,了解 CI/CD 持续集成流程;

4. 熟练使用 Claude Code / Codex 等 AI 编程工具,了解 Skill / Plugin / Agent等机制的原理与作用,能在日常研发中显著提升效率;

5. 良好的沟通与协作能力，主动性强，具备较强的学习能力和责任心
