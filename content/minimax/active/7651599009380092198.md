## 职位描述

-负责 MaaS 网关的架构规划与技术演进，持续提升高并发、大流量场景下的吞吐、延迟、可靠性和可扩展性。

-完善模型服务的质量治理体系，围绕可用性、延迟、成功率、异常响应、调用效果和服务 SLA 建设监控、评估、告警和复盘机制。

-与业务、客户成功及技术团队紧密配合，推动复杂问题快速响应并形成闭环。

## 任职要求

-5 年以上后端开发经验，有 API 网关、服务治理、开放平台、AI 平台、云服务或高并发服务端系统建设经验。

-熟悉网关核心能力，包括 API 服务、鉴权认证、密钥管理、路由转发、限流熔断、故障切换、监控告警等。

-具备复杂服务链路治理经验，能够围绕稳定性、可用性、延迟、成功率和服务质量建立系统化保障机制。

-具备较强业务理解和 owner 意识，能面向客户场景和商业化目标，推动接入、服务、稳定性、计量计费等问题闭环。

-有团队建设经验，能制定技术方向、建立工程规范，辅导成员成长。

加分项

-有云厂商、MaaS 服务厂商或复杂 ToB 项目推进经验。

-做过技术、业务、商务之间的桥接角色，并沉淀过方法论或机制。

-有复杂客户项目、重大线上问题处理经验。
