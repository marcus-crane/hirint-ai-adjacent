## 职位描述

1. 负责 MaaS（Model as a Service）平台的设计与开发，构建统一的模型服务化能力。

2. 支持模型的部署与服务化，包括多模型、多版本的管理与发布。

3. 参与推理服务链路建设与基础优化（如延迟、吞吐、资源利用率等）。

4. 设计并实现基础平台能力，如服务管理、简单调度、监控与稳定性保障。

5. 提供标准化接口（API/SDK），提升模型能力的接入与使用效率。

6. 与算法及基础设施团队协作，推动模型能力的稳定落地。

## 任职要求

1. 计算机相关专业，本科及以上学历。

2. 扎实的软件工程基础，熟练掌握 Go / Python / C++ 至少一门语言。

3. 有后端开发经验，了解基本的分布式系统设计。

4. 对模型推理流程有基本理解，有一定性能优化意识。

5. 具备良好的问题分析与工程实现能力。

加分项

1. 有模型部署或推理服务相关经验。

2. 熟悉 Kubernetes 或容器化部署。

3. 有推理优化经验（如 TensorRT、vLLM、Triton 等）。

4. 有 AI Infra 或平台开发经验。
