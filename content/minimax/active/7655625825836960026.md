## 职位描述

1. 支持 MiniMax Agent 相关业务的数据分析工作，包括用户行为、功能使用、转化漏斗、留存等数据分析；

2. 协助搭建和维护 Agent 产品的数据看板，跟踪核心指标变化；

3. 通过 SQL 提取和分析数据，发现产品使用中的问题和优化机会；

4. 配合产品、研发、运营团队，完成专题分析、实验效果分析和业务复盘；

5. 尝试使用 AI 工具 / Vibe Coding 提升数据分析、报表生成和问题定位效率；

6. 有 Vibe Coding / Cursor / Claude Code / Codex 等 AI 编程工具使用经验者优先。

## 任职要求

1. 本科及以上在读，统计、数学、计算机、数据科学、经管等相关专业优先；

2. 有 SQL 基础，能独立完成基础数据提取和分析；

3. 对 AI Agent / 大模型产品感兴趣，愿意主动学习和体验 AI 工具；

4. 逻辑清晰，对数据敏感，有较好的问题拆解和沟通能力；

5. 有 Python、BI 工具、数据分析项目经验者优先。

【加分项】：

1. 深度使用过 AI Agent、AI Coding、AI 办公工具等产品；

2. 有互联网产品、B 端平台、开发者工具、AI 产品相关分析经验；

3. 能快速学习业务，并用数据支持产品决策。
