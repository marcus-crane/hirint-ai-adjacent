岗位职责：

1. 协助完成机器人整机及核心硬件模块（电机、传感器、控制器、电路板等）的功能、性能、可靠性、EMC等测试工作，参与测试计划制定、测试用例编写与执行。
2. 准确记录测试数据，协助定位硬件缺陷，跟踪缺陷整改进度，参与回归测试及测试报告的编制。
3. 维护测试设备与工装，保障测试环境稳定，协助优化测试方法，提升测试效率与覆盖度。
4. 配合研发、生产团队反馈测试问题，参与硬件缺陷复盘，协助输出测试优化建议。

专业技能：

1. 了解机器人整机硬件架构，掌握电机、传感器、控制器、电路板等核心硬件模块的基本工作原理及常见测试方法。
2. 能基本操作万用表、示波器、信号发生器等常用硬件测试仪器，协助完成测试数据采集与基础分析。
3. 了解硬件功能、性能、可靠性测试的基本流程，能在指导下编写测试用例、执行测试、协助定位硬件缺陷。

任职要求：

1. 本科及以上学历在读，电子信息工程、通信工程、自动化、测控、机电等相关专业。
2. 具备扎实的电路、模电/数电基础知识，有课程设计、电子竞赛或实验室项目经验者优先。
3. 工作细致、有责任心，具备良好的文档编写能力和沟通协作意识。
4. 能够保证每周至少4-5天的实习时间，持续3个月以上者优先。

加分项：

1. 有机器人、智能硬件或类似系统的硬件测试、调试经验（含课程项目）。
2. 熟悉至少一种MCU（如STM32）的基本原理，能编写简单的测试代码。
3. 了解EMC、环境测试、安规测试基本概念或常见标准。
4. 能使用Altium Designer等EDA工具查看原理图/PCB图。
5. 了解缺陷管理工具（如JIRA、禅道）的基本使用。

我们希望你：

1. 对硬件测试工作有热情，细致耐心，愿意深入一线发现并解决问题。
2. 具备主动学习的自驱力，善于总结测试经验并推动问题闭环。
3. 不只关注测试执行本身，也能从整机和产品质量视角思考改进方向。
4. 乐于沟通协作，适应开放、务实、高效的团队氛围。

你将获得：

1. 开放、务实、自由的团队氛围，资深工程师一对一指导。
2. 深度参与人形机器人核心硬件从模块到整机的完整测试流程，接触真实量产级测试标准与方法。
3. 系统学习硬件功能、性能、可靠性、EMC、环境测试等专业知识与实践技能。
4. 有机会参与测试工装设计、自动化测试平台搭建等进阶工作。
5. 在高成长性机器人赛道中积累从测试执行到质量体系理解的完整项目经验。
