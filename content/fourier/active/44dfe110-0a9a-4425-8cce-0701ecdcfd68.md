岗位职责：

1. 协助完成人形机器人嵌入式系统的软件设计、开发和调试工作，覆盖底层驱动、中间件及上层应用模块。
2. 参与嵌入式系统功能的需求分析与实现，遵循代码规范，编写可维护、可测试的代码，支持代码重构与优化。
3. 配合硬件、算法及测试团队完成系统联调，协助定位软件相关问题并推动闭环解决。
4. 参与机器人关节驱动、传感器数据采集、通信协议栈等核心模块的嵌入式软件开发与验证。
5. 辅助编写软件开发文档、接口说明、测试记录及相关技术资料。

任职要求：

1. 本科及以上学历在读，计算机、电子工程、自动化、通信、测控、机电等相关专业。
2. 熟悉 C/C++，了解 Rust 或其他常用编程语言，具备良好的编程习惯，了解工程配置与编译流程。
3. 了解常见通信接口（如 CAN、RS485、EtherCAT、I2C、USB、SPI、UDP、TCP）的基本原理，有实际使用经验者优先。
4. 了解常见传感器（如 IMU、LiDAR、相机）的数据读取与解析方法，有相关项目经验者优先。
5. 熟悉常见单片机（如 ESP32、STM32）的工作原理与开发流程，能独立完成基础外设驱动开发。
6. 了解常见嵌入式系统（如 Linux 驱动模块、RTOS 任务开发）的基本工作原理，有实时操作系统开发经验者优先。
7. 具备良好的逻辑沟通能力和解决实际问题的意识，工作踏实、认真负责，乐于主动思考与学习。

加分项：

1. 有机器人、无人机、智能车或类似嵌入式系统项目开发经历（含课程设计、竞赛或实验室项目）。
2. 熟悉 EtherCAT、CANopen 等工业总线协议或有实际调试经验。
3. 有 ROS/ROS2 开发经验，了解机器人软件架构。
4. 有 Python 或 Shell 脚本开发经验，能辅助自动化测试与调试。
5. 有版本控制（Git）和单元测试相关实践经历。

我们希望你：

1. 对机器人行业有热情，乐于接受复杂系统带来的技术挑战。
2. 具备自驱力和探索精神，能在指导下独立完成模块开发与问题排查。
3. 不只关注代码本身，也能从整机系统视角理解软件与硬件的协同关系。
4. 乐于分享与协作，适应开放、直接、高效的团队沟通风格。

你将获得：

1. 开放、务实、自由的团队氛围，尊重技术判断与个人成长节奏。
2. 深度参与人形机器人核心嵌入式系统研发，接触真实机器人产品中的驱动、通信、传感器融合等前沿问题。
3. 与资深嵌入式工程师共同工作，获得系统性的软件设计与工程实践指导。
4. 有机会参与技术方案讨论、架构评审和工程决策，拓展软件架构视野。
5. 在高成长性机器人赛道中积累从模块开发到系统联调的完整项目经验。
