# **About the Team**

The Security Engineering team is responsible for building and scaling the security foundations that protect 1X’s people, infrastructure, robotics platforms, cloud environments, and operational systems. We partner closely with Infrastructure, AI, Robotics, Manufacturing, and Enterprise teams to ensure security is deeply integrated into how we build and operate embodied AI systems.

Cloud infrastructure is foundational to how 1X develops, deploys, manages, and scales humanoid robotics systems globally.

# **Your Charter**

Design and lead the cloud security architecture that secures 1X’s infrastructure, applications, and operational environments across a rapidly scaling robotics organization. This role is critical to establishing secure-by-default cloud infrastructure, scalable security controls, and resilient operational systems that support embodied AI development and deployment at scale.

# **Key Outcomes**

* Design and implement scalable cloud security architectures across AWS, GCP, Azure, and hybrid infrastructure environments
* Improve organizational security posture through implementation of secure infrastructure patterns, cloud governance controls, and automated security enforcement mechanisms
* Partner cross-functionally with Infrastructure, DevOps, AI, Robotics, and Security teams to integrate security into CI/CD pipelines, cloud platforms, and engineering workflows
* Reduce operational risk by implementing scalable monitoring, detection, vulnerability management, and incident response capabilities across cloud environments
* Establish infrastructure security standards, policies, and operational best practices supporting rapid company growth and global scale

# **Key Competencies**

* Deep expertise securing large-scale cloud infrastructure and distributed systems environments
* Strong hands-on experience with infrastructure-as-code, automation, and cloud-native security tooling
* Strong understanding of identity, network security, secrets management, zero trust principles, and cloud governance
* Excellent systems-thinking and architecture design skills across highly complex technical environments
* Strong collaboration and communication skills across engineering, infrastructure, and security organizations

# **Minimum Requirements**

* 8+ years of experience in cloud security, infrastructure security, security architecture, or related engineering domains
* Strong hands-on experience securing AWS, GCP, Azure, or hybrid cloud environments at scale
* Experience with Terraform, Kubernetes, CI/CD systems, and infrastructure automation tooling
* Strong scripting or programming experience using Python, Go, Bash, or similar languages
* Bachelor’s degree in Computer Science, Information Security, Engineering, or related technical field

# **Preferred Skills**

* Familiarity with container security, workload isolation, and Kubernetes security best practices
* Experience implementing detection engineering, SIEM, cloud monitoring, or incident response systems
* Exposure to compliance frameworks such as SOC2, ISO 27001, NIST, or FedRAMP
