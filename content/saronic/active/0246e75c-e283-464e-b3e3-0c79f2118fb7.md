Saronic Technologies is a leader in revolutionizing autonomy at sea, dedicated to developing state-of-the-art solutions that enhance maritime operations through autonomous and intelligent platforms.

# **Job Overview**

We are seeking a Full Stack Engineer with a heavy frontend focus to take ownership and drive the development of the command & control web applications. The ideal candidate will have a strong understanding of frontend technologies and frameworks while proficiently developing the supporting backend servers and APIs.

# **Responsibilities**

* Develop and maintain high-quality web applications using HTML, CSS, JavaScript, and modern frontend frameworks (e.g., React, Angular, or Vue)
* Communicate with external UX designers when requiring advanced designs capabilities
* Communicate and incorporate feedback from external clients to ensure the UI/UX decisions conform to their needs
* Balance utilizing known/reliable solutions while tracking new technologies to improve usability

# **Qualifications**

* A desire to work with DoD technology
* Strong UI/UX design principles and tools experience
* Basic proficiency with Rust and Python
* Strong experience developing SPAs/PWAs via modern frontend frameworks (e.g., React, Svelte, WebComponents, Vue)
* Frontend performance optimization and debugging tools (e.g., Chrome DevTools)
* Proficiency in developing backend servers and APIs with [Node.js](http://Node.jsDeploying)
* Deploying to Cloud Service Providers (e.g., Azure, AWS, GCP)
* Knowledge of databases technologies (e.g., Postgres, SQLite, Redis)
* Utilizing Web Workers to push events and notifications
* Familiarity with video encoding, decoding, and streaming
* Web Gamepad API building and utilizing WASM modules
* Web accessibility standards
