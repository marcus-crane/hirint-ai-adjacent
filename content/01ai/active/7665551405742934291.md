## 职位描述

我们正在打造面向企业的新一代 Agent 平台。

我们相信，企业 AI 不只是回答问题的工具，而是能够理解业务目标、自主规划任务、持续感知企业环境，并与人协同完成复杂工作的智能系统。

1、这个岗位将负责企业级 Agent 平台及 Agent Governance 平台的产品规划与持续迭代。我们希望你不仅能够打造稳定、易用、可扩展的平台产品，更能够与我们一起探索下一代企业 AI 产品的新范式。

2、我们期待与你一起思考：

企业如何构建真正可持续演进的 Agent System，而不仅仅是一个个 Agent？

Agent 如何持续理解企业 Environment，而不仅仅理解用户输入？

人与 AI 如何建立高效、可信、可控的人机协作关系？

企业级 Agent 平台未来的产品形态应该如何演进？

当越来越多公司都在做 Agent 平台时，我们真正的产品竞争力应该来自哪里？

如果这些问题让你兴奋，我们希望与你一起定义下一代企业 AI 产品。

1、负责企业级 Agent 平台及 Agent Governance 平台的产品规划、需求分析、方案设计与持续迭代，对产品体验和产品结果负责。

2、深入理解企业客户、开发者及业务用户需求，负责 Agent 的构建、运行、协同、治理等核心能力设计，持续提升平台易用性、生产效率和规模化能力。

3、负责企业级 Agent Governance 能力建设，包括组织、权限、资源、应用、安全、运营、审计等核心治理能力设计，支撑企业级 AI 的稳定运行。

4、从客户实践和复杂业务场景中抽象共性能力，推动产品标准化和平台化建设，建立可复用、可扩展的产品能力，而不是简单满足单一需求。

5、持续研究国内外 Agent、AI 平台及开发者产品的发展趋势，深入分析竞品，探索新的产品形态、人机协作方式和平台能力，推动产品持续创新。

6、与研发、算法、设计、业务及客户团队紧密协作，在用户价值、技术实现和商业目标之间做出合理判断，推动产品持续迭代。

## 任职要求

1、本科及以上学历，3 年以上 ToB 产品经验，有平台产品、AI 产品或复杂企业级产品经验。

2、具备优秀的产品抽象能力，能够从复杂业务场景和客户需求中沉淀标准产品能力，在产品灵活性、标准化和可扩展性之间做出平衡。

3、熟悉企业级产品设计，理解组织、权限、资源、配置、安全、运营等企业治理体系，有大型企业产品经验者优先。

4、对 Agent、大模型及企业 AI 产品保持持续关注，理解 Agentic AI、Multi-Agent、Harness、Environment、Ontology 等新一代 AI 产品理念，并能够结合企业场景完成产品设计。

5、具备优秀的 Product Sense，能够独立负责复杂产品模块，完成需求分析、产品设计、版本规划及持续迭代。

6、 具备良好的沟通与协作能力，能够与研发、算法、设计及业务团队共同拆解问题，并在用户价值、技术实现和商业目标之间做出合理判断。

7、保持持续学习和探索精神，对优秀 AI 产品保持好奇心，乐于尝试新的产品形态，并能够将创新想法真正落地。

加分项

1、有 Agent 平台、大模型应用平台、开发者平台、AI Infra、低代码平台或复杂企业平台产品经验。

2、有从 0 到 1 打造平台产品，或推动成熟平台完成重大升级的经验。

3、有企业级治理平台、多租户、权限体系、开放平台、平台运营等相关经验。

4、深度体验并持续关注 Claude、ChatGPT、Cursor、Codex、Lovable、Dify、Coze 等 AI 产品，对其产品设计有自己的理解，并能够形成独立的产品判断。

5、熟练使用 AI Coding 工具，并能够通过 AI 提升产品设计、原型构建和团队协作效率。

我们期待你

1、对优秀产品保持好奇，而不仅仅完成需求。

2、能够从第一性原理思考产品，而不是简单复制竞品。

3、能够从复杂需求中提炼长期价值，坚持标准化，而不是功能堆砌。

4、保持持续学习，对 AI 技术和产品的发展保持敏锐判断，并将新的理念转化为真正可落地的产品能力。

我们相信，未来优秀的 AI 产品经理，不是需求的执行者，而是产品范式的探索者。我们期待与你一起重新定义企业如何构建、使用和管理 AI。
