## 职位描述

1 .参与云手机管理系统、多端 SDK 及相关生态服务的测试用例设计与执行；

2. 完成功能、接口、稳定性、兼容性测试，定位并推动缺陷修复；

3. 使用 Python 设计测试脚本，实现日志分析，测试效率提升，基础场景回归测试等；

4. 输出测试报告及质量风险评估文档，沉淀测试资产，保障 PaaS 平台版本质量。

## 任职要求

1. 本科，计算机相关专业，PaaS / 云手机 / 服务端测试经验优先；

2. 熟悉软件测试流程，能独立设计用例，熟悉接口测试、缺陷管理流程；

3. 掌握 Python ，独立开发测试脚本和自动化任务，了解 Shell 命令；

4. 了解 Docker 概念及基础操作（启停容器、查看日志、环境部署）；

5. 熟悉 ADB、Logcat，有 Android SDK / 兼容性测试经验优先；

6. 熟练使用 Linux、具备基础问题定位与沟通能力；

7. 具备良好的沟通能力、问题分析能力，责任心强，能快速适配云手机业务测试场景。
