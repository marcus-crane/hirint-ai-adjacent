## 职位描述

1. 负责 AI Agent 平台相关后端服务的设计、开发和维护。

2. 参与万智 Workspace、Workflow、Toolbox 等模块与 Agent Sandbox 的集成适配。

3. 对接云端沙箱的 Session 生命周期、代码执行、浏览器操作、文件传输、日志及运行结果管理能力。

4. 建设统一模型服务接入能力，包括：

- 多模型供应商接入；

- OpenAI 兼容 API；

- 流式输出与异步任务；

- 模型路由、重试、降级和故障切换；

- Token 用量统计与成本核算。

5. 建设 API Key、租户、项目、配额、限流、权限和审计等平台能力。

6. 参与计量计费、余额校验、调用记录、对账及异常补偿等功能开发。

7. 负责服务的性能优化、可观测性和线上问题定位，保障高并发场景下的稳定性。

8. 与产品、前端、运维及外部云服务团队协作，推进需求上线。

## 任职要求

1. 计算机或相关专业本科及以上学历，3 年以上后端开发经验。

2. 熟练掌握 Java、Go 中至少一种语言；以 Java 为主亦。

3. 熟悉Spring Cloud，Spring Boot，Dubbo，MyBatis/Hibernate等开发框架，熟悉 RESTful 架构风格设计，使用Maven/SVN/GIT等工具；

4. 精通多线程编程，熟悉常见的中间件、缓存、消息队列等，熟悉PostgreSQL/Mysql，熟悉Linux下DevOps；

5. 熟悉 Linux、Docker，能够完成基本的容器化部署和故障排查。

6. 具备良好的接口设计、问题分析和跨团队沟通能力。

7. 对 AI Agent、模型 API 或云端执行环境有兴趣，能够快速学习相关技术。
