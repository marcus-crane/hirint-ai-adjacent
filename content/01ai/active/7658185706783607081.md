## 职位描述

负责公司新一代 Agent + Ontology 驱动的数据智能平台产品建设。

我们正在探索一种面向企业核心业务场景的数据产品形态：通过 Ontology 建模企业业务对象、状态、关系与证据，通过 Agent 持续理解业务上下文、识别风险与机会、生成判断建议，并在业务反馈中持续优化。

工作职责

1. 负责 Agent + Ontology 数据平台的产品规划与 0→1 建设，定义核心产品架构、关键模块与Roadmap。

2. 通过 Agentic 的方式，沉淀企业核心业务场景的决策流程，并分别翻译为人与Agent可读的语义协议。

3. 设计 AI-Ready Data 能力，包括数据源接入、业务对象识别、字段语义映射、数据质量校验、证据追溯与权限控制。

4. 与算法、工程、业务专家和FDE协作，将企业结构化与非结构化数据转化为 Agent 可理解、可调用、可推理的业务上下文。

## 任职要求

任职要求

1. 3 年以上 ToB 产品经验，有数据平台、AI知识库/AI搜索等复杂企业级数据产品经验优先。

2. 具备优秀的业务抽象能力，能将复杂业务问题拆解为业务对象、状态、信号、证据、规则、判断逻辑与反馈机制。

3. 具备较强的数据产品意识，理解数据源、字段语义、对象映射、数据质量、权限治理、数据血缘、证据追溯等概念。

4. 理解 Agentic AI 产品形态与算法全生命周期。

5. 具备 0→1 产品定义能力，能在没有成熟行业范式的情况下搭建产品框架、设计 MVP、推动快速验证。

6. 熟悉Ontology、GraphRAG等相关技术和方法是加分项。
