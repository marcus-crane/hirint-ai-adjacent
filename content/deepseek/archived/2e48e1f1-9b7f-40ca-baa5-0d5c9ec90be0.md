**职位描述：**

1.负责将业界领先的 Agent 开发工具集成到内部强化学习（RL）基础设施中，打通外部工具与内部训练体系的链路；

2.负责 Agent 容器服务的日常维护，包括框架版本同步、依赖管理、环境稳定性保障等运维工作；

3.搭建并完善 Agent 评测平台，开发轨迹查看、调试分析等可视化工具，提升模型迭代效率；

4.维护和优化内部 Agent 集成框架，支持便捷对接 RL 训练框架及外部 Agent Scaffold（如 LangChain、OpenAI Agents SDK 等），降低研发接入成本；

5.与算法团队紧密协作，参与 Agent 能力的工程化落地，推动 Agent 系统的可靠性与可扩展性提升。

**职位要求：**

1.具备前端开发经验，熟练掌握 JavaScript/TypeScript，能独立完成前端功能开发；

2.熟悉 Python、Rust，具备扎实的编程功底和工程能力；

3.熟悉 Docker、Kubernetes 等容器技术，有容器化服务部署与运维经验；

4.熟悉 Claude Code、Cursor、Copilot 等 AI 编程工具的使用与原理，有相关二次开发经验者优先；

5.具备良好的代码规范意识，熟悉 Git 工作流、CI/CD、代码审查等工程实践。

**加分项：**

1.有 Agent 评测平台、轨迹回放或可视化工具的开发经验；

2.对大语言模型（LLM）有系统性认知，理解 Context Window、Token 机制等核心概念；

3.熟悉 Agent 交互协议与规范，包括 MCP、Tool Use、Function Calling 等；

4.深度使用 Claude Code、Cursor 等 AI 编程工具，具备高质量 Prompt 编写能力。

5.参与过CTF竞赛。
