**岗位职责：**

1. 负责 DeepSeek iOS / Android 客户端的功能开发与持续迭代；

2. 与产品团队紧密合作，灵活选用合适的技术打磨极致的用户体验。

**岗位要求：**

1. 具备 iOS / Android 原生开发经验，熟悉 Swift / Objective-C 或 Kotlin / Java 等编程语言；

2. 对新技术抱有热情，具备扎实的编程能力、优秀的设计能力和代码品味，具有强烈的责任心和主动性；

3. 对应用性能敏感，不断改善性能表现；

4. 对交互和体验有极致的追求，能开发出用户友好的交互体验；

5. 具有优秀的团队合作能力和沟通协作能力，善于学习，乐于分享，热爱挑战。

**加分项：**

1. 对技术底层原理、实现有深入的了解；

2. 了解前后端技术栈，具备全栈开发经验；

3. 有优质个人项目、开源社区贡献或独立开发作品者优先。
