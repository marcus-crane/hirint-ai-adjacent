**【团队使命】**

AGI 时代，人类与机器交互的革命已经开启，AI 赋予了产品主动思考的能力与无限的可能，我们正站在时代的前沿，将智能通过用户界面具象化，建立人类与 AI 相互触达、沟通、协作的桥梁，在这里，你将：

1. 探索 AGI 时代尚未被定义的人机交互与产品可能，从对话式交互走向 Agent 协作、主动式服务等新范式，定义人与 AI 协同工作的新方式。
2. 持续优化 DeepSeek 面向各类用户的终端产品，打磨细节，让性能与体验始终先人一步。

招聘方向**【实习/全职】**：**前端开发方向、客户端开发方向**

**前端开发方向**

**【工作职责】**

1. 负责 DeepSeek 网页版、开放平台等服务的迭代开发，探索、预研 Agent 等概念的新交互范式，寻找 AGI 时代的人机交互方式。深度参与前端设计与实现，持续优化体验，服务千万日活用户，发现 AI 产品的更多可能性。

**【岗位要求】**

1. 熟练使用 JavaScript、TypeScript。
2. 熟练使用至少一个主流前端框架。
3. 熟练掌握 HTML、CSS、HTTP 等前端基础知识。
4. 熟练掌握各类前端技术加分（如 Node.js，前端工具链，WASM 等）。
5. 对用户体验、交互操作及用户需求分析等有一定了解，有产品或界面设计经验者优先。
6. 有优秀的个人技术产品、开源作品，或活跃的开源社区贡献者优先。

**客户端开发方向**

**【工作职责】**

1. 负责 DeepSeek iOS / Android 端 App 的迭代开发，探索、预研 Agent 等概念的新交互范式，结合硬件与系统特性，寻找 AGI 时代的人机交互方式。深度参与数千万日活 App 的体验优化、性能优化，共同探索 AI 产品的更多可能性。

**【岗位要求】**

1. 具备 iOS / Android 原生开发经验，熟悉 Swift / Objective-C 或 Kotlin / Java 等编程语言（iOS、Android 有一方面熟练即可）。
2. 对应用性能敏感，能熟练定位性能瓶颈并优化。
3. 对新技术抱有热情，具备扎实的编程能力、优秀的设计能力和代码品味，具有强烈的责任心和主动性。
4. 具备用户洞察能力，有产品、交互设计相关经验者优先。
5. 有优秀的个人技术产品、开源作品，或活跃的开源社区贡献者优先。
