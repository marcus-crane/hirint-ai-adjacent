**【团队使命】**

每一次技术革命，最终都要通过产品抵达人们的生活。

随着模型能力不断突破，过去关于软件、交互和生产力的边界正在被重新书写，而产品团队正是这场变革的设计者与推动者。

DeepSeek 产品团队站在模型与世界之间，负责将最前沿的智能能力转化为千万用户触手可及的产品体验。在这里，你面对的不只是功能设计，而是在定义人与 AI 协作的新范式，通过连接模型与用户、算法与场景，让 AI 从实验结果变成改变世界的产品。

从一个按钮、一次对话，到全新的工作流和产品形态，你的每个决策都可能影响数千万人的使用习惯。你将与模型、算法、研发和运营团队共同探索，在技术能力与用户价值之间寻找最佳路径。

招聘方向**【实习/全职】**：**AI 产品方向、Agent Harness 产品方向**

**AI 产品方向**

**【工作职责】**

1.产品规划：紧密跟踪 AI 技术发展，制定并执行 DeepSeek 在 Chat 网页端、APP 端、开放平台的产品策略，敏锐洞察用户需求，合理规划优先级，不断推动产品创新和体验优化。

2.功能设计：综合运用数据分析、用户反馈和定性观察，从真实用户需求和使用场景出发，定位并拆解产品问题，降低用户使用门槛，提升使用深度、留存率、粘性和转化率。

3.项目管理：与模型、算法、研发和运营团队紧密合作，以快速节奏推进需求从调研、方案、评审、开发到上线，如期保证高质量交付。

**【核心要求】**

1.有产品经历或完整的从需求到落地的交付经验，AI / 对话 / Agent 类产品经验优先。

2.是 AI 产品深度用户，对模型能力边界、不同产品的体验差异和用户场景有真实体感，不只是看过，而是自己用过、拆过。

3.有好的产品品味和用户洞察，注重用户的日常交互细节，追求极致的产品体验。

4.逻辑清晰，有数据意识，能把用户体感、产品直觉和行为数据结合起来做判断。

5.自驱力强，认真细致，沟通力强，能独立推进复杂项目的规划、排期、落地与迭代。

**【加分项】**

1.对 AI 工具有深度使用经验，深度使用过 Agent / AI Coding / Workflow 类产品，有独特的见解和强烈的热情。

2.具备 UI/UX 设计素养，能够在 AI 辅助下完成产品原型图设计、UI 设计等相关的产品设计工作。

3.有工程背景，或能使用代码 / AI coding 工具提升自己的工作效率。

**Agent Harness 产品方向**

**【工作职责】**

1.规划 DeepSeek Harness 产品路线图，连接研究员、工程师、开源社区和广大用户。

2.理解判断用户的最真实需求，定义与衡量“Agent 是否真的在更多场景下更深入的帮助到更多的人”的指标。

3.帮助 Harness 产品落地，持续跟踪真实场景下的用户任务。推动产品创新和体验优化。

4.维护 Harness 产品用户社群，从潜在海量用户群体中获取反馈、提取信号，指导产品迭代。

5.与模型训练团队的研究员深度沟通与合作，实现模型与 Harness 的共同进化。

6.协助项目管理相关工作。

**【核心要求】**

1.2年以上产品经理从业经验，产品逻辑清晰。特别优秀的候选人可放宽年限。

2.知名高校本科学历及以上。

3.能够使用 vibe coding 写代码，不一定需要技术背景。

4.具备UI/UX设计素养。能够在 AI 辅助下完成产品原型图设计、UI设计等相关的产品设计工作。

5.是 Agent 产品的高强度用户，熟悉 Agent 产品的各种形态，对 Agent 有极大的热情，对模型行为有品味有判断力，对开发者体验及用户体验有强感知。深度使用过 Claude Code、Cowork、Codex、Cursor、OpenCode、GitHub Copilot、Manus、OpenClaw、Hermes 等类似产品，并将相关产品的使用融入到自己的工作和生活中。

6.理解 LLM 以及 Agent 基本机制及其技术原理，包括 LLM API、KV Cache、Agent Loop、Tool Use、Reasoning、Planning、Skills、MCP、Memory、Subagent、Multi-Agent 等相关知识。对 Prompt Engineering、Context Engineering、Harness Engineering 等课题有第一手实践。

7.能够设计系统性的收集数据的方法（包括问卷、访谈、A/B测试、灰度测试等），并使用统计学的工具严谨科学的分析数据。对此有系统性的思考和实践。

8.对产品细节有强感知力，对异常分支、边界条件和失败场景有敏锐的嗅觉。

9.优秀的中文沟通能力。能用英文与开源社区、用户社群书面沟通。

**【加分项】**

1.AI行业从业经验、AI相关产品经验。

2.作为小团队中的产品经理主导产品路线的经验。

3.与研究员深度协作的经验。

4.深度参与开源社区或维护开源社区关系的经验。

5.其它超乎常人的与此工作相关的才能。
