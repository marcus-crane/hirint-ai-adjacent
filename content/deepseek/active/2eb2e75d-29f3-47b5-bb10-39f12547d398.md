**招聘方向【实习/全职】**：**线上核心服务、Agent 后端、数据仓库**

**【团队使命】**

**与智能共演化，让工程即作品。**我们致力于以创新、优雅的工程方案，构建与 AI 协同进步的软件系统，让模型智能稳定、高效地驱动社会进步。

对内，你将与资深研究员一起协作，共同构建用于提升模型智能边界的工程基建；对外，你将深度参与 AI 产品的创新迭代，让智能触达亿万用户。

**【岗位类别】：实习/全职**

**线上核心服务方向**

**【岗位职责】**

**负责 DeepSeek 线上服务及相关基建的各类研发工作，深度参与：**

1.面向数千万日活用户的大模型应用与 API 服务架构设计和持续迭代，性能、网络和数据库优化。

2.内部研发效能工具与平台搭建。

3.不断探索各种新技术，持续优化服务的可靠性、可扩展性、可观测性、可维护性。

**【岗位要求】**

1.具备扎实的计算机基础和编程能力，熟练掌握至少一门编程语言。

2.熟练掌握常用数据结构与算法。

3.具备良好的学习能力和开拓创新精神，有责任心，积极主动，肯钻研。

**【加分项】**

**（非强制要求，也欢迎没有工程经验但基础素质扎实的同学加入）**

1.有过大型业务系统的开发经验。

2.在大型开源软件中有深度贡献，或有优秀个人开源项目。

3.在信息学、数学、CTF 竞赛中取得过优秀成绩。

**Agent 后端研发方向**

**【岗位职责】**

1.负责 Agent 执行环境快照系统的搭建与维护。

2.负责接入各类 Agent 框架，支持高效、稳定的评测工作。

3.与研究员协作，产出高质量 Agent 数据，用以提升模型智能。

**【岗位要求】**

1.具备扎实的计算机基础和编程能力，熟练掌握至少一门编程语言。

2.熟练掌握常用数据结构与算法。

3.具备良好的学习能力和开拓创新精神，有责任心，积极主动，肯钻研。

4.理解大模型及 Agent 的基本原理。

**数据仓库方向**

**【岗位职责】**

**你将参与以下一种或多种方向，使用业界前沿的数据技术栈，参与下一代数据架构的建设，用最佳实践承接 DeepSeek 核心产品的数据分析需求，撬动巨大的业务价值：**

1.数据管道：负责大规模数据 Pipeline 的设计、开发和落地。

2.离线计算：协助业务团队和数据分析团队，开发正确、高效、稳定的数据处理任务。

3.实时计算：建设、维护流处理系统，为监控、实时分析、风控系统等业务提供可靠的数据底座。

4.架构与稳定性建设：发现现有数据架构中的瓶颈，主导内部平台、工具的建设，提出并落地关于性能提升、链路稳定性、资源优化的解决方案。

**【岗位要求】**

1.本科及以上学历，计算机或相关专业，1～5 年数据开发/数据仓库/数仓建设工作经验。

2.除 SQL 外，熟练掌握至少一门编程语言。

3.有良好的代码风格和技术品味。
