**【团队使命】**

文字延伸了记忆，印刷术延伸了知识传播，互联网延伸了感官与连接。我们相信，Agent 将成为对人本身的又一次「延伸」—— 以搜索打破静态知识的围墙，以工具触达数字世界的疆域，让人与 AI 的关系从对话走向托付，从而真正解放人的行动力与创造力，构建「人机共生」的新文明。

**【岗位类别】：实习/全职**

**【工作职责】**

1.定义并推进大模型在真实任务场景中的 Agent 能力，深入办公与日常生活等场景下的复杂环境，使模型能自主规划、执行并交付完整结果，持续识别能力瓶颈，驱动模型迭代优化。

2.围绕 AI 搜索及自主任务执行的全流程，设计并落地多维度自动化评测与归因体系，建立能真正区分“能用”与“好用”的评测标准，覆盖端到端任务完成度与过程行为质量，打通人工评估与自动化评估，精准定位模型短板。

3.针对各类 AI 搜索与 Agent 任务，定义回答及执行结果的理想态，设计并迭代高质量、高多样性的数据生产管线，系统性提升搜索回答与 Agent 产物的准确性、完整性和用户体验。

**【岗位要求】**

1.拥有主流 AI 搜索及 Agent 产品的深度使用经验，熟悉当前主流 Agent 架构与产品形态，能够敏锐感知模型能力差异，定位背后的数据需求与问题。

2.对任务质量有清晰的判断标准与内在追求，对“好”有明确的界定能力，能精准辨识输出质量的层级差异，并能将主观感受拆解为可传达、可检验的具体维度。

3.理解模型训练与优化的基本原理，熟悉模型评测方法，精通 prompt engineering 和（vibe）coding；动手能力强，有丰富的使用自动化手段解决真实问题的经验。

4.对 AI Agent 领域有真实的热情，持续关注前沿进展，对 AI 能力的边界保持探索欲与想象力。相信 Agent 终将深度改变人机协作的方式，让工作更高效，让创造更自由。

5.有以下经验者优先：Agent 产品深度使用或构建、自动化工作流设计、评测体系搭建或大模型数据构造经验。
