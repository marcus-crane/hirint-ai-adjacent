## 职位描述

1，参与 RISC‑V CPU 微架构设计和方案制定

2，编写 RTL（Verilog/SystemVerilog），优化满足 PPA（性能、功耗、面积）目标，进行时序分析与 RTL 调优

3，与验证团队合作，支持功能 & 指令集验证

4，SoC 级别集成与系统交互设计，包括 CPU 与其他 IP 的整合与验证

## 任职要求

1，电子工程、计算机工程、微电子、计算机科学等相关专业本科 / 硕士毕业

2，熟悉RSICV/ARM指令集

3，精通 RTL 开发与调试，熟练掌握 Verilog / SystemVerilog，具 FPGA 或 ASIC 开发经验优先

4，深入理解 CPU 微架构设计原理（流水线、branch prediction、cache、内存系统等）

5，英语阅读与技术文档写作能力强，具备良好沟通协作能力

加分项：

1，熟悉脚本语言，有高级语言编程能力（C / C++ / Python）

2，参与过香山/一生一芯等开源项目，熟悉Gem5/Nemu环境
