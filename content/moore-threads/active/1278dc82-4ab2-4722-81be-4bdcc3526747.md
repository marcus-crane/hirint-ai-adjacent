## 职位描述

摩尔线程 AI 研发团队正在寻找一位热爱技术、擅长写作的 Technical Writer 加入团队。你将负责为 AI 框架、分布式训练/推理系统和大模型产品创建高质量的技术内容。

你将参与的产品：vLLM、SGLang、AI Agent、PyTorch3D、SimuMax、MooER、AnimateAnyone、LiteGS、torch\_musa 等。

核心职责

1. 技术文档撰写

○ 编写 AI 框架（PyTorch、Megatron、DeepSpeed、vLLM、SGLang 等）的 API 文档、快速入门指南、教程

○ 创建分布式训练/推理系统的架构文档、部署指南和故障排查手册

○ 维护 GitHub 开源项目文档（README、Wiki、示例代码）

2. 开发者体验优化

○ 与产品、研发团队紧密合作，早期介入文档规划

○ 收集用户反馈，持续改进文档质量

○ 建立文档规范、风格指南和术语库

3. 技术内容创作

○ 撰写技术博客、产品发布文章、最佳实践案例

○ 翻译和国际化技术文档（中英互译）

4. AI 工具应用

○ 运用 AI 工具辅助内容创作、翻译和审校

## 任职要求

任职要求：

1. 本科及以上学历，计算机、软件工程、通信、数学或相关专业

2. 3 年以上技术文档写作经验，有 AI/ML/云计算/GPU 领域优先

3. 理解深度学习框架（PyTorch、TensorFlow、PaddlePaddle 等）

4. 了解分布式训练原理（数据并行、模型并行、流水线并行等）

5. 熟悉大模型概念（Transformer、LLM、LoRA、量化、推理优化等）

6. 能阅读 Python 代码示例，了解 CUDA/C++ 更佳

7. 优秀的中英文书面表达能力（CET-6 或同等水平）

8. 熟练使用 Markdown、Git、Docs-as-Code 工作流

9. 熟悉至少一种文档平台（GitBook、Docusaurus、ReadTheDocs 等）

加分项：

● 有开源项目贡献经历

● 有大模型推理框架（vLLM、SGLang、TGI）使用经验

● 熟悉 MLOps、模型部署、服务化

● 有技术社区运营、DevRel 经验

● 在技术媒体、公众号、知乎有文章发表
