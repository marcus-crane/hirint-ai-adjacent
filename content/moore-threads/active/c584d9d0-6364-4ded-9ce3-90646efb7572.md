## 职位描述

虚拟原型平台开发工程师，负责虚拟原型平台（QEMU）的搭建与维护，重点实现 GPU 芯片软硬件协同设计，推动 CPU 和 GPU 模拟器的功能建模与优化。

岗位职责：

1. 负责虚拟原型平台（QEMU）的搭建与优化，支持芯片设计团队的软硬件协同设计。

2. 负责 CPU 和 GPU 模拟器的功能建模，尤其是针对 ISA（指令集架构）指令的功能实现，确保模拟器精度和效率。

3. 深入了解并调试 QEMU 的运行机制，以便针对具体的芯片架构进行有效扩展与定制。

4. 根据项目需求，设计并实现基于 RISC-V 架构的模拟功能，支持相关硬件组件的模拟与验证。

5. 与硬件开发、验证团队紧密合作，确保虚拟原型平台能够支持设计验证、性能调优和功能验证等工作。

6. 持续进行虚拟原型平台的性能优化和工具链完善，以适应不同芯片的开发需求。

7. 提供技术支持和文档，确保团队成员对平台功能的理解与应用。

## 任职要求

1. 计算机科学、电子工程或相关专业本科及以上学历。

2. 精通 C++ 编程语言，具备优秀的编码能力和问题解决能力。

3. 具有虚拟化技术（如 QEMU）相关经验，能够搭建并优化虚拟原型平台。

4. 深入了解 RISC-V 架构及其 ISA（指令集架构），具备芯片架构设计、功能建模的实际经验。

5. 熟悉硬件与软件协同设计流程，理解 CPU 和 GPU 的软硬件协同工作机制。

6. 具备良好的团队合作精神，能够与跨团队合作并推动项目进展。

7. 能接受一定程度的加班，尤其在项目交付和关键任务时。

加分项：

• 具备 GPU 架构、性能优化相关经验者优先。

• 具有硬件设计验证（如仿真、原型验证）经验者优先。
