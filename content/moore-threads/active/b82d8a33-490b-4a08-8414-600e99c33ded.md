## 职位描述

负责摩尔线程企业级智能体系统架构设计及研发工作，构建可扩展、可训练、可评测、可长期演进、安全可靠的 Agent 平台。

岗位职责描述：

• 架构设计：负责企业级 AI Agent 的整体系统架构设计，包括感知层（Perception）、规划层（Planning）、记忆层（Memory）及工具调用（Tool Use）的设计。

• Agent Infra：负责研发 Agent Infra 底座，构建可扩展高可用 Agent系统，将业务需求转化为可执行的智能体协作流程，赋能企业实现自动化、智能化业务流程。

• Agent 安全：构建 Agent 权限认证体系，保障企业数据安全，在系统架构层面内置严格的合规性检查和隐私保护机制 RAG 方案落地：设计高性能的检索增强生成（RAG）系统，优化向量库检索效率、文档切分策略及重排序（Rerank）机制。

• 工程化与评测：构建 Agent 的闭环评测体系（Eval），通过量化指标监控模型幻觉、工具调用准确率及端到端任务完成度。

• 性能优化：通过 Prompt Engineering、计算图优化、缓存机制（Cache）及模型微调（SFT）提升 Agent 的响应速度与成本效率。

任职资格

• 计算机科学、人工智能或相关专业硕士及以上学历

• 3年以上AI/机器学习领域工作经验，其中至少2年专注于大模型应用或AI Agent相关项目

• 精通Python，熟悉主流深度学习框架（PyTorch/TensorFlow）

• 熟悉LangChain、AutoGen、LlamaIndex等Agent开发框架

• 具备系统架构设计能力，熟悉分布式系统、微服务架构和高性能计算
