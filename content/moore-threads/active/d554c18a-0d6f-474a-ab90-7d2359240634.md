## 职位描述

1.工具链设计与开发： 负责GPU板级产品研发及测试过程中的自动化工具、PC端上位机软件、测试工具的设计与开发。

2.自动化框架搭建： 搭建和维护GPU板级产品的自动化测试框架（HIL/SIL），实现从代码构建、固件烧录、功能测试到报告生成的全流程自动化。

3.设备交互实现： 开发PC与被测设备之间的通信库，实现对设备的远程控制、日志抓取和数据分析。

4.CI/CD集成： 将自动化工具集成到Drone/GitLab CI等持续集成流水线中，提升研发团队的交付效率和质量。

5.数据可视化与分析： 开发用于展示设备运行状态、测试结果统计、性能分析的Dashboard或Web平台。

6.需求对接与支持： 收集内部需求，持续优化工具的用户体验和稳定性。

## 任职要求

1. 全栈开发能力

2. 熟练使用 Python3，至少熟悉一种 Web 框架（Sanic / Flask / FastAPI）

3. 熟练掌握 HTML5、CSS、TypeScript，熟悉 Flex 布局，至少掌握一种前端开发框架（Vue3 / React）

4. 熟练掌握 Java8+，了解 EJB 框架

5. 至少熟悉一种关系型数据库 MySql / PostgreSQL，具有 SQL 编写能力

加分项：

熟悉 Docker容器化技术

熟悉 Redis

熟悉 Done

有HIL (Hardware-in-the-Loop) 硬件在环测试系统开发设计经验。
