## 岗位职责

1. 设计并实现机器⼈多模态交互数据的分布建模与处理体系，对⼤规模具身数据进⾏价值评估；

2. 设计⾃动化数据筛选、难例挖掘与负样本⽣成机制，实现⾼质量数据的规模化⽣产与迭代优化；

3. 建⽴⾯向具身智能模型的评估体系，定义数据质量与模型性能之间的量化指标；

4. 与模型算法与训练团队协同，构建“数据–模型”协同进化机制，持续提升机器⼈操作泛化；

## 任职要求

1. 精通 Python 及主流深度学习框架（如 PyTorch，JAX)。

2. 具备扎实的统计学习与机器学习基础，深⼊理解机器⼈多模态数据分布特性。

3. 具备⼤规模数据处理与分布式训练经验；对机器⼈学（运动学、动⼒学）具备基础认知。

4. 具备具身多模态数据处理经验者优先。

5. 具备⼤规模数据及模型能⼒评估分析经验者优先。

6. 曾在CV, AI, Robotics 相关顶会、顶刊（如：CVPR，ICLR，NeurIPS，CoRL，TRO)发表论⽂者优先。
