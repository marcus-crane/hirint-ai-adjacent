## 职位描述

1、负责机器人运控仿真环境的设计与开发，构建用于动作验证、运控算法联调与安全评估的云端仿真系统

2、建立机器人动力学与运动学仿真链路，支持关节级控制、轨迹跟踪、状态回放与误差分析；

3、将动作生成 / 重定向 / 运控控制器（MPC / RL / IK） 接入仿真环境，实现算法在真机前的充分验证；

4、设计并实现安全与稳定性仿真机制：1. 关节限位、自碰撞检测 2.姿态稳定性、跌倒风险评估 3.地面接触、步态连续性分；

5、支持多机器人、多场景仿真，用于编舞、协作与复杂动作验证；

6、将仿真能力服务化，支撑灵创平台“仿真校验 → 一键下发真机”的生产流程；

7、与运控、感知、动作生成团队协作，推动仿真结果与真机行为的一致性（Sim-to-Real） 持续优化。

## 任职要求

1、计算机、机器人、自动化等相关专业本科及以上，3 年以上相关经验；

2、熟悉机器人运动学 / 动力学基础（FK/IK、刚体动力学、接触建模）；

3、熟悉至少一种机器人仿真引擎：MuJoCo / Isaac Gym / Isaac Sim / Gazebo / Bullet 等；

4、熟练使用 Python / C++，具备较强的工程实现能力；

5、了解常见运控算法：轨迹跟踪、MPC、RL 控制、全身控制（WBC）等；

6、良好的系统设计能力与跨团队协作能力。
