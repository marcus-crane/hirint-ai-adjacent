## 职位描述

1. 嵌入式构建系统管理：负责设计、维护和优化基于 Yocto Project 的嵌入式 Linux 固件构建系统，打造稳定、高效的“固件工厂”。

2. 嵌入式 CI/CD 流水线建设：基于 Jenkins 和 Conan 包管理器，构建端到端的嵌入式软件自动化流水线，实现从代码提交、编译、测试到固件版本发布的持续集成与交付。

3. 构建环境容器化：利用 Docker 容器化技术统一和固化嵌入式交叉编译环境，解决环境依赖问题，提升构建环境的一致性与资源利用率。

## 任职要求

1. 开发与系统能力：掌握 Bash/Python 进行自动化脚本开发；熟悉 C/C++ 及 CMake 构建系统，能够协助分析和调试嵌入式构建问题。

2. 嵌入式开发基础：熟悉 Linux 系统及嵌入式开发环境，熟悉交叉编译工具链及相关调试工具。

3. CI/CD 与容器化技术：熟练使用 Jenkins 设计和维护复杂流水线，具备将自动化测试集成到流水线中的实践经验；掌握 Docker 容器化技术。
