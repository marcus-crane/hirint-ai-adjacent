## 职位描述

1、制定公司国内 PR 传播策略，围绕产品里程碑、技术发布、战略合作、行业活动等关键节点，策划并落地全年 PR 节奏，提升品牌行业声量与权威认知。

2、搭建并维护科技、财经、产业、AI 及机器人垂直媒体矩阵，负责日常媒体对接、选题沟通、专访安排、稿件投放，保持高频、高质量正面曝光。

3、独立撰写新闻稿、深度稿、创始人专访、行业观点稿、对外口径、演讲 PPT 等；把控所有对外内容质量，将技术语言转化为大众与行业易懂的传播语言。

4、协同品牌、活动团队，完成品牌发布会、论坛、赛事、挑战赛、顶会 Workshop 等活动的传播、媒体邀请、现场公关与后续声量放大，打造行业标杆形象。

5、负责日常舆情监测、预警、分析与应对；建立危机公关预案与响应机制，快速处理负面信息，维护品牌口碑稳定。

6、负责机器人 / AI 类自媒体内容策划、分发与互动；拓展并维护行业 KOL、专家、博主资源，开展合作传播，提升品牌在科技圈层影响力。

## 任职要求

1、本科及以上，2年以上科技 / AI / 机器人 / ToB 硬科技 PR 相关经验。

2、具备优秀文案能力，可独立完成新闻稿、深度稿、专访稿等全类型内容创作。

3、熟悉国内主流科技 / 财经 / AI / 机器人媒体资源，有成熟媒体人脉优先。

4、有机器人 / AI 自媒体运营、内容创作或推广经验优先。

5、对具身智能、物理 AI、数据基建、机器人行业有浓厚兴趣与基础认知。
