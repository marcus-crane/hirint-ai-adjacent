## 职位描述

1. 仿真平台搭建与维护：基于 NVIDIA Isaac Sim / Isaac Lab / Omniverse 搭建并维护人形机器人仿真环境，配置机器人、传感器与物理参数，构建可配置、可批量、可复现的场景与任务流水线，沉淀可复用的场景库与仿真资产。

2. 感知导航算法仿真验证：在仿真中复现真实场景，验证感知导航算法的正确性、鲁棒性与边界表现，设计典型及极端工况，支撑算法的回归与对比验证。

3. 大规模合成数据生成：设计并实现合成数据生成 pipeline；通过域随机化提升 sim-to-real 泛化能力。

4. 算法评测与 Benchmark 建设：针对感知导航算法定义评测指标，搭建自动化评测 pipeline，支持回归测试、横向对比与团队级 benchmark，并通过误差归因与 bad case 分析输出可行动反馈。

5. 基础设施沉淀：与上下游紧密协作，将需求转化为明确的数据/评测规格，持续沉淀可复用的工具、资产与流程，提升团队整体数据与评测效率。

## 任职要求

1. 本科及以上学历，计算机、机器人、自动化、控制、人工智能等相关专业；

2. 熟悉 Isaac Sim 全家桶（Isaac Sim、Isaac Lab 或 Isaac Gym/Orbit、Omniverse、USD 场景编排），能独立搭建仿真环境；

3. 扎实的 Python/C++ 工程能力，熟悉 Linux 与 GPU 开发环境；

4. 具备优秀的系统设计及工程架构能力，有大规模分布式数据生成的架构经验的优先；

5. 熟悉域随机化与sim2real，有合成数据驱动真实模型迭代及benchmark的经验的优先。
