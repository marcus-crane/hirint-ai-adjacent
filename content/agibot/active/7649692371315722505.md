## 职位描述

本岗位隶属于软件开发团队，核心定位为嵌入式系统底层开发，聚焦智能硬件、工业机器人、边缘计算等产品的板级支持包（BSP）全生命周期开发、维护与优化。你将负责嵌入式硬件平台的 Linux 系统底层适配，解决从芯片到外设的底层技术问题，为产品提供稳定、高效、可扩展的底层系统支撑，成为连接硬件与上层应用的核心桥梁。

岗位职责

1 BSP 开发与定制化适配

负责主流嵌入式芯片平台（如Rockchip RK3588/3568、NVIDIA Jetson Thor/Orin、DRIVE Orin/Thor 等）的 BSP 开发，完成 U-Boot/TF-A 启动加载器、Linux 内核（主线内核 / 厂商定制内核）、设备树（DTS/DTSI）的移植、定制与版本迭代。

基于产品需求定制系统镜像（rootfs），优化系统启动流程，裁剪冗余模块，提升系统启动速度与资源利用率。

维护厂商 SDK 及 Linux 内核的定制化补丁，保障 BSP 与硬件平台的深度适配及版本迭代的兼容性。

2 外设驱动开发与调试

负责硬件外设（GigE/10G Ethernet、PCIe 3.0/4.0、SPI/I2C/UART/CAN-FD、USB 3.0、摄像头 / 显示屏、传感器等）的驱动适配、调试与问题排查，解决外设兼容性、稳定性及性能问题。

针对自研硬件模块开发专用驱动，完成驱动的功能验证与性能优化，确保外设功能满足产品规格要求。

3 系统调试与性能优化

定位并解决 BSP 层各类故障，包括系统启动异常、内核 Oops/Panic、内存泄漏、中断风暴、外设通信失败等底层问题。

使用 perf、ftrace、gdb/kgdb、trace-cmd、ethtool 等工具分析系统性能瓶颈，针对 CPU 调度、内存管理、网络传输等模块进行优化，提升系统实时性与整体性能。

针对人形机器人场景的实时性需求，优化 Linux 内核调度策略（如 RT\_PREEMPT 实时补丁适配），保障任务执行的低延迟。

4 跨团队协作与技术支持

为上层应用开发团队提供底层技术支持，解决系统调用、驱动接口、性能瓶颈等问题，协助完成应用与底层系统的集成调试。

与测试团队配合，制定 BSP 测试用例，复现并解决测试过程中发现的底层问题。

## 任职要求

1 计算机/电子/自动化等相关专业本科及以上学历。

2 编程与架构：精通 C 语言，熟悉 ARM64/ARM32 架构及汇编指令，理解 Linux 内核核心原理（进程调度、内存管理、中断处理、设备模型等）。

3 BSP 核心技能：熟练掌握 U-Boot/TF-A、Linux 内核、设备树的开发与调试，能独立完成芯片平台的 BSP 移植与定制。

4 调试能力：熟练使用 gdb/kgdb（内核调试）、perf/ftrace（性能分析）、串口调试、示波器 / 逻辑分析仪等工具，能独立定位并解决复杂的底层故障。

5 驱动开发：具备外设驱动（如网络、CAN-FD、USB、传感器等）的开发或调试经验，理解 Linux 驱动框架设计思想。

加分项

1 有机器人、自动驾驶等场景的 BSP 开发经验，熟悉功能安全（ISO 26262）或信息安全相关要求者优先。

2 具备 Linux 内核补丁编写经验，或参与过内核社区贡献、开源项目维护者优先。

3 熟悉 RTOS（如 FreeRTOS、RT-Thread）与 Linux 异构系统开发，或有裸机程序开发经验者优先。

4 掌握 PTP/IEEE 1588v2 时间同步、GPU/DPU 驱动适配、硬件虚拟化等技术者优先。
