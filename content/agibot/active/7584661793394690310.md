## 职位描述

1. 多本体 SDK 接入集成：负责将数据采集 SDK 适配到不同类型的机器人本体（人形、机械臂、移动底盘等），解决不同硬件平台、不同软件架构下的兼容性与集成问题。

2. 端侧数据采集系统：在机器人端实现高效、低侵入的数据采集，协调多传感器（相机、力矩、关节状态等）的数据同步与打包，确保数据质量与完整性。

3. 端侧分布式组件部署：负责分布式系统组件在端侧的部署与运维，包括数据缓存、上报队列等模块，保障端到云数据链路的稳定性。

4. 跨平台适配与优化：针对不同嵌入式平台（如 Jetson Orin）的资源特点，进行性能调优与资源管理，在保证数据吞吐的同时控制 CPU/内存占用。

## 任职要求

1. 机器人软件框架：熟悉 ROS/ROS2 生态，理解其通信机制与节点架构；有 ZeroMQ 等消息中间件的使用经验。

2. 系统编程能力：精通 Python，具备 C++ 开发能力；熟悉多进程/多线程编程，理解 GIL 机制对性能的影响。

3. 嵌入式系统经验：有 Jetson Orin 等边缘计算平台的开发经验，熟悉 ARM 架构与资源受限环境下的软件优化。

4. 系统集成能力：具备良好的问题排查与调试能力，能够快速定位跨模块、跨系统的集成问题。

加分项：

有机器人、自动驾驶等领域的端侧软件开发经验

有多传感器同步与标定的实践经验

熟悉机器人本体控制系统的软件架构

主导或参与过机器人领域的开源项目
