## 职位描述

岗位职责】

1. 负责公司“具身智能数据采集设备”配套移动端App的架构设计与核心功能开发，主要基于 Flutter 框架，实现 Android、iOS 与鸿蒙（HarmonyOS）等多端覆盖；

2. 负责 App 与智能硬件设备之间的底层通信链路开发，包括但不限于蓝牙（BLE）、Wi-Fi、局域网 Socket、MQTT 等协议的对接与数据传输；

3. 负责高频传感器数据、视频流或音视频采集画面的前端实时渲染与可视化展示，保障 App 在高并发数据流下的性能与稳定性；

4. 编写并维护跨平台框架与原生端（Android/iOS）交互的插件（Plugin），解决复杂的设备适配与底层接口调用问题；

5. 参与产品的需求分析与系统架构设计，与产品经理、嵌入式软件工程师、硬件及UI设计团队紧密配合，能从前后端全局视角优化用户体验与采集链路的稳定性。

## 任职要求

【任职要求】

1. 学历与专业：本科及以上学历，计算机、通信、电子、自动化等相关专业优先；

2. 核心技术（必须）：5年以上大前端/移动端开发经验，精通 Flutter 跨平台开发，扎实的 Dart 语言功底，深入理解 Flutter 渲染机制及主流状态管理（如 Provider / Riverpod / GetX 等），能独立完成跨端 App 的架构、性能调优及上架流程；

3. 前端基础技能：熟练掌握 HTML5、CSS3、JavaScript/TypeScript 等现代前端开发技术，熟悉至少一种前端主流框架（如 React / Vue ），能够承担部分配套的 Web 后台管理系统或工具页面的开发；

4. 后端基础技能：具备基本的后端开发能力，熟悉 Node.js / Python / Go 中的至少一种后端语言及其主流 Web 框架（如 Express/NestJS, FastAPI, Spring Boot 等），了解常用数据库（如 MySQL, Redis），能够独立开发基本的 API 接口或与后端团队高效对接接口协议；

5. 硬件交互经验：熟悉移动端设备底层能力调用，有智能硬件控制、物联网（IoT）设备联调、低功耗蓝牙（BLE）或 Wi-Fi 直连开发经验者优先；

6. 综合素质：具有优秀的跨团队沟通能力和极强的技术自驱力，能够适应软硬件交叉领域的复杂业务场景。

【加分项】

1. 具备原生开发经验，熟悉 Android (Java/Kotlin) 或 iOS (Objective-C/Swift) 原生开发，能独立编写并优化 Flutter Native 插件；

2. 有音视频流处理（WebRTC/FFmpeg）、Canvas / WebGL / Three.js 3D 渲染开发经验，或接触过 ROS 机器人系统；

3. 熟悉鸿蒙（HarmonyOS NEXT / ArkTS）生态，有 Flutter 鸿蒙化（如 flutter\_flutter ohos分支）适配与打包经验者优先。
