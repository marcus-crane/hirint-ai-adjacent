## 职位描述

1. 主导 WebRTC 媒体服务器的选型、定制、部署与优化，编写高质量核心代码

2. 设计支持低延迟、高并发的 SFU 架构，涵盖信令、媒体传输、NAT 穿透（TURN/STUN）、QoS 监控等全链路

3. 优化音视频传输质量（QoS/QoE），包括抗弱网、拥塞控制、FEC、Jitter Buffer 等

4. 对接或定制开源 WebRTC 栈，提升系统稳定性与性能

5. 推动音视频核心技术在业务场景（如远程协作、AR 互动）中的落地与创新

6. 跟踪 WebRTC、AV1、SFrame、WebTransport 等前沿技术，评估并引入生产环境

## 任职要求

1. 5 年以上音视频或实时通信领域经验，3 年以上大型分布式系统架构经验

2. 深入掌握 WebRTC 协议栈，对 ICE、DTLS、SRTP、GCC/BWE、NACK/PLI/FEC 等有实战调优经验

3. 熟悉主流的webrtc开源项目如 Mediasoup、LiveKit、Pion，具备大规模生产部署经验

4. 了解音视频编解码（H.265/VP9/AV1）与硬件加速

5. 精通 Go / C++ 之一，具备高性能网络服务开发能力（高并发、低延迟、内存优化）

6. 熟悉 Kubernetes、微服务治理、可观测性（Prometheus/Grafana/ELK）在音视频场景的应用

7. 具备从 0 到 1 构建高可用、可扩展音视频平台的成功案例

8. 优秀的技术判断力、跨团队协作能力与技术领导力

加分项：

1. 熟悉端侧 SDK（Web/iOS/Android）与服务端协同优化

2. GitHub 有 WebRTC 相关开源贡献
