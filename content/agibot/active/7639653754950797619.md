## 职位描述

1. 负责机器人交互系统在端侧的运行时架构设计与工程实现，打通语音、视觉、模型、TTS、表情、动作、技能调用等模块。

2. 设计并实现交互状态管理与事件编排机制，支持首反馈、流式响应、打断恢复、上下文续接等能力。

3. 负责 fast path / slow path 的工程实现与链路优化。

4. 负责端侧与云端协同链路接入，包括路由执行、fallback、异常兜底与结果切换。

5. 负责端侧交互链路的性能分析、日志排查、稳定性治理与版本回归。

6. 支持 Demo 场景联调与展示效果打磨，保障系统在真实环境中的可运行性与可复现性。

## 任职要求

必备项：

1. 本科及以上学历，计算机、软件工程、自动化、机器人、电子等相关专业优先。

2. 3 年以上端侧工程、实时系统、机器人软件、推理服务、复杂业务链路或相关经验。

3. 熟悉 Python / C++ 至少一种，具备较强的系统工程实现与调试能力。

4. 有以下至少两类经验：

\* 端侧 AI 推理接入或模型服务编排

\* 实时事件流 / 状态机 / 调度系统

\* 多模块联调与复杂链路排障

\* 流式语音交互 / 打断恢复 / 上下文续接

\* 端云协同 / fallback / 降级设计

5. 对时延、稳定性、异常处理有较强意识，能将交互体验目标转化为端侧工程方案。

加分项：

1. 做过机器人、语音助手、车载、会议系统或智能终端交互链路。

2. 熟悉 VAD、ASR、TTS、LLM/VLM、Agent 等模块接入方式。

3. 熟悉 ROS2、异步消息、日志回放、性能 profiling 工具。

4. 有 Jetson、Thor、移动 SoC、边缘设备等平台经验。

5. 做过端侧 Demo、样机场景联调或系统展示支持。
