## 职位描述

岗位职责

1. 主导头部 6DOF VIO/SLAM、手部 IR 星座光学追踪、全身 Tracker（IMU + 骨骼运动学）三大子系统的算法架构设计与技术选型；

2. 制定多传感器（多目 RGB/Mono 相机 + 多 IMU + IR）的融合和标定方案；

3. 设计采集数据的质量评估与有效性验证流程（运动模糊、遮挡、同步误差、位姿精度等），建立数据清洗、标注、对齐的自动化 pipeline，保障下游训练数据资产质量。

4. 作为算法技术对接人，主导与外部算法/硬件供应商的需求定义、方案评审与精度验收，指导算法团队工程落地，沉淀核心 Knowhow。

## 任职要求

任职要求：

1. 精通视觉 SLAM / VIO（ORB-SLAM、VINS、OpenVINS、MSCKF 等），熟悉多目立体视觉、鱼眼模型、回环检测；

2. 精通 IMU 算法（预积分、EKF/UKF 紧耦合融合、AHRS 姿态估计、零偏与漂移处理）；

3. 有 VR 头显或 Inside-Out 追踪产品的量产级算法落地经验。

4. 精通 C++ / Python，具备嵌入式或边缘端（Qualcomm / Rockchip 平台）算法部署优化经验；

5. 有光学追踪（IR 星座 / PnP / 多相机几何）、全身动作捕捉、骨骼运动学（FK/IK、SMPL 等）等算法经验
