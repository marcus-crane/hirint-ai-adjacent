## 职位描述

1.负责轮式仿人机器人视觉、激光SLAM以及语义SLAM算法，包括建图、全局定位，局部高精度定位，重定位等任务

2.负责机器人的EOL产线标定和在线标定工作

3.负责开发各种动静态要素的4D真值系统，并用于自动标注，并形成数据闭环

4.负责双目视觉算法的研发，输出实时、可靠、一致性强的深度信息

## 任职要求

1.熟悉主流 SLAM 算法（如 ORB-SLAM、VINS-Mono、LIO-SAM、FAST-LIO2 等），具备视觉 SLAM 或激光 SLAM 实际量产经验

2.对图像/点云的动/静态场景要素进行检测和重建，包括实例追踪，实例分割等，形成表征环境的4D标注结果

3. 熟悉手眼标定算法、各种传感器的内外参标定算法，具备多传感器联合标定的量产经验

4. 有自动标注系统开发经验，能完成从算法研发到工程化落地的全流程

5. 熟练掌握 C++ / Python 编程，熟悉 ROS / ROS 2 框架，具备良好的软件工程能力

加分项：

1. 有自动驾驶自动标注相关项目经验值优先

2. 有地图/静态自动标注项目下的工作经验

3. 有NERF或者3DGS相关项目经验者优先

4. 有过双目立体视觉算法经验者优先
