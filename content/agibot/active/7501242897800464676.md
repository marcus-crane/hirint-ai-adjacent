## 职位描述

负责模型训练框架尤其是 Python 工程化相关的设计和研发。

负责基于视频的数据加载模块相关的设计和研发。

与算法工程师密切合作，提高训练框架的易用性，提升模型研发迭代效率。

## 任职要求

具备坚实的 Python/C++ 编程基础。

具备良好的代码规范和强烈的工程意识。

掌握 Python 工程化相关的规范，了解相应的工具及其基本使用。

熟练掌握 Docker 等容器化工具的使用。

对 Python 开源项目尤其是框架类项目有相关经验。

对 PyTorch 以及基于 PyTorch 的训练框架有一定了解。

【加分项】

有相关开源项目贡献经历。
