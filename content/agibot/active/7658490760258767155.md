## 职位描述

1. 负责 PC，H5前端需求迭代和前端技术优化。

2. 能与产品经理、UED等合作职能团队协作完成需求方案的合理设计，并转换成合适的前端技术方案。

3. 主导前端基建规划和建设，提升端上性能和用户交互体验。

4. 负责web技术栈相关的代码规范在团队内的落地和最佳实践。

5. 负责保障交付质量和生产系统的稳定性。

6. 用科学的手段发现和定位团队开发过程中的主要矛盾，并制定规划执行解决。

## 任职要求

1. 计算机相关专业优先，本科以上学历

2. 6年以上互联网前端开发经验，参与大型互联网产品设计研发工作优先

3. 熟悉Vue, React, NodeJs, SSR, SSG等主流技术和框架，并有相关实践经验，熟悉前端工程化方案，熟悉webpack/vite等构建工具。了解其他服务端技术。有丰富的前端技术排障经验。

4. 熟悉HTTP协议以及浏览器原理，了解CDN原理

5. 有独立完成复杂前端模块的设计与实现的经验，持续优化代码并保持良好兼容性

6. 善于思考，能独立分析和解决问题，有很强的学习能力，责任心强，具有良好的团队协作与沟通能力 。

【加分项】

1. 跨端经验： 有 React Native 或小程序（Taro）等跨端开发经验者优先。

2. 服务端开发： 熟悉 Node.js，有 NestJS/Koa/Express 等服务端框架开发经验，或有 Serverless 落地经验者优先。

3. 技术深度： 有大型复杂项目的前端架构经验，或在 GitHub 上有高质量开源项目/技术博客。

4. 具有使用AI Coding应用和经验者优先。
