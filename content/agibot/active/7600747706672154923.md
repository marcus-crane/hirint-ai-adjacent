## 职位描述

1、全面负责机器人动作重定向算法体系的设计与落地；

2、设计与实现通用的动作表示和数据规范化与重定向管线；

3、构建统一的动作资产处理与重定向框架，支撑灵创平台；

4、构建机器人动作执行安全体系；

5、姿态与关节角限制检查；

6、动作平衡与稳定性约束（防跌倒、防自碰撞）；

7、不同机器人机型之间的动作迁移与“能力裁剪”；

8、持续跟踪动作重定向、角色动画、机器人运动生成领域的前沿研究，并在产品中实现可落地的创新；

## 任职要求

1、计算机图形学、计算机视觉、机器人学、动画技术等相关专业硕士及以上（博士优先）；

2、在以下至少两个方向具备深入经验：

动作重定向（Human / Character / Robot）

骨骼动画与 Rigging（Humanoid Rig / Skeleton Mapping）

3D 动作格式（BVH/FBX/SMPL/AMASS 等）的解析与转换

逆运动学 / 正运动学（IK/FK）与优化

动作风格保持与风格化重定向（Style-preserving Retargeting）

3、熟悉常见的动作与骨骼数据格式：

如 BVH、FBX、GLTF、SMPL/SMPL-X、AMASS、Mixamo、Unity/Unreal Humanoid 等。

4、扎实的数学基础：线性代数、数值优化、多体运动学、刚体变换；

5、熟练掌握至少一种深度学习框架（PyTorch / TensorFlow），并具有将算法工程化落地的经验；熟悉 C++/CUDA 优先；

6、了解机器人运动学与运动链（URDF、关节 DOF、关节限制、平衡控制）者优先；

7、良好的工程能力与代码规范，能与平台研发、运控、动作生成等团队高效协同；

加分项

1、有动作重定向 / 角色动画 / 3D 动作相关论文（SIGGRAPH / CVPR / ICCV / ECCV / Eurographics 等）；

2、有实际项目：

BVH/FBX → SMPL / 标准骨骼 的转换工具

角色动画导入导出工具链（Unity / Unreal / Maya / Blender 插件等）。

3、有 Human-to-Robot 动作迁移经验，支持多机型与多自由度配置；

4、有批量动作库清洗、归一化、打标签经验。
