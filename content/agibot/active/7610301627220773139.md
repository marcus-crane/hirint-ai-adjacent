## 职位描述

1、构建覆盖“开发 - 测试 - 诊断 - 迭代”全生命周期的机器人工具平台；

2、使用 Golang / Python 开发高性能后端服务，支撑数据回放、Issue 管理、自动化测试、远程诊断等能力；

3、主导基于 ROS 与 Foxglove（MCAP）的可视化与数据处理工具链建设；

4、开发面向产测、售后、运维的一站式工具（如刷机、日志采集、故障恢复）；

## 任职要求

1、硕士研究生及以上学历，机器人、计算机、电子信息等相关专业，5年及以上工作经验优先；

2、精通 Golang 或 Python，具备构建 CLI 工具、Web 服务及后台任务系统的经验；

3、熟悉机器人数据格式（如 MCAP、ROS Bag），能高效处理大规模时序数据；

4、能使用 Python 快速构建原型、自动化脚本或与 ROS 生态集成；

5、熟悉 Kubernetes 基础设施，能将工具服务容器化并集成到 CI/CD 流程中；

6、有 C++ 调试或日志解析经验者优先（如解析端侧 core dump 或 binary log）；

7、注重开发者体验，擅长将复杂问题工具化、产品化。
