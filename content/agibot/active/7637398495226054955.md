## 职位描述

1、负责电子产品生产过程中治具控制软件、下线标定软件及固件烧录软件的开发与维护；

2、独立承担软件的需求分析、架构设计、编码实现及现场调试，确保软件稳定可靠，满足产线节拍要求；

3、基于 Windows/Linux 平台，使用 C++ / Qt 开发图形化上位机应用，实现与产线设备、仪器、治具的通信与控制；

4、实现与 MES、SCADA 等系统的对接，完成生产数据上报、参数下载等业务逻辑；

5、运用 Socket、HTTP、MQTT 等网络协议，实现与设备、服务器之间的高效数据交互；

6、编写技术文档（如软件设计说明书、使用手册、测试报告），并配合现场工程师完成产线导入和故障排查。

## 任职要求

1、工业上位机开发经验：具备 3 年以上工业自动化、测试测量或产线软件开发经验，能独立承担完整项目；

2、C++ / Qt 开发能力：

a、精通 C++（11/14/17 标准），熟练使用 Qt（Widgets/QML，信号槽，多线程，串口/网络模块）；

b、熟悉 Linux/Windows 跨平台开发，能解决平台兼容性问题；

3、网络通信：

a、扎实掌握 Socket（TCP/UDP）、HTTP/RESTful API、MQTT 协议；

b、有使用 libcurl、cpp-httplib、Mosquitto 等库的实际经验；

4、构建与版本管理：

a、熟练使用 CMake（编写 CMakeLists.txt，管理依赖与编译配置）；

b、熟练使用 Git（分支管理、协同开发、常规冲突处理）；

5、了解常见产线设备（如 PLC、扫码枪、程控电源、数字万用表）的通信接口（RS232、Modbus、GPIB、USB HID）；

6、具备良好的问题分析与现场抗压能力，能接受产线调试需要的短时跟线支持。

加分项：

1、熟悉 MCU 固件烧录原理（如 SWD、JTAG、ISP、DFU）及常用烧录算法；

2、了解 标定算法（如传感器线性校正、PGA 校准）或有过量产标定软件开发经验；

3、熟悉 Python 或 Lua，用于编写产线快速测试脚本；

4、有使用 Docker、Jenkins 等持续集成/部署经验
