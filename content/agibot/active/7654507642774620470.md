## 职位描述

1、负责整机及关节模组（电机 / 谐波 / 行星 / 丝杠 / 编码器）的振动噪声测试、诊断与优化，归零异响、啸叫、共振、抖动等问题；

2、负责机器人典型工况（步态、搬运、跌倒缓冲）下的 NVH 主观评价与测试体系建设；

3、负责机器人自噪声对麦克风拾音的影响评估与降噪，配合语音团队完成扬声器选型、放音调校与声学透明件（头壳 / 胸前壳）打孔评估；

4、负责跌倒、碰撞、卡顿等瞬态工况的机械 / 控制优化建议，跨部门（结构、关节、算法、测试）联合归零；

## 任职要求

1、本科及以上，振动噪声 / 声学 / 机械 / 机电 / 机器人工程等相关专业，3 年以上 NVH 经验；

2、熟悉 LMS / B&K / HEAD / HyperMesh / Nastran / Abaqus / ADAMS / MATLAB 等工具，具备振动、声学、信号处理基础；懂电声 / 语音前端 / 伺服控制者优先；

3、有机器人 / 关节模组 / 谐波减速器 / 电机 / 丝杠任一领域 NVH 经验者优先；能通过 FFT / 阶次 / 阶比 / 模态分析定位根因；

4、兼具主观听感 + 客观测试视角，跨部门协同与抗压能力强，可主导异响归零专项。
