## 职位描述

1. 负责机器人交互云二开SDK开发、维护及toB客户的语音交互基础能力（ASR、LLM、TTS、知识库及技能等）的对接和调试等工作，确保链路低延迟、稳定运行。

2. 运用 Java 和 Python 等语言进行开发，适配toB不同业务场景的交互需求，包括接口开发、数据处理、逻辑封装等工作。

3. 协同算法团队优化toB场景语音交互效果，解决识别准确率、语义理解精度及响应流畅度等问题，跟进功能从开发到测试验收的全流程。

4. 参与toB语音交互场景需求分析，并输出技术方案文档、接口文档及测试用例，支持toB场景业务快速迭代。

## 任职要求

1. 本科及以上学历，计算机科学、软件工程等相关专业，1 年以上语音交互或相关领域开发经验。

2. 熟练掌握 Java 和 Python 语言开发，具备扎实的代码功底，能独立完成模块开发、单元测试及问题修复。

3. 了解语音交互技术原理（ASR/LLM/TTS），了解主流语音平台（如科大讯飞、百度、阿里等）的 API 使用及集成方法。

4. 具备较强的跨团队沟通能力、需求理解能力和问题解决能力，能快速适应敏捷项目的节奏与协作模式。

5. 工作认真负责，抗压能力强，具备良好的代码规范和文档编写习惯。
