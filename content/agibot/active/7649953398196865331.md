## 职位描述

1. 负责企业级开放平台与 API Gateway 相关系统的设计、开发与维护，支撑机器人云平台能力对内、对外开放。

2. 参与开放 API 管理、开发者应用管理、密钥管理、认证鉴权、权限控制、多租户、流量治理、审计日志、可观测性等核心能力建设。

3. 参与 API Gateway 基础能力研发，包括路由转发、协议适配、签名校验、OAuth2 / JWT 鉴权、限流、熔断、灰度发布、访问控制等。

4. 支撑开放平台与 IAM、业务系统、运维平台、数据平台、应用商城等系统的集成，建设统一、稳定、可扩展的平台底座。

5. 推动平台在高可用、高性能、安全性、可维护性方面的持续优化，支撑大规模业务接入与生态开放。

6. 与产品、架构、业务研发、SRE 团队协作，持续完善开放平台能力体系、开发者接入流程和工程规范。

## 任职要求

1. 3-5 年后端开发经验，具备扎实的后端工程能力，熟悉 Go 语言开发优先；有其他后端语言经验并愿意转 Go 也可以。

2. 熟悉 REST API、WebSocket、OAuth2、JWT、API Key、签名校验等常见协议与认证机制。

3. 了解 API Gateway、开放平台、微服务治理相关体系，有网关、鉴权、限流、配置中心、多租户、权限系统等项目经验优先。

4. 熟悉 MySQL、Redis、MQ 等常用基础组件，具备良好的数据建模、接口设计和系统排查能力。

5. 了解 Docker、Kubernetes、CI/CD、OpenTelemetry、日志、Trace、Metrics 等云原生与可观测性体系者优先。

6. 具备良好的问题拆解能力、工程质量意识和团队协作能力，能够独立负责模块设计、开发、联调和上线。

加分项

1. 有开放平台、开发者平台、API 网关、IAM、OAuth2 / OIDC、权限系统相关经验。

2. 有 Kong、APISIX、Envoy、Nginx、Istio、Spring Cloud Gateway 等网关或服务治理经验。

3. 有多租户 SaaS、企业级平台、云平台、IoT 平台、机器人平台相关经验。

4. 有高并发接口、流量治理、稳定性治理、灰度发布、审计合规相关经验。
