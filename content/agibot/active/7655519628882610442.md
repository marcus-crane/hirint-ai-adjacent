## 职位描述

1. 负责企业协同与数字化产品的整体规划与迭代，制定产品路线图

2. 构建高效协同办公平台（飞书 / OA）、一体化人事平台、一体化行政平台等核心数字化系统

3. 主导业务流程梳理与需求设计，输出 PRD 与原型，牵头跨部门需求评审

4. 主导多系统集成与数字化体系建设，推动流程优化、组织效能提升与员工体验改善

5. 协同 IT、研发、业务团队，推进系统选型、实施、上线与运维全流程

6. 通过用户运营与数据反馈，持续优化数字化系统体验，沉淀可复用流程与产品资产

## 任职要求

1. 本科及以上学历，具备管理信息系统、计算机、人力资源、行政管理等相关专业背景

2. 5 年以上企业协同 / 数字化产品经理经验，具备多系统从需求到上线的端到端落地经验

3. 熟悉飞书、OA（钉钉 / 企业微信 / 泛微 / 致远等）、HR、行政等主流企业级系统的产品架构与实施

4. 擅长业务流程分析、系统集成与需求转化，具备优秀的 PRD 与原型设计能力

5. 擅长结构化思考与跨团队推动，具备制造业 / 机器人 / 数字化转型行业经验加分
