## 职位描述

1. 负责 Genie Studio 平台的功能测试和问题跟踪，确保 Genie Studio 平台的交付质量。

2. 建立云平台的质量保证流程，负责软件测试过程优化与改进，指导、跟踪、控制项目测试过程，确保项目测试符合流程与规范。

3. 解决测试过程中的复杂技术问题和风险，开发工作中需要的测试工具。

4. 构建 Genie Studio 平台的测试用例，保障 Genie Studio 平台的私有化部署能力。

5. 质量体系与度量指标建设，参与构建平台级质量体系，包括但不限于：测试策略、测试分层模型（单测 / 集成 / 接口 / 系统 / 回归 / 验收）、风险识别与覆盖策略，推动“测试左移”。

## 任职要求

1. 熟练 Python/Go 及常用测试框架（pytest、pytest-asyncio、pytest-benchmark；Go Test/Testify）。

2. 熟练掌握 PyTest / Testify 等测试框架的使用。

3. 熟悉 Playwright 或 Selenium，能编写稳定的端到端用例与选择器策略。

4. 具备良好的编码规范、文档能力与问题拆解能力，能推动质量门禁（lint/format/覆盖率/测试时长/flake 率）

5. 了解私有化部署与交付流程，对端云一体/异构算力/数据通道或机器人业务有基本认知。

6. 对 Genie Studio 平台以及机器人业务有一定了解。
