## 职位描述

【工作职责】

可选择以下一个或多个方向深入参与：

1. 建设具身智能原生的统一训练框架，支撑 VLA、世界模型、分层决策模型等具身基础模型的预训练与后训练，覆盖十亿、百亿至千亿级参数规模，并持续面向 300B+ 模型演进。

2. 建设大规模分布式训练系统，支持数据并行、张量并行、流水线并行、序列并行、混合并行、MoE、MoT、长上下文及低精度训练与推理，支撑万卡 GPU 集群稳定运行。

3. 建设规模化后训练与混合强化学习系统，涵盖在线 / 离线 RL、仿真 / 真机 RL，以及云端训练与边缘异步 Rollout 协同架构。

4. 建设训练效率优化体系，对数据、计算、通信及调度全链路进行度量与优化，包括 IO、算子融合、CUDA Graph、计算通信 Overlap、梯度同步及性能 Profiling 等。

5. 快速复现并工程化落地前沿模型与算法，基于统一 Benchmark 完成正确性、效果与性能验证，输出复现与测试报告，持续丰富框架支持的模型与算法。

工程师需要能够独立负责完整模块；专家需要能够负责方向规划、核心架构设计与复杂系统问题解决。

## 任职要求

【岗位要求】

1. 计算机基础扎实，学习能力强，愿意深入复杂系统与底层细节。

2. 代码能力强、工程习惯良好，对代码质量和长期可维护性有较高要求；善于使用 AI Coding / Code Agent 提升研发效率。

3. 具备良好的系统设计能力，能够在正确性、功能、效率、稳定性和可维护性之间合理取舍。

4. 理解至少一个相关算法方向的基本原理，能够从模型结构、数据流及训练 / 推理过程识别 Infra 需求，并具备跨数据、算法与系统定位问题的能力。

5. 熟悉 Python 和 PyTorch，了解 Megatron、DeepSpeed、verl、RLinf 等训练框架中的一种或多种。

6. 在大模型训练 / 推理、分布式系统、强化学习系统或具身智能算法任一方向具备实际项目经验。

7. 计算机、AI、机器人等相关专业本科及以上学历。

【加分项】

- 有大规模 GPU 集群训练经验，理解吞吐、调度、容错及稳定性等系统级问题。

- 熟悉 Megatron、DeepSpeed、verl、RLinf 等训练框架，有从零构建或深度重构训练框架的经验。

- 熟悉 CUDA / Triton 算子开发、RDMA 编程、CPU / IO 优化，或熟练使用 Nsight Systems、PyTorch Profiler 等性能分析工具。

- 有分布式真机 RL、云边协同或异步训练系统经验，或熟悉 Sim2Real 及 RoboSuite、ManiSkill、Isaac 等仿真器。

- 复现或工程化落地过 LLM、RL、多模态、VLA、世界模型相关论文或项目。

- 在大型开源软件中有深度贡献，拥有优秀个人开源项目，或在相关方向发表过高质量论文。
