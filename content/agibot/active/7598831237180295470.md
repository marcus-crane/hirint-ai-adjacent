## 职位描述

\* 面向 移动 / 桌面 / 嵌入式多平台 的 Flutter 应用架构设计与开发

\* 参与产品需求评审，与产品、后端、嵌入式软件，硬件团队协作，推动需求合理落地

\* 负责复杂 UI、动画、性能优化及稳定性建设

\* 设计并维护通用组件、基础库，提升代码复用率与整体工程质量

\* 排查并解决线上问题，持续改进用户体验

\* 参与技术选型、代码评审和工程规范建设

## 任职要求

\* 3 年及以上 Flutter / Dart 实际项目经验

\* 熟练掌握 Flutter 常用组件、布局系统、渲染机制

\* 熟悉 Dart 语言特性（async / isolate / mixin / 泛型等）

\* 熟悉至少一种状态管理方案（Riverpod / Bloc / Provider / Redux 等）

\* 熟悉 Flutter 性能调优（build 优化、repaint、内存分析等）

\* 了解 Android / iOS 基础原理（生命周期、线程模型、平台差异）

\* 具备良好的代码规范意识和文档习惯

加分项（具备其一即可）

\* 有 复杂业务 App 或 长期维护项目 经验

\* 熟悉 原生混合开发（Platform Channel / FFI）

\* 有 音视频 / 蓝牙 / 网络协议 / MQTT / WebSocket 等经验

\* 有 嵌入式 / IoT / 机器人 / 车机 / 大屏 Flutter 项目经验

\* 熟悉 CI/CD、自动化测试、工程化工具（FVM、build\_runner 等）

\* 对架构设计、性能优化有深入理解
