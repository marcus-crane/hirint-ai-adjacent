## 职位描述

1. 数据采集与数采设备建设：跟踪管理数采设备的校准和维护，负责设计和实施数据采集方案，跟踪产出指标，对接训练/评测/仿真团队的数据需求；

2. 数据管理与版本平台：建设统一的数据存储、格式、元数据、版本与权限体系，打通本地与云端存储/计算资源，支撑数据全链路的高效访问与复用；

3. 数据回流与质量闭环：打通部署态真机多传感器数据回流通道，建立质量检查、检索复用与失败案例回流机制；

4. 标注流水线统筹：协同构建"预标注—自动质检—人工抽检—版本发布"标注流水线，确保数据质量。

## 任职要求

1. 本科及以上学历，计算机、机器人、自动化、电子信息等相关专业;

2. 熟悉 Linux / Python / ROS，熟悉常用数据存储系统(PostgreSQL、MySQL、Redis、ElasticSearch 等);

3. 有大规模数据采集/存储系统的设计与工程落地经验，理解数据从采集到训练的全链路;

4. 具备较强的项目统筹与跨团队协调能力，保障采集量、质量与进度;

5. 有机器人/自动驾驶/具身智能行业数据采集、标注平台建设经验者优先。
