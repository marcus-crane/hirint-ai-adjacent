## 职位描述

1. 负责智能体产品与解决方案的设计与需求定义，输出 PRD 与产品规划。

2. 深入企业级重点客户，开展客户访谈与需求洞察，抽象不同客户的共性诉求，沉淀可复制的解决方案与交付方法论。

3. 把客户的业务流程拆解为可被智能体承接的"场景 / 任务"，定义智能体的技能、能力边界与人设。

4. 设计智能体应用的价值度量体系（采纳率 / 解决率 / 工时节省 / ROI），支撑售前讲解与交付验收。

5. 作为业务与研发之间的枢纽，负责需求优先级、节奏把控与跨团队协同。

6. 参与重点企业级项目的售前方案与落地交付。

## 任职要求

任职要求

1. 咨询背景优先（MBB / 四大 / 头部数字化咨询），有企业数字化转型、流程咨询或解决方案设计经验。

2. 极强的结构化思维与文档 / 表达能力，能把复杂业务抽象成清晰框架。

3. 有 B2B / 企业级软件 / SaaS / AI 产品相关经验，理解企业级交付的复杂度。

4. 优秀的客户沟通与项目推动能力，能直面客户决策层、独立推进项目。

5. 理解 Agent / RAG / 知识库 / 工作流等技术概念，能与研发高效对话。

6. 对大模型与智能体有热情和基本理解，愿意持续学习。

加分项

1. 有大型企业级客户的行业 know-how 或客户资源。

2. 主导过从 0 到 1 的产品或解决方案。

3. 较强的数据分析能力，能用数据驱动产品决策。
