## 职位描述

负责在 Android / 嵌入式Linux平台 上落地大模型与多模态推理系统，深入 底层系统、GPU/DSP/NPU 异构计算，优化模型性能和内存管理，确保 AI 推理在端侧高效、稳定运行。

工作内容包括：

1、端侧语言模型 / 多模态模型推理部署

2、异构硬件性能调优与内存管理

3、深入研究各主流芯片厂商技术栈演进

4、推理框架、工具链、跨平台构建系统开发与优化

5、模型量化、KV Cache 管理与端侧推理优化

6、跟进前沿模型架构（如 Transformer 及其衍生变体、投机解码加速技术 EAGLE 系列等）的迭代演进

## 任职要求

1、了解主流模型架构，包括 Transformer 系列、LLaMA、Qwen、Whisper 等，能分析其计算与内存特点

2、 熟悉至少一种端侧推理框架（ONNX Runtime / TensorRT / QNN）

3、 了解投机采样（Speculative Decoding / Sampling）等推理加速思路，能够分析其对端侧延迟、内存和工程复杂度的影响

4、精通 python/C/C++， 有 Android NDK / aarch64 交叉编译经验

5、具备多线程、内存管理实战经验

6、熟悉 mmap / GPU OOM / 内存调优，能分析内存带宽与计算瓶颈

加分项：

1、Jetson平台开发经验

2、GPU kernel / OpenCL / 自定义推理算子实现经验

3、 llama.cpp / 自定义推理 runtime 修改经验

4、有模型压缩、量化、蒸馏技术经验
