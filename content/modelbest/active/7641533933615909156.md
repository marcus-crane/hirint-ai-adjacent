## 职位描述

1. 设计并搭建适配 PineaPi 的端侧 Agent Runtime 框架

2. 实现任务调度系统：多任务队列、优先级管理、前台/后台任务的抢占与协调

3. 设计 Agent 记忆管理机制：短期对话记忆、长期用户画像、记忆压缩与检索

4. 实现工具调用（Tool Use）框架：让 Agent 能调用设备能力

5. 设计多 Agent 协作机制（如有需要）：不同能力的 Agent 之间的路由与编排

6. 在有限的端侧资源下做好 context 管理：KV Cache 持久化、上下文窗口优化

7. 定义 Agent 行为的评估与调试机制，确保助理回复质量可控

## 任职要求

1. 3 年以上后端/系统开发经验，有 Agent 框架或 LLM 应用的工程落地经历

2. 熟悉主流 Agent 框架的设计思想（LangChain / Dify /openclaw/Hermes等）

3. 扎实的系统编程能力，理解进程调度、内存管理、并发控制

4. 有向量数据库 / RAG / 记忆系统的实战经验

5. 能在资源受限环境下做工程取舍

加分项：

有 OS 内核或 RTOS 的任务调度经验
