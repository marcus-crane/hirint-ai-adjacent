## 职位描述

1. 负责 MiniCPM-o 全模态全双工模型的评测系统建设，覆盖 ASR、VoiceBench、OmniBenchmark、FullDuplex

Bench、LiveSports 等任务。

2. 建设自动化评测流水线和评测框架的设计、维护与重构。维护评测可用性、结果可视化工具和 badcase 分析工具。

3. 与算法同学协作，基于评测结果定位模型问题，推动训练数据、训练策略和模型能力迭代。

4. 参与设计全双工模型专项评测，包括说话时机、打断、多说话人、主动提醒、延迟、多轮上下文记忆、工具调用等。

## 任职要求

1. 软件工程能力强，熟练使用 Python，熟悉 Linux / Shell / Git。能够维护复杂代码仓库。

2. 熟悉大模型或多模态模型推理流程，有相关开源/闭源项目经验。

3. 对模型效果评估有基本理解，能结合 case、指标和日志产出清晰的评测报告。

加分项

1. 熟悉 ASR / TTS / 语音对话 / 视频理解评测 / 全双工其中之一领域评测经验。

2. 有 LLM-as-judge、评测框架、可视化经验其中之一。

3. 有集群任务系统、Docker、K8s、CI/CD、并行计算项目经验优先。
