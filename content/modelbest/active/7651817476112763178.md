## 职位描述

1、定制化部署与开发：在客户现场进行产品的安装、集成、调试与适配，解决系统兼容性问题。

2、技术落地与兜底：使用真实客户数据，把模型或系统在生产环境中真正跑通，并对最终交付结果负责。

3、需求转化：将客户模糊的、非技术的业务痛点，翻译为具体的技术需求与可执行的解决方案。

4、产品反哺：在现场趟出的解法和踩过的坑，抽象并沉淀为标准化的产品功能，反馈给研发团队。

## 任职要求

1、技术要求：熟练掌握主流开发语言（如 Python, Node.js, React 等），能深入陌生的代码库排查问题（Debug），熟悉 API 调用与 Agent 编排。

2、客户沟通能力：具备优秀的业务洞察力，能听懂客户的真实需求，将技术成果转化并解释给非技术背景的管理层和业务人员。

3、环境适应与抗压能力：能够适应较高频率的驻场或出差，在限制多、环境复杂的客户现场快速推进项目。
