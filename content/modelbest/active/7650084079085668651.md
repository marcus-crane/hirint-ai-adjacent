## 职位描述

"负责 Agent Harness 与多 Agent 协作架构的设计与演进，并建设标准化开发管线，支撑 Agent 从研发、评测到上线的全流程。

主要职责：

1. 设计并演进 Agent Harness（运行时编排、工具调用、上下文与执行环境），支撑编码、搜索等 Agent 的后训练与应用落地。

2. 设计 Agent-to-Agent（A2A） 协作与任务分解方案，满足生产环境的可靠性、安全与可观测性。

整合现有 Agent 架构与训练/评测流程，建设 标准化开发管线（SDK、规范、CI/评测），加速 Agent 产品迭代。

## 任职要求

基本要求：

1. 硕士及以上学历，2 年以上 Agent 相关研发经验，主导或深度参与过通用 Agent 或 Agent 平台/Harness 的设计与实现。

2. 熟悉 Agent 构建全流程：数据 → 中训练/CPT/后训练 → 评测 → 部署中的至少两个环节，并有实战经验。

对 Agent RL、Runtime、Multi-Agent、Tool Use 等方向有持续跟进，能支撑架构选型与技术判断。

加分项：

3. 在大模型智能体方向有 顶会或期刊论文（一作、共一或实质性核心贡献）

4. 作为 核心开发者或 Maintainer 参与过 Agent 相关开源项目（Runtime、协议、编排框架、训练/评测工具链等）

参与过 Agent 产品研发或平台化建设，有从原型验证到上线迭代的完整经验；具备 生产环境部署与稳定性 实践者优先"
