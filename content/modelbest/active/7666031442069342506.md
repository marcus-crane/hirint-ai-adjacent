## 职位描述

1. 系统架构与技术方案 针对企业级场景，设计“模型-平台-软硬件”一体化的智能体系统方案；定义 Multi-Agent 协作、RAG 检索、Tool Calling 及端云协同/软硬集成架构。

2. 全栈系统交付与工程落地 主导 Agent 系统从 POC 到私有化/边缘算力设备（一体机）部署的全流程；调优 Prompt 与工作流，管控模型幻觉、吞吐时延及数据主权/零信任安全等系统级挑战。

3. 跨域技术对接与推进 高效打通业务端、算法团队与硬件/工程团队，将复杂需求抽象为系统开发任务，把控软硬件联调与交付里程碑。

4. 前沿架构与生态追踪 跟踪大模型、Agent 平台架构及异构算力发展，评估系统落地可行性。

## 任职要求

1. 专业与经验：计算机、软件工程等专业；2 年以上 AI / 大模型 / 软硬一体系统项目交付经验，有完整企业级智能体系统落地案例。

2. 技术功底：

- 深入理解 LLM 局限与 Agent 架构（规划、记忆、工具调用、Multi-Agent）。

- 熟悉主流 Agent 平台/框架及私有化/边缘端部署。

- 了解软硬协同部署、异构算力适配或端云联动逻辑优先。

3. 综合能力：具备极强的系统性思维、技术方案撰写与软硬件跨团队协作能力，适应高复杂度交付环境。
