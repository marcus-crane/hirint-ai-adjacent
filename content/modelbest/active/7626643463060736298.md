## 职位描述

1. 负责端侧 LLM / ASR / TTS 等各类模型的推理引擎的选型、适配与性能优化

2. 持续优化推理速度：量化（INT4/INT8）、KV Cache 优化、speculative decoding、FlashAttention 等

3. 适配硬件平台的异构算力（CPU / GPU / NPU），设计最优的推理调度策略

4. 搭建端侧模型评估体系：精度、延迟、功耗、内存占用的综合 benchmark

5. 跟踪开源端侧各类开源模型进展，持续引入最优模型

6. 优化语音链路端到端延迟（从用户说完话到助理开始回复 < 目标值）

7. 与硬件团队协作，针对目标芯片平台进行算子级优化

## 任职要求

1. 3 年以上模型推理优化经验，有端侧 / 嵌入式设备部署实战经历

2. 精通模型量化、剪枝、蒸馏等压缩技术

3. 熟悉至少一种推理框架：llama.cpp / ONNX Runtime / TensorRT / SGLang

4. 熟悉 C/C++，能做算子级性能调优

5. 理解 NPU / GPU 硬件架构，有异构计算调度经验

加分项：

有语音模型（Whisper / Sherpa / Piper / Kokoro）的端侧部署经验

熟悉 ARM 平台或 Qualcomm / RK3588 / Jetson 等边缘芯片
