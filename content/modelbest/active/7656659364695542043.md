## 职位描述

1. 负责企业级智能体平台的整体技术架构设计与演进。

2. 主导智能体核心能力与配套平台的架构设计，持续提升平台的通用性与可扩展性。

3. 负责平台在低延迟交互与高并发服务等不同场景下的性能与稳定性架构设计。

4. 负责多租户、私有化部署、安全与合规架构（数据不出企业，满足企业级合规要求）。

5. 主导智能体反馈闭环与持续迭代机制的工程实现，支撑智能体上线后持续优化。

6. 负责技术选型、架构规范与评审，带领工程团队保障交付质量。

## 任职要求

任职要求

1. 扎实的分布式系统、高并发、大规模服务架构经验，有大型系统从设计到落地的完整经历。

2. 有 LLM 应用架构经验：Agent、RAG、向量检索、工作流编排、记忆系统等。

3. 熟悉云原生与私有化交付（Kubernetes / Helm / Docker），有 To B 交付经验。

4. 能针对不同业务场景在性能、成本、稳定性之间做合理的架构权衡。

5. 扎实的编程功底（Python / Go / Java / TypeScript 之一或多），有良好工程规范与团队带领能力。

加分项

1. 有 Agent 框架 / 智能体平台 / LLMOps 建设经验。

2. 有开源项目经验。

3. 有大型企业级客户的私有化合规交付经验。

4. 有模型推理优化、向量检索优化、低延迟服务优化经验。
