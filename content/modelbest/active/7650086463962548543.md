## 职位描述

"负责构建和维护可扩展的多场景 Agent 沙盒环境，并建设 统一托管与调度平台。

主要职责：

1. 构建和维护可扩展的 Agent 沙盒环境，覆盖代码执行、命令行、浏览器/GUI、隔离 workspace 等形态，支撑评估、测试及后续业务扩展。

2. 建设 Agent 沙盒统一托管与调度平台，提供创建、调度、回收与基础监管能力。

3. 与 Agent 框架及业务方对接，沉淀环境模板、API 与使用规范，推动沙盒能力在团队内落地。

## 任职要求

1. 本科及以上学历，2 年以上 Agent 或 LLM 应用相关研发经验，做过需要执行环境 / 工具调用的 Agent 系统。

2. 有沙盒或隔离执行环境的落地经验（容器、远程执行、浏览器自动化等），能独立排查环境与稳定性问题。

3. 具备平台工程或基础设施开发经验，能负责沙盒托管平台的设计与实现（API、调度、环境模板、资源与权限管理等），支撑多业务统一接入与规模化使用。

加分项：

1. 参与过 Agent 执行评测、rollout 批跑或类似大规模环境调度场景。

2. 有多形态沙盒（代码 + 浏览器/GUI 等）的统一抽象或落地经验。

3. 建设或深度使用过内部环境托管平台（调度、配额、控制台或 SDK）。

4. 熟悉常见远程执行 / 沙盒方案，能借鉴并做内部适配。"
