## 职位描述

1. 负责具身数据采集设备的整体设计与落地:遥操作 / 主从系统、数据手套、力反馈、多相机 / 深度 / 触觉传感方案、动捕与本体等,从需求定义、方案选型到工程实现全流程负责。

2. 主导采集设备从原型到量产的全过程:结构设计、选型打样、装配调试、成本控制与批量制造,保证多台设备之间的一致性与可复制性。

3. 攻克采集端核心技术难点:多传感器时间同步与标定、动作精度与时延、机械可靠性与耐用度、人机工学与采集效率。

4. 与算法、数据、采集团队紧密协作,基于模型训练需求反推硬件规格,持续迭代设备形态,提升单位时间、单位成本下的有效数据产出。

5. 建立硬件相关的标准化流程:标定规范、质检标准、故障排查与维护体系,支撑规模化采集运营。

## 任职要求

1. 本科及以上,机械 / 机器人 / 自动化 / 电子等相关专业,5 年以上硬件研发或设备设计经验。

2. 精通机械结构设计与工程实现,有从原型到量产的完整落地经验(而非仅停留在打样阶段)。

3. 熟悉机器人本体、传感器(视觉 / 力 / 触觉 / IMU 等)选型与集成,理解多传感器同步与标定。

4. 具备成本、良率、可制造性(DFM)意识,能在性能与规模化制造之间做合理权衡。

5. 能把模糊的采集需求转化为清晰的硬件规格与工程方案,并推动跨团队落地。

加分项

- 有遥操作 / 主从系统、数据手套、力反馈设备或动捕系统的设计经验。

- 有机器人整机 / 消费级硬件量产经验,熟悉供应链与制造工艺。

- 对具身数据采集、机器人操作策略有理解,能从数据 / 模型视角优化硬件。

- 有从零搭建硬件团队或产线的经验。
