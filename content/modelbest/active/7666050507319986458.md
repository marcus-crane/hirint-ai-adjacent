## 职位描述

1. 负责企业身份认证（SSO）、IAM 及权限治理体系建设与运营，推进统一身份认证、账号生命周期及权限治理。

2. 负责办公网络、机房、VPN、无线网络及企业 IT 基础设施的规划、建设、运维与持续优化。

3. 推动企业 IT 平台、协同办公平台及业务系统集成，负责终端设备、账号、资产等生命周期管理，提升数字化办公效率。

4. 推进终端统一管理（MDM）、IT 自动化运维及终端安全能力建设，提升设备合规性与运维效率。

5. 建立 IT 运维标准、监控体系及应急响应机制，负责 IT 服务运营、供应商协同及持续优化，保障企业基础设施稳定运行。

## 任职要求

1. 3 年以上企业 IT 基础设施、数字化办公或企业 IT 运维相关经验。

2. 熟悉 Linux 系统管理，具备 Shell、Python 等自动化开发能力。

3. 熟悉 TCP/IP、BGP、OSPF、交换、路由、VPN、WiFi、802.1X、Portal 等企业网络技术。

4. 熟悉 SAML、OIDC、OAuth2 等身份认证协议，具备 SSO/IAM 建设或运维经验。

5. 熟悉 Windows、macOS 终端管理及 MDM，了解终端安全和设备生命周期管理。

6. 具备良好的问题分析、项目推进、跨团队沟通及文档编写能力。

加分项

1. 有 IAM、零信任、身份治理项目经验。

2. 熟悉 Docker、Kubernetes、Ansible 等自动化运维工具。

3. 有飞书、Microsoft 365、HR、OA 等企业平台集成经验。

4. 熟悉公有云（AWS、阿里云、华为云等）基础设施及 IAM 对接。

5. 熟悉 EDR、终端安全、软件正版化及企业 IT 合规建设。
