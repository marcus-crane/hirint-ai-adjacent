## 工作内容

工作职责

1. 围绕智能驾驶端到端方向，参与智驾虚拟仿真系统的设计、搭建和子系统研发，支撑智驾算法评测， 参与仿真3D场景重建、传感器仿真、场景渲染 & Unreal工程等子系统的方案设计和开发工作

2. 负责 / 参与仿真3D场景、传感器仿真、场景渲染等模块的仿真度优化， 以持续提升仿真系统的置信度

## 任职要求

职位要求

1. 硕士以上 / 优秀本科学历，3年以上工作经验，熟练掌握C/C++语言，深入理解面向对象原则，有较强的Linux平台C++编程能力

2. 有良好的计算机图形学基础，对几何处理、线性代数有扎实掌握，熟悉至少一种图形 API 、熟悉GLSL/HLSL等至少一种Shader语言，熟悉至少一种开源 / 商业/自研 图形引擎

3. 有丰富的自动驾驶虚拟仿真系统研发经验，具备以下任一方向 / 多个子方向的研发经验：

（1）高真实感仿真3D场景重建：仿真场景程序化重建，支持仿真道路环境规模化生成

（2）传感器仿真：负责虚拟仿真系统中Camera、Lidar、Radar、IMU等传感器仿真，包括传感器内参、外参、缺陷和物理特性模拟

（3）场景渲染、Unreal工程：负责虚拟仿真系统工程搭建、场景渲染真实感和性能优化等

4. 对技术有热情，具备良好的团队合作能力，积极主动，务实自驱

5. 同时熟悉3D GS、NeRF等Neural Rendering技术、有Neural Rendering与图形方案结合经验者优先

6. 熟悉自动驾驶业务、有自动驾驶仿真开发经验、熟悉CARLA、Simulink等其他仿真工程/ 软件者优先
