## 工作内容

1.负责各独立系统之间的打通与集成（如：飞书审批流和金蝶凭证对接、飞书审批流与每刻报销对接、每刻报销与金蝶凭证对接等）。编写高质量的集成接口或中间件代码，解决企业内部信息孤岛问题。

2.负责系统上线后的日常运维、故障排查、权限管理及数据备份，编制系统使用手册并对内部员工进行培训。

3.协助IT Leader推进公司内部信息化系统的整体架构设计与落地，推进金蝶ERP、飞书、每刻报销等核心系统部署上线。制定IT系统迭代路线图，确保架构具备高可用性与可扩展性。

4.深入各业务部门（财务、HR、供应链、行政等），通过高效沟通理顺业务痛点，将业务语言转化为清晰的技术需求文档。优化和重构现有业务流程，确保IT系统设计紧密贴合业务实际运作。

## 任职要求

1.计算机、软件工程、信息管理等相关专业本科及以上学历。3年以上企业内部信息化系统开发、集成或架构经验。

2.扎实的编程功底，熟练掌握至少一种主流开发语言（如Python / Java / Go / Node.js等），具备实际的API接口开发与对接经验。

3.熟悉主流数据库（如MySQL、SQL Server等）及SQL编写与优化。

4.熟练掌握常见的数据交换格式（JSON、XML）及通信协议（RESTful API、Webhook）。熟悉飞书aPaaS平台。
