## 工作内容

MCU 端

- 负责可穿戴设备端 MCU 固件开发，包括多路传感器数据采集、信号处理与无线传输；

- 低功耗设计与电源管理策略实现；

- 设计并实现多设备间的无线时间同步方案，保证多节点数据时间对齐；

- 与硬件工程师协作完成板级调试与外设驱动开发；

Linux 端

- 基于 Rockchip / Allwinner 平台进行 BSP 适配与驱动开发（Camera、I2C/SPI 外设等）；

- 开发 Linux 应用层程序，完成数据采集、处理与传输链路；

- 参与设备树配置、内核裁剪与系统优化；

跨层协作

- 定义 MCU 与 Linux 主机之间的通信协议（USB/UART/SPI）；

- 参与硬件原理图评审，从软件角度提出引脚分配、时序与接口设计建议；

- 协助搭建自动化测试与固件升级（OTA）流程；

- 参与产品从原型到量产的工程化落地，输出生产测试方案与烧录工具；

## 任职要求

- 扎实的 C 语言功底，熟悉嵌入式开发全流程

- 有 MCU 开发经验（STM32/ESP32 或同级别平台），能独立完成固件开发与调试

- 有 Linux 驱动开发经验，理解字符设备、Platform 驱动模型、设备树

- 能编写 Linux 应用层程序（Socket、多线程、进程间通信）

- 了解常见的无线时间同步方案（PTP/gPTP、PPS、CSI Timestamp 对齐等）

- 能看懂原理图，能参与硬件评审并提出有效建议

- 有项目从原型到量产的实际工程化落地经验，了解产测、烧录、良率相关流程

- 熟练使用示波器、逻辑分析仪等调试工具

加分项

- 有 Rockchip 或 Allwinner 平台的实际开发经验

- 熟悉 V4L2 / Camera 驱动子系统

- 有无线通信经验（BLE / Wi-Fi）

- 有音视频采集链路或 MIPI CSI 相关经验

- 了解 RTOS（FreeRTOS / Zephyr）
