## 工作内容

1. 负责系统的持续集成（CI）与持续交付（CD）流程搭建与优化，持续提升交付效率和系统稳定性。

2. 设计、实施并维护自动化运维体系，支持多云或混合云环境下的容器化部署（Docker/K8s），保障业务高可用与弹性扩展。

3. 监控和提升系统的安全、性能与可观测性，响应并解决生产环境中的突发问题和故障。

4. 持续探索并验证行业前沿的 DevOps 与 AI Ops 方案，推动最佳实践在团队落地。

5. 主动与开发、产品、运营等多团队沟通，保障“需求-开发-上线-运维”全流程的顺畅衔接。

## 任职要求

1. 本科及以上学历，计算机科学与技术、软件工程、信息管理或相关专业，3 年及以上相关工作经验。

2. 善于沟通协作，能够在多团队多项目并行环境下推动 DevOps 文化落地，具备持续学习和自我驱动的能力。

3. 熟悉 Linux / Docker / Kubernetes / CI/CD 等 DevOps 基础能力

4. 对 LLM / Agent / RAG 的基本原理和工程落地问题有实际理解或经验

加分项

1. 有 向量数据库（Milvus / Pinecone / Chroma） 运维或调优经验
