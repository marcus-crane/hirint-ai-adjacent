## 工作内容

1、负责公司产品后端系统的设计、开发、重构与持续优化；

2、负责业务逻辑实现、接口开发、数据库设计与数据处理链路建设；

3、参与基于消息中间件、缓存组件等相关功能的开发与优化，提升系统实时处理能力；

4、负责微服务架构下的服务治理、性能优化、故障排查与线上稳定性保障；

5、编写高质量、可维护的代码，推动后端基础能力组件化、标准化；

6、与产品、算法、测试、前端、运维团队合作，共同推动产品交付；

7、跟踪行业技术趋势，并持续优化系统架构与工程实践。

## 任职要求

1、本科及以上学历，3 年以上 Java 后端开发经验，具备扎实的 Java 基础（包含多线程、集合框架、JVM 原理等）。

2、熟练掌握 Spring Boot、Spring Cloud、Nacos、MyBatis-Plus 等技术栈，具备微服务架构的设计与开发经验。

3、熟悉 Kafka、MQ、Redis 等中间件的使用、配置与调优，能够解决中间件相关性能瓶颈。

4、熟悉 MySQL / TiDB / ClickHouse / Cassandra / Elasticsearch 等数据库或搜索引擎的应用及性能调优。

5、熟练使用 Git、Jenkins、Maven、JUnit 等 DevOps 工具，具备良好的工程化实践能力。

6、熟悉 Kubernetes 与 Docker，具备容器化与集群环境的实际应用经验。

7、熟悉常见数据传输协议及开放接口标准，具备丰富的 API / SDK 设计与开发经验。

8、有大数据量高并发场景下的性能调优经验者优先。

9、具备 Flink、Kafka Streams 等实时流式计算框架的开发经验者优先，熟悉事件流处理、规则计算或状态管理者更佳。

10、有 AI 产品相关系统、平台开发经验者优先。

11、具备良好的沟通表达能力与团队协作精神，极强的责任心，具备持续学习与自我驱动能力，能够在压力环境下保持稳定产出。
