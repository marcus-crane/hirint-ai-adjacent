## 工作内容

1.参与公司产品在客户项目中的 定制化开发与系统集成工作，根据项目需求进行功能开发与技术实现

2.基于 Java 技术栈（Spring Boot / Spring Cloud） 开发和维护后端服务，支撑项目交付与业务系统运行

3.设计和开发 RESTful API / 微服务接口，完成与客户系统或第三方系统的对接与集成

4.参与 Web 前端开发（Vue / React），实现项目管理后台或业务系统界面

5.配合项目团队完成 系统部署、测试支持及上线工作，保障项目顺利交付

6.参与现场问题排查、系统调优及技术支持，保障系统稳定运行

7.编写技术文档，包括 接口文档、部署文档及开发文档

8.与售前、架构师、交付工程师及客户技术团队协作，推动项目需求落地

## 任职要求

1.5年以上软件开发经验，有项目交付或定制化系统集成开发经验优先

2.熟练使用 Java 进行后端开发，熟悉 Spring Boot / Spring MVC 等主流开发框架

3.熟悉 HTTP / RESTful API 设计与开发

4.熟悉 Docker / Kubernetes 容器化，且具备部署与运维经验

5.熟悉常见中间件，如MySQL、Cassandra、Kafka 等

6.熟悉至少一种前端框架Vue 或 React

7.熟练使用 Git 等版本管理工具

8.具备良好的 沟通能力和团队协作能力，能够参与项目需求讨论和技术方案实现

- AI 提效与工具应用：

AI 协同开发： 熟练使用 AI 辅助编程工具（如 GitHub Copilot, Cursor 或 ChatGPT/Claude）进行代码编写、重构及 Bug 排查，具备利用 AI 提升研发效率的实战经验。

LLM 落地探索： 了解大语言模型（LLM）生态，有 RAG（检索增强生成） 开发经验、LangChain / LlamaIndex 使用经验或接入主流 AI 模型 API 经验者大加分。
