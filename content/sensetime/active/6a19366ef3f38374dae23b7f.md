## 工作内容

1. 设计、实施并维护自动化运维体系，支持多云或混合云环境下的容器化部署（Docker/K8s），保障业务高可用与弹性扩展。

2. 监控和提升系统的安全、性能与可观测性，响应并解决生产环境中的突发问题和故障。

3. 主动与开发、产品、运营等多团队沟通，保障“需求-开发-上线-运维”全流程的顺畅衔接。

## 任职要求

1. 本科及以上学历，计算机科学与技术或相关专业，5年以上工作经验

2. 熟悉 Linux / Docker / Kubernetes 等基础操作

3. 有 Minio / PostgreSQL 等基础设施部署和维护经验

4. 对 AI Agent、RAG有基本了解，有Langfuse、Milvus等组件部署和维护经验
