## 工作内容

1. 负责大模型推理集群的架构设计与运维体系建设，包括 GPU 集群管理、资源调度、稳定性保障等

2. 负责多云环境下 Kubernetes 集群的部署、管理与自动化运维，支持跨云高可用架构

3. 负责Kubernetes 相关组件或插件开发，包括但不限于Scheduler/Operator/自定义调度策略等

4. 设计并建设AI 推理平台的基础设施能力（AI IaaS），包括资源管理、任务调度、容量管理、自动扩缩容等

5. 负责大规模 GPU 推理集群的稳定性保障与性能优化，包括 GPU 利用率、网络性能、存储性能等

6. 构建自动化运维体系，提升集群部署、升级、故障恢复、扩容等流程的自动化程度

7. 建设完善的监控、告警与故障定位体系，提升 AI 推理服务 SLA

8. 带领 运维开发团队，推进技术方案落地，并持续优化平台工程能力

## 任职要求

1. 计算机相关专业本科及以上学历，5 年以上相关工作经验

2. 深入理解Kubernetes 架构与生态，具备生产级 Kubernetes 集群运维经验

3. 必须具备 Kubernetes 插件开发经验，包括但不限于Scheduler/Operator/自定义调度策略等

4. 具备多云环境 Kubernetes 集群管理经验

5. 熟悉云原生基础设施组件，如containerd / Docker/Prometheus / Grafana/Helm/ArgoCD等

6. 熟练使用Go / Python 等语言进行平台或运维工具开发

7. 具备良好的系统架构能力和问题排查能力

8. 具备团队技术带领经验，有较强的技术推动力
