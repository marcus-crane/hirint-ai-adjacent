## 工作内容

负责或参与以下产品的战略与规划、需求挖掘与定义、产品设计、技术协同与交付、全生命周期管理、效果评估与迭代，

1. AI Observability 产品

2. MLOps 平台产品

3. 数据指标体系与 BI 可视化产品

4. AI 算力池产品

5. API产品

## 任职要求

1. 3年以上 toB / 平台型 产品管理经验

2. 熟悉云原生 / Kubernetes 生态，并至少在以下方向之一有较深入理解或实践经验：

· API / AI 算力平台 / 异构算力 / 调度与资源管理

· Observability（Metrics / Logs / Tracing）

· MLOps / 模型训练与推理平台

3. 理解模型从 开发 → 训练 → 部署 → 推理 → 运维 的基本流程，能与工程和算法团队高效沟通

4. 具备良好的结构化思维、沟通能力和跨团队协作能力，能推动复杂产品落地

5. 能接受一定频率的客户沟通或市内公出，具备 ToB 产品的耐心与抗压能力

加分项

1. 有 AI 算力池、公有云 / 私有云平台或科研机构客户支持经验

2. 熟悉 Kubeflow / Ray Serve / vLLM / Airflow 等平台组件之一

3. 有 SRE、后端或平台工程背景

4. 有多集群、多地域平台管理经验
