## 工作内容

1. 负责大模型（LLM/VLM）在车载AI芯片、PC等边缘设备的量化部署、推理加速；

2. 完成模型量化压缩（INT8/INT4、GPTQ/AWQ 等）、格式转换（ONNX/TFLite 等）及推理引擎集成；

3. 针对目标硬件平台，优化推理性能，满足延迟、吞吐量、算力消耗、内存带宽等指标要求；

4. 参与端侧推理引擎架构设计、SDK工程化开发及大模型微调、量化部署相关工具链开发；

5. 参与客户量产项目交付，保障大模型端侧推理性能及软件质量满足客户要求；

## 任职要求

1. 计算机 / 电子工程 / AI 相关专业本科及以上，2年以上端侧 AI 推理部署经验；

2. 扎实的 C/C++ 和 Python 编程能力；

3. 熟悉Nvidia/高通/MTK/Intel/AMD 等主流AI推理芯片，熟悉CUDA/TensorRT、QNN、NeuroPilot、OpenVINO、ROCm等；

4. 熟悉PTQ/QAT 模型量化原理，掌握 GPTQ、AWQ、SmoothQuant 之一；

5. 熟练使用两种以上推理框架（ONNX Runtime / TFLite / NCNN / MNN / OpenVINO等）;

6. 深入理解Transformer 架构，熟练掌握KV Cache、GQA/MQA 等推理优化原理;

7. 具备端侧推理 benchmark 与性能瓶颈分析能力；

加分项：

1. 有多模态大模型（VLM）端侧部署及产品化落地经验；

2. 熟悉Eagle Draft Model / 投机采样（Speculative Decoding），有实践经验；

3. 有自定义算子开发经验（TensorRT Plugin、QNN Op Package、OpenVINO Custom Op 等）；

4. 熟悉Flash Attention / PagedAttention 等注意力优化技术；

5. 有开源社区贡献（llama.cpp、MLC-LLM、MNN、OpenVINO、Tensorrt-Edge-LLM 等）；
