## 工作内容

1、负责 AI 教育方向产品的全栈研发，包括前端、后端、数据库、部署与运维相关工作；

2、独立完成从需求理解、技术方案设计、系统架构、开发实现到上线交付的端到端 Web 项目开发；

3、负责 AI Agent 系统的设计与开发，包括多 Agent 协同、工具调用、工作流编排、RAG、记忆机制、任务规划等能力建设；

4、基于大模型能力构建可落地的 AI 应用，将 AI 深度融入产品核心流程，而非简单功能外挂；

5、负责 AI 应用的工程化建设，包括稳定性、性能、可观测性、权限、安全、测试与 CI/CD

6、参与产品架构演进，推动系统模块化、服务化与长期可维护性建设；

7、研究并引入前沿 AI Agent 技术、开发框架与工程实践，持续优化研发效率与产品体验；

8、与产品、算法、教学团队协作，将教育场景需求转化为高质量技术实现。

## 任职要求

1、具备扎实的全栈开发能力，能够独立完成一个具备前后端、数据库与部署能力的 Web 产品交付；

2、深入理解现代 Web 技术体系与传统互联网架构，包括但不限于：

- 前端工程化、状态管理、SSR/CSR

- 后端服务设计、数据库建模、缓存、消息队列

- API 设计、权限体系、微服务/模块化架构

- 云原生、容器化、CI/CD、监控与日志体系

3、对“古法编程”有深入理解，具备扎实的软件工程基础，而不仅依赖 AI 生成代码；

4、具备 AI Agent 或 LLM 应用开发经验，理解 Agent 的规划、推理、工具调用、上下文管理与工作流机制；

5、熟悉至少一种主流 Agent 或 AI 应用框架，如 LangChain、LangGraph、AutoGen、OpenAI Agent SDK、CrewAI 等；

6、熟悉 RAG、向量数据库、Prompt Engineering、模型调用与 AI 应用工程化实践；

7、熟悉 TypeScript / Python 等至少一种主力开发语言，具备现代前后端技术栈开发经验；

8、具备较强的问题分析与系统设计能力，能够在需求不明确的情况下主动推进项目落地；

9、对 AI 产品有热情，关注 Agent、AI IDE、自动化研发等方向的最新进展。
