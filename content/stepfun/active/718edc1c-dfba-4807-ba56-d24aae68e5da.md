**岗位职责**

1. 与算法、数据、工程团队紧密协作，负责 Agent 相关产品规划、模型能力建设与版本迭代，推动模型及agent关键能力模块（工具调用、记忆/上下文、多智能体协作等）落地并持续优化。
2. 持续跟踪多模态、LLM 、RAG与 Agent use 技术与产品趋势，结合客户需求反馈，定期开展对标分析与竞争力复盘，推动模型能力升级及产品的创新，支撑商业化落地并达成业务目标。
3. 定义数据与效果理想态，深度参与数据采集/标注、训练、评估、实验与上线全流程，推动训练-评测-迭代闭环，提高模型迭代效率与效果稳定性。
4. 从用户视角定义全新的人机交互体验，并细化到对模型效果的理想态定义和评测体系的构建；持续迭代/完善评估体系，包括但不限于评估方法论、标准、Benchmark与指标口径等，对模型效果与关键业务指标负责。
5. 建立 badcase 机制与问题归因框架，推动问题定位、方案设计、验证上线与线上监控闭环，持续提升模型可用性、稳定性与可控性。

**﻿岗位要求**

1. 本科及以上学历，计算机、软件工程、人工智能等相关专业优先。
2. 2年以上 AI 产品经验，至少1年 LLM / 多模态 / Agent 相关产品或评测经验，优秀的项目管理能力。
3. 具备扎实的技术理解力，熟悉 LLM / 多模态/MCP/multi-agent / GUI Agent 的基本原理与业界进展，理解 agent 基本模块（工具调用、记忆/上下文等）；愿意持续阅读论文跟直接体验海内外产品，并将使用体验转化为评测与产品改进思路。
4. 数据驱动意识与逻辑能力强，具备基础数据分析能力与指标体系搭建能力，能通过 badcase 洞察模型本质问题，并提出优化方案。
5. 熟练运用 Prompt Engineering，能将业务流程抽象为模型可理解、可执行的指导性原则。
6. 具备良好的客户沟通与跨团队协作能力，能深入理解需求并输出解决方案，具备强交付意识，适应快节奏工作方式。
