你将参与构建

* Agent 训推框架维护：负责 Agent 训练与推理全链路框架的开发、优化与稳定性保障，支撑大规模模型训练实验的高效运转
* Agent 沙箱环境稳定性：设计并维护 Agent 执行沙箱，保障代码执行、工具调用等复杂场景下的安全性、隔离性与可靠性
* Agent 实验平台：构建支撑 Post-Train 团队日常实验的基础平台，涵盖实验管理、数据流水线、评测对比等核心能力

我们在寻找这样的你：

* 独立思考，不随波逐流
* 你有自己的判断，不迷信权威，不满足于"跟着做"。你知道什么是对的，也敢于说出来——哪怕结论之后会被推翻。
* 有工程项目维护经验，对质量有执念
* 你维护过真实跑在生产或实验环境里的项目，踩过坑、修过 bug、做过重构。你在意代码是否可维护、接口是否合理、系统是否可观测——不只是跑通了就行。
* 真正理解系统，而不是背答案
* 你不靠记忆语法，靠理解原理解决问题。能从一个 Agent 调用链路讲到调度、容器、网络、存储，并在任意层面展开。对分布式训练、推理优化、系统瓶颈有第一手认知。
* 把 AI 用到极致
* 你不只用 AI 补全代码——你用它分解任务、加速实验、验收产出。你清楚各主流模型的能力边界，知道在哪里信任 AI、在哪里必须自己把关。
