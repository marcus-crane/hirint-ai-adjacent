岗位职责

* 设计并实现基于大语言模型（LLM）的多模态 Agent 系统架构，支持多轮对话、工具调用、任务规划与执行；
* 集成外部工具、API、Agent Skills（搜索、数据库、代码执行等），扩展 Agent 能力边界；
* 设计并实现 Agent 行为监控、效果评估与对齐机制，保障系统输出的稳定性、安全性与可控性；
* 优化 Agent 系统的推理性能与并发处理能力，通过缓存策略、批处理、模型压缩与量化等技术手段提升效率并降低成本；
* 与算法训练团队协作，系统化提升 Agent 成功率、工具调用准确率与任务完成闭环能力；

任职要求

* 计算机 / AI / 软件工程相关专业，本科及以上学历，3 年以上工程经验；
* 精通 Python / Go / Rust / TypeScript 至少一门语言，具备复杂系统设计与工程落地能力；
* 深入理解大模型训练与推理机制，熟悉 LLM 后训练（Post-training）体系；
* 熟悉 Claude Agent SDK / OpenAI Agent / LangGraph / AutoGen / CrewAI 等框架优先；
* 对 AI 技术有强烈热情，具备快速学习前沿技术的能力；
