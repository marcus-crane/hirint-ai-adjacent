**岗位职责**

1. **生态合作与资源对接**

* 负责 AI 大模型供应商的资源对接、商务洽谈与合作落地，维护长期合作关系
* 跟踪模型版本、定价、配额、SLA 等关键信息，持续优化供应商组合

1. **渠道拓展与销售支持**

* 配合销售团队拓展外部 B 端渠道，输出合作方案与商务素材
* 协助完成客户需求对接、方案沟通与转化跟进，沉淀行业案例

1. **用户运营与活动策划**

* 策划并执行面向开发者及企业客户的运营活动（新品首发、激励计划、技术沙龙等）
* 把控活动全流程，关注注册、调用量、留存、付费转化等核心指标
* 配合市场与内容团队完成品牌曝光与开发者社区运营

1. **技术文档与开发者体验**

* 梳理并撰写 API 接入文档、SDK 指南、最佳实践与 FAQ
* 与产品、研发协同，保障文档与产品迭代同步，持续优化开发者接入体验

**任职要求**

* 本科及以上学历，3 年以上互联网 / SaaS / 云服务 / 开发者平台相关运营经验
* 熟悉 AI 大模型行业生态，对 API 产品、开发者工具链有基本认知
* 具备技术理解力，能读懂 API 文档与基础代码示例（Python / cURL）
* 良好的中英文书面表达能力，能独立产出对外沟通材料与技术文档
* 较强的商务沟通、跨团队协作与项目推进能力
* 结果导向，具备数据分析意识，能基于运营数据驱动决策

**加分项**

* 有 AI / 大模型 / 云厂商 / API 平台 相关运营或 BD 经验
* 有面向开发者的内容运营、技术写作或 DevRel 经验
* 熟悉主流模型供应商生态（OpenAI、Anthropic、Google、国内主流厂商等）
