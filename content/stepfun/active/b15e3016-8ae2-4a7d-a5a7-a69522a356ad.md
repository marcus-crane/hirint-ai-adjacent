岗位职责：

1、负责 Agentic RL 训练基础设施建设，覆盖 Sandbox / Environment Runtime、Scaffold、多阶段 RL Workflow、Eval 与 Replay 等核心链路；

2、支持大规模 Environment 生产、数据回流与训练闭环，持续优化系统稳定性、吞吐与可扩展性。

3、需要与算法、数据、Infra 团队深度协同，推动复杂 Agent 任务训练体系工程化落地。

任职要求：

1、要求具备扎实工程能力，熟悉 Python / Go 等至少一种语言；

2、具备分布式系统或训练平台经验；

3、理解 RLHF、Agentic RL、Tool Use、Environment、Replay 等核心机制；

4、有后训练、Agent、Sandbox 或大规模数据循环相关经验优先；

5、希望你既能理解算法需求，也能抽象通用系统能力，推动训练基础设施持续演进。
