岗位职责

1. 负责云资源管理系统的架构设计与建设（ECS、网络、存储等）

2. 设计支持海量资源（十万级节点）的管理与调度体系

3. 主导系统从 0→1 的落地：架构设计、核心开发、上线与持续演进

4. 构建高可用运维体系（发布、变更、监控、故障处理、自动化）

岗位要求

1. 基础扎实，熟悉操作系统、网络、分布式系统等

2. 熟悉 Go / Python / Java 至少一门，具备较强工程能力

3. 有大规模系统设计经验（云平台 / 集群管理 / 调度系统 / 运维平台）

4. 能独立完成系统设计 → 核心实现 → 上线落地

加分项

1. 熟悉云产品开发设计。

2. 有网关或流量体系开发设计经验

3. 有自动化运维大型系统的开发设计经验

4. 已经在熟练使用AI工具提升日常工作效率
