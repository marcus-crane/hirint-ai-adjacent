岗位亮点

AI 大模型独角兽的企业级 Agent 业务线，直接负责 ToB 产品从 0 到 1 的搭建。你将深入金融、连锁零售等行业一线，把大模型能力变成企业创造价值的"数字员工"。

岗位职责

1. 企业级 Agent 产品设计：深入金融投顾、连锁零售等行业场景，调研客户需求和业务流程，挖掘可被 Agent 自动化或增强的高价值环节，独立输出产品方案和 PRD

2. Skill 体系与工作流设计：基于 StepClaw 企业版平台，设计和搭建行业 Agent Skill（如投研辅助、合规审查、门店巡检、供应链调度等），定义 Workflow 逻辑，推动从 Demo 到企业生产环境的完整落地

3. 模型能力产品化：理解大模型 API 能力边界，针对不同企业场景设计最优的模型调用策略和 Prompt 工程方案；从 Badcase 中定位问题根因，驱动模型和工程侧快速迭代

4. 竞品与行业洞察：持续跟踪国内外企业级 AI Agent 产品动态（Claude for Enterprise、OpenAI GPTs for Business、Coze、Dify 等），输出有深度的竞品分析和非共识观点，反哺产品路线图

5. 客户共创与商业闭环：主导标杆客户（金融机构、头部连锁品牌）的需求对接和方案共创，跟进交付效果和客户反馈，推动产品从单点验证到行业可复制

岗位要求

1.本科及以上学历，2 年以上产品经理经验，有 AI / 大模型 / SaaS 产品背景优先

2.对 AI Agent、大模型应用有真实热情和持续关注，能说清楚当前 Agent 赛道的格局和机会

3.逻辑严谨，文档能力强，能独立完成从需求分析到 PRD 输出的全链路

4.抗压能力强，适应 ToB 业务的客户导向节奏和高不确定性环境，愿意深入行业一线

5.有良好的跨团队协作和向上沟通能力，能在产品、算法、工程之间高效推进

加分项

1.有 ToB / 企业级产品经验，参与过客户交付或解决方案设计

2.熟悉金融或连锁零售行业，了解其业务流程和核心痛点

3.有 Prompt Engineering 实践经验，或深度使用过 Claude Code / Cursor 等 AI 工具

4.有 Coze / Dify / n8n 等 Workflow 平台搭建经验

5.会写 Python 脚本，能用代码快速验证产品想法

你能获得什么

1.定义一条新产品线：团队早期核心成员，你的判断直接影响产品方向和路线图，不是执行层的螺丝钉

2.稀缺的行业纵深：深入金融投顾、连锁零售等万亿级行业，建立"AI + 行业"的复合壁垒，这在市场上极度稀缺

3.前沿技术资源：日常使用 StepClaw、Claude Code 等最前沿 AI 工具，海量 Token 额度随便用，技术资源充足

4.标杆客户背书：直接服务金融机构和头部连锁品牌，每个落地项目都是行业标杆案例

5.高密度成长环境：团队成员来自一线大厂，信息密度高，成长速度快
