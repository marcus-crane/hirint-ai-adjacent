岗位职责

* 负责 **AI 产品及 AI Agent / 大模型后台系统** 的整体架构设计、技术选型与核心开发，保障系统高性能、高可用与安全性。
* 主导 **AI 产品研发与工程化落地**，推动大模型能力在 ToB 业务场景中的产品化应用。
* 主导复杂业务需求的技术方案设计，平衡产品迭代效率与长期技术演进。
* **带领研发团队完成** 产品迭代与项目交付，持续提升团队技术能力与协作效率。
* 持续优化系统架构，攻关高并发推理、模型部署、多租户隔离、数据安全等关键技术问题。
* 协调 **算法、产品及业务团队**，推动跨团队协作与技术对齐。

任职要求

* 5 年以上后台开发经验，2 年以上团队管理或技术负责人经验。
* 本科及以上学历，计算机相关专业优先。
* 精通 **Golang / Java / Python** 至少一门语言，具备分布式系统设计经验。
* 熟悉微服务架构、容器化、Kubernetes、消息队列、缓存及 MySQL 等技术。
* 有 **AI 产品研发、大模型系统（Agent / RAG / 推理平台）** 经验者优先。
* 具备良好的跨团队沟通能力、责任心和业务理解能力。

加分项

* 有大模型在金融、政务、零售、消费电子等行业的落地经验。
* 熟悉 **终端设备、端侧大模型** 或端云协同架构。
* 对 **AI 推理系统**（如高并发推理、模型加速、部署优化等）有深入理解。
* 有大型系统架构设计或技术开源贡献经验。
* 熟悉主流云平台及云原生生态。
