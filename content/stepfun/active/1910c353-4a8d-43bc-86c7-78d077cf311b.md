职责：1. 负责手机 / 智能设备显示屏效果的调试与优化，包括但不限于亮度、色温、对比度、Gamma、HDR 等核心参数，提升屏幕色彩准确度、一致性及视觉表现力；2. 参与色彩管理体系的设计和优化，包括色域映射、色彩校准（sRGB、DCI-P3、Display P3、Rec.2020 等）3. 主导解决显示效果相关的售后问题（如偏色、亮度不均、色准偏差等），通过数据分析与实验验证定位根因并推动闭环；4. 承接显示屏效果相关的新技术、新功能预研（如低蓝光优化、动态刷新率适配、OLED 烧屏防护等），输出技术方案并推动项目落地；5. 制定显示效果调试标准与流程，联合硬件团队（屏体、背光）、驱动团队（DDIC、HWC）及测试团队，协同推进效果指标达成；6. 跟踪行业竞品显示效果特性，参与显示效果技术对标分析，支撑产品显示竞争力提升。要求：1. 本科及以上学历，光学工程、电子工程、计算机、自动化等相关专业；2. 2年以上消费电子（手机 / 平板 / 智能座舱等）显示效果调试经验，熟悉 LCD/OLED 屏幕的光学特性及效果原理；3. 熟练使用主流光学测量设备（如 CA310、CA410、Konica Minolta 光谱仪等），能独立完成显示效果参数校准与验证；4. 熟悉高通 / MTK 平台显示效果调试工具（如 QDCM、Miravision、DisplayTune 等），了解 Android 显示系统基本流程（如 SurfaceFlinger、HWC 工作机制）；5. 具备基础软件开发能力，熟悉 GIT 版本控制工具，能独立完成代码下载、提交及简单脚本编写；6. 有 HDR 效果调试、色彩管理算法（如 3A 校准）、OLED/LCD 屏体特性调优经验者优先；7. 具备良好的跨团队沟通能力，逻辑清晰，有较强的问题攻关与创新意识。加分项1. 有 HDR10/HDR10+、Dolby Vision 调试经验；2. 有 GPU 色彩管理（Gamut Mapping、Tone Mapping）优化经验；
