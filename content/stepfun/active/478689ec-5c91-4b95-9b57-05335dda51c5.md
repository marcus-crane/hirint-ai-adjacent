**核心职责**

**一、定价策略 & 计量计费产品**

• 主导 API 定价体系设计：按 Token 定价、套餐包、企业订阅等多种计费模型的产品方案

• 深入分析 OpenAI、Anthropic、Google、国内厂商的定价策略，结合阶跃模型能力制定有竞争力的定价方案

• 负责计量计费产品的需求定义：用量统计、账单明细、配额管理、欠费提醒等完整链路

• 设计定价实验机制，通过数据追踪定价调整对转化率、留存率、ARPU 的影响

**二、模型上架 & API 产品**

• 定义模型上架流程与标准：模型能力说明、接口规范、限速策略、SLA 承诺等产品规范

• 主导 API 文档、Playground、SDK 等开发者接入体验产品的规划与迭代

• 推动 OpenAI 兼容接口的产品对齐，降低开发者迁移成本，提升接入转化率

• 规划多模态 API 产品（图像、语音、视频）的上架路径与差异化定位

**三、平台 Infra 产品规划（与工程协作）**

• 对齐工程团队，将商业化需求转化为清晰的技术产品需求（PRD），推动限流、鉴权、计费等核心模块落地

• 参与 API 网关、计量系统等基础设施的产品评审，确保技术方案满足商业化目标

• 建立平台核心指标体系（API 调用量、付费转化率、ARPU、P99 延迟等），推动数据驱动决策

**任职要求**

**基础能力（必须）**

• 5 年以上产品经理经验，其中有 2 年以上 To B 平台类产品或 SaaS 产品经验

• 有完整的商业化产品经验：参与过定价设计、计费系统、或订阅增长产品的规划与落地

• 技术背景扎实：能读懂 API 文档、理解 REST / HTTP 协议基础，能与工程团队无障碍沟通技术方案

• 数据驱动：熟练使用 SQL 或 BI 工具做数据分析，能独立定义指标、拆解问题

• 极强的跨职能推动力，能在研发资源紧张的环境下高效排优先级并落地

**加分项（有则更好）**

• 有云厂商 API 平台或 AI 开放平台产品经验

• 了解大模型 API 的技术特性：Streaming 输出、Token 计费逻辑、上下文窗口限制、多模态接口差异

• 有开发者社区运营或开发者生态产品经验

• 有 0 到 1 搭建 SaaS 计费 / 订阅产品的完整经历

• 具备基础编程能力（Python / JS），能自己跑 API 验证产品方案
