岗位描述

负责模型与数据相关项目的全流程管理与推进，协同算法、数据、工程团队，在需求不明确、节奏快速变化的环境中，保障项目高质量交付，并推动数据驱动的模型效果优化。

工作职责

* 负责需求收集、拆解与优先级管理，制定项目计划并推动落地
* 跟进项目进度，识别并解决风险、依赖与资源冲突
* 协调算法 / 数据 / 工程团队，保障高效沟通与决策
* 参与数据标注项目设计（标注方案、规范、流程）及数据集构建（采集 / 清洗 / 标注 / 质检）
* 管理标注资源（内部 / 外包），控制质量、成本与产能
* 参与模型评测（评测集、评测流程、A/B test 等），分析结果并推动优化
* 建立数据 → 评测 → 模型迭代的闭环，持续提升模型效果

任职要求

* 本科及以上学历，3 年以上 TPM / 项目管理经验
* 有数据或模型相关项目经验，能适应快节奏、多依赖环境
* 具备良好的需求管理与项目推进能力

**具备以下经验者优先：**

* 数据标注项目经验，熟悉标注流程（文本 / 图像 / 多模态等）
* 熟悉数据集构建流程（采集、清洗、标注、质检）
* 有模型评测经验（benchmark / 人工评测 / A/B test）
* 理解数据质量控制方法（如一致性、抽检等）
* 理解模型迭代流程（数据 → 训练 → 评测 → 优化）

加分项

* 有 LLM / 多模态模型相关经验
* 有标注平台或评测平台经验
* 有标注团队管理经验
* 有数据驱动优化经验（不仅执行）
