最近我们在推进两个方向：

1）一篇金融 Agent 评测和benchmark建设

关注 LLM Trading Agent 的真实能力，不只看收益率，而是拆解收益到底来自市场、风格，还是选股 Alpha。我们希望用更严谨的 Benchmark，评估并推动模型在真实金融工作场景中的能力提升。

2）一组 7×24 小时工作的市场信息 Agent

持续收集和分析市场信息，覆盖风口挖掘、热点前瞻、主题选股、个股深研等场景。目标不是做一个 demo，而是让 Agent 真正参与到金融研究、信息处理和决策辅助的工作流里。

我们希望你：

1. 对 LLM Agent 有兴趣，最好有相关项目经验；

2. 对金融、投研、二级市场、量化或产业研究有基本理解；

3. 愿意一起探索如何提升模型在真实金融工作场景中的能力；

4. 能做信息收集、数据分析、Prompt / Workflow 设计，愿意快速试错；

5. 有模型训练、SFT / RL、数据构造、评测或模型迭代经验会加分；

6. 有工程实现能力、数据处理能力或金融研究经验会加分；

7. 同时有 LLM Agent、金融和模型训练经验的同学优先。

如果你想参与一个真正落地的金融 Agent 系统，从论文、Benchmark、数据、训练到产品一起做，欢迎来聊。
