岗位职责：

1、负责通用数据与 Agent 数据平台建设，覆盖数据采集、生产、清洗、评测与回流全链路；

2、搭建高稳定性 Pipeline、数据工具链与 Workflow，支撑算法训练、评测与实验迭代；

3、建设数据质量治理、资产管理与可观测体系，持续提升数据一致性、可追溯性与生产效率。

**任职要求：**

1、要求具备扎实工程能力，熟悉 Python / Go / Java 等至少一种语言，

2、具备分布式数据系统或大规模 Pipeline 经验；

3、能够理解算法训练与评测需求，并将其抽象为平台能力；

4、有大模型数据、Agent 数据、RL 数据或 AI Infra 相关经验优先。

5、希望你兼具工程能力与产品思维，能够与算法团队深度协同，推动数据体系工程化与平台化演进。

**加分项**

1、有大模型数据平台或 AI Data Infra 经验；

2、熟悉 Spark、Ray、Kafka、Airflow、ClickHouse 等技术栈；

3、有 Preference Data、Synthetic Data 或 Eval 数据经验；

4、有 Agent Workflow、Tool Use 或 RL 数据链路相关背景；

5、有从 0 到 1 搭建数据平台或数据治理体系经历。
