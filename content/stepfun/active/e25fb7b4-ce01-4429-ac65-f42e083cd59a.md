**岗位愿景**

我们正在构建大宗商品领域的"决策大脑"——将 LLM、深度学习时序预测与运筹优化深度融合，打造从信息感知 → 趋势预判 → 最优执行的端到端智能系统。如果你渴望让 AI 真正落地到万亿级产业，这里就是你的舞台。

**核心职责**

1. **多因子时序预测建模**

•构建覆盖全球宏观指标（汇率、利率、PMI）、行业基本面（库存、开工率、发运量）及替代品价格的多因子时序预测模型

•探索 Transformer、Mamba 等前沿架构在商品价格预测中的应用，持续迭代模型精度

1. **LLM 驱动的动态供需分析**

•利用大语言模型（LLM）自动提取研报、新闻、政策文本中的关键供需信号，实现非结构化信息的定量化

•开发 RAG + Agent 架构的自动化供需平衡表系统，实现数据采集、清洗、预测、报告的全流程自动化

1. **全局供应链优化**

•针对大宗商品贸易中的调拨、加工、多式联运（海/铁/公）场景进行数学建模

•优化物流成本与库存周转，解决复杂的路径规划与配额分配问题

1. **风险量化与智能套保**

•利用 Monte Carlo 模拟、VaR/CVaR 等手段量化价格波动风险

•结合强化学习探索动态套保策略，为业务部门提供最优下单与对冲方案

**任职要求**

**必备条件**

1. •学历：运筹学、应用数学、计算机科学、工业工程等相关专业硕士及以上学历
2. •算法能力：精通以下至少一个方向——时序预测（ARIMA/Prophet/Transformer-based）、组合优化（路径规划/排程/装箱）、或运筹建模
3. •工程能力：熟练使用 Python，掌握 PyTorch/TensorFlow 等深度学习框架；熟悉至少一种数学规划工具（Pyomo / PuLP / OR-Tools / Gurobi）
4. •LLM 素养：了解主流大模型（GPT / Claude / 开源 LLM）的能力边界，有 Prompt Engineering 或 Fine-tuning 实践经验

**加分项**

1. •有大型制造业（钢铁、化工、半导体）排产或供应链优化项目落地经验
2. •熟悉 RAG、Agent、Function Calling 等 LLM 应用范式
3. •有强化学习在工业优化或量化交易中的应用经验
4. •对大宗商品市场（黑色系、有色、能化等）有深入了解
