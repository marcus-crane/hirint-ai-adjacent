**岗位职责**

1. 参与沙盒化基础设施平台的设计、研发和持续演进，支撑多类业务场景下的安全运行环境需求。
2. 负责运行时、资源管理、任务调度、存储访问、网络隔离等平台核心能力建设。
3. 参与容器、轻量虚拟化、云原生存储等相关技术方案的调研、选型、工程实现和优化。
4. 持续提升平台在性能、稳定性、弹性、安全性和资源利用率方面的表现。
5. 参与大规模场景下的系统问题定位、性能分析、容量规划和成本优化。
6. 与业务、安全、基础架构等团队协作，推动基础设施能力在生产环境中稳定落地。

**任职要求**

1. 熟悉 Docker、Containerd、OCI、CRI、Kubernetes 等容器与云原生生态。
2. 熟悉 gVisor、Kata Containers、Firecracker、QEMU、KVM 等沙盒或虚拟化相关技术。
3. 熟悉 CSI、OverlayFS、镜像分层、快照、缓存、懒加载或分布式文件系统等存储技术。
4. 有大规模集群调度、资源编排、任务系统、控制面组件研发经验。
5. 有多租户隔离、安全沙盒、资源限制、审计、可观测性或故障恢复相关经验。
6. 有 AI Infra、Agent Runtime、云原生底层组件或基础设施平台建设经验。
7. 有开源项目贡献，或在操作系统、虚拟化、存储、云原生方向有较深入积累。

**加分项**

1. 熟悉 CSI，有 Kubernetes 存储插件、分布式文件系统、块存储或高性能文件系统研发经验。
2. 有沙盒平台、Serverless Runtime、轻量虚拟机、云桌面、远程开发环境、在线 IDE 或 Agent 执行环境建设经验。
3. 熟悉 Firecracker、Kata、gVisor 等安全容器或轻量虚拟化方案的生产实践。
4. 有大规模容器集群降本、弹性伸缩、资源超卖、冷启动优化经验。
5. 有多租户隔离、安全合规、PrivateLink、VPC 网络或企业级云服务接入经验。
