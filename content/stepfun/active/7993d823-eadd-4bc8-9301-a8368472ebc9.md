**岗位概述**

负责公司大模型服务平台面向企业客户的产品规划与落地，围绕 AI Coding 和 AI Working 两大场景，将模型路由、资源调度、Token 计量计费、预算管理等核心能力产品化，对接企业客户需求并输出解决方案，推动从售前到交付到持续运营的全流程闭环。

**岗位职责**

**-产品规划**

1. 负责大模型服务平台的产品规划与路线图设计，推动产品从内部基础设施向企业级服务演进
2. 设计企业级核心能力：商业化计费、多模型路由与降级策略、Token 计量计费、资源池隔离、租户、SLA 保障、KEY 管理与权限体系等
3. 设计产品的定价模型与商业化方案，支撑不同规模企业客户的采买与结算需求
4. 跟踪行业动态与竞品（火山方舟、阿里百炼、智谱开放平台等），持续优化产品竞争力

**-客户与解决方案**

1. 对接企业客户（集团内公司及外部企业），深入理解业务场景，输出定制化解决方案

2. 围绕 AI Coding 场景（Cursor / Claude Code / Codex 等）和 AI Working 场景（Claude Desktop / 办公 Agent 等），设计标准化接入方案与最佳实践

3. 主导客户技术方案宣讲、POC 验证、需求评审等售前工作

**-交付与运营**

1. 协调研发、运维、安全等团队完成客户交付，建立 oncall 支持机制，保障客户服务稳定性

2. 持续收集客户反馈，驱动产品迭代，形成"需求 → 方案 → 交付 → 反馈"闭环

**任职要求**

* 3–5 年 toB 产品经理经验，有完整的企业级产品从 0 到 1 或从 1 到 10 经历
* 熟悉大模型 / LLM 领域，了解主流模型（Claude、GPT、DeepSeek、GLM、Kimi 等）的能力与 API 调用方式
* 理解 API 网关相关概念：路由、限流、降级、负载均衡、计量计费等
* 有企业客户对接经验，能独立完成需求调研 → 方案设计 → 产品 PRD 全流程
* 较强的跨团队沟通能力，能与研发平等对话
* 优秀的文档与方案输出能力

**加分项**

* 有 PaaS / 云服务 / API 平台 / MaaS 产品经验（如火山方舟、阿里百炼、智谱开放平台等）
* 有 SaaS 计费体系设计经验（按量计费 / 席位订阅 / 混合模型）
* 了解 AI Coding 工具生态（Cursor / Claude Code / Copilot / Codex）
* 有 DevTools / 开发者工具方向产品经验
* 技术背景出身（计算机 / 软件工程相关），能读懂代码和接口文档
