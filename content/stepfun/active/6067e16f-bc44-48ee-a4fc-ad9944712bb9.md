岗位定位

负责 AI Coding 场景下的模型能力产品化，深入理解前沿大模型在代码生成、推理、理解等方面的能力边界，定义模型能力到编程产品的转化路径，推动模型持续进化以满足开发者真实需求。

主要职责

* 模型能力产品化 — 与研究团队协作，判断模型在编程场景的能力边界，定义哪些能力可产品化、哪些需继续投入，将用户痛点转化为模型优化目标
* 行业洞察 — 跟踪 AI Coding 竞争格局，前瞻性规划产品方向，探索编程之外的相邻场景拓展
* 数据回流与飞轮建设 — 设计 AI Coding 场景下的用户行为数据采集策略，定义 good case / bad case 的识别标准，推动高质量训练数据从产品侧持续回流至模型训练，形成「产品使用 → 数据沉淀 → 模型进化」的正向飞轮

任职要求

必须具备

* 3 年以上工作经验，其中至少 1 年工程师经历，优先工程师转产品。
* AI技术理解，熟悉大模型的训练、评测、推理等基本原理
* 日常深度使用 AI Coding 工具，对模型能力有 hands-on 体感
* 能在模糊场景中从矛盾反馈里提炼优先级，推动决策

优先考虑

* 做过模型评测体系设计或 AI benchmark 相关工作
* 有 Developer Tools、IDE、CLI 等开发者产品经验
* 能用 Claude Code 等 AI 工具做原型验证
* 有数据回流相关经验，了解用户行为数据采集、标注、清洗流程，或参与过训练数据飞轮建设
