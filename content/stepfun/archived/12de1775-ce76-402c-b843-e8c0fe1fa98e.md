岗位亮点

这不是传统的项目管理岗。你将作为企业级 AI Agent 项目的"交付中枢"，直面金融机构、头部连锁品牌等标杆客户，把大模型从 Demo 推进到企业生产环境。每一个你推动落地的项目，都可能成为整个行业的参照系。团队扁平，早期加入意味着你将定义交付体系本身。

岗位职责

-项目全周期管理：负责企业级 AI Agent 项目（StepClaw 企业版部署 / 行业 Agent 方案交付）的计划制定、进度把控、资源-协调和风险管控，确保项目高质量按时交付

-客户需求管理与期望对齐：主导客户需求对接和评审，准确识别需求范围，判断技术可行性和风险点，合理管理客户期望值，把"客户想要的"翻译成"团队能交付的"

-跨团队协调与推进：协调算法、工程、产品、销售等多职能团队资源，主动对齐目标，组织项目例会，跟踪 Action Item，识别并推动阻塞问题解决

-交付体系建设：搭建和持续优化 Agent 项目交付 SOP（方案和客户侧部署、Skill 适配、安全测试、验收上线等环节），沉淀-可复用的交付模板和知识库，让交付效率随项目积累持续提升

-项目回溯与质量改进：收集项目过程数据（工时、进度偏差、问题分布等），定期进行项目回溯总结，驱动流程和质量的持续改进

岗位要求

-本科及以上学历，2 年以上技术项目管理或 AI 项目交付经验

-具备系统性思维，能准确识别需求范围、技术风险，在复杂情况下做出有效决策

-抗压能力强，能在客户压力和多方需求中保持冷静，有韧性和推动力

-出色的沟通协调能力，能与客户高层、内部算法/工程/产品等不同层级和职能高效协作

-对 AI / 大模型技术有基本了解，能听懂技术讨论中的关键信息并做出项目层面的判断

加分项

-有 ToB 项目交付经验

-有金融、连锁零售、消费电子或政务行业的任职或客户服务履历

-熟悉敏捷开发流程，有规模化项目管理实战经验

-有基本的编程能力，日常使用 Claude Code / Cursor 等 AI 工具提效

-了解 Agent、Skill、Workflow、RAG 等大模型应用概念

你能获得什么

-搭建交付体系的机会：不是执行现有流程，而是从零定义企业级 AI Agent 的交付标准和方法论

-标杆项目履历：直接服务金融机构和头部连锁品牌的 AI Agent 落地，每个项目都是行业级案例

-稀缺的行业纵深：深入金融投顾、连锁零售等行业，积累"AI + 行业"的交付 Know-how，市场上极度稀缺

-前沿 AI 工具畅用：日常使用 StepClaw、Claude Code 等最前沿工具，海量 Token 额度不限量

-高密度成长环境：团队成员来自一线大厂，扁平文化，早期加入意味着更大的成长空间和话语权
