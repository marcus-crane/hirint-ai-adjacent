核心职责1.负责安卓设备内核层 CPU性能相关核心模块的优化，聚焦 CPU 调度、动态调频、IPC 通信及锁机制，提升系统性能响应与多任务处理能力2.调度器优化：优化Linux内核调度器（如EAS）算法，平衡性能与功耗，降低关键任务的延迟时间，提升用户体验。3.调频优化：设计并开发更智能的动态电压频率调节策略DVFS，优化调频算法，结合场景负载特征，实现频率精准调控4.锁优化：分析内核关键路径的锁竞争，优化并发场景下的锁策略，减少关键任务的等锁延时，搭建锁观测器，帮助识别锁滥用导致的性能瓶颈5.IPC优化:深入分析和优化进程间通信机制，减少通信延迟（如 Binder 事务处理耗时），提升跨进程通信效率6.搭建 CPU 性能监控体系，开发自动化工具（结合 perf、ftrace、systrace）追踪调度延迟、调频响应、IPC 耗时、锁竞争等指标，实现问题的量化分析与根因定位7.跨团队协作推动优化落地，与硬件团队、Framework 团队、应用团队协同，解决 “游戏卡顿”“多任务切换慢” 等用户痛点岗位要求1.精通 Linux 内核核心机制，3年以上内核CPU优化经验，掌握子系统深度优化：1）CPU 调度：掌握 EAS/CFS/RT 调度器及负载均衡原理，熟悉walt、pelt负载算法2）调频：精通内核cpufreq/devfreq/cpuidle工作原理，具备调频算法优化的能力4）锁：精通 Linux 内核主流锁机制：futex、mutex、rwsem、spinlock等，负责过内核锁的优化项目，有建立锁性能评估指标的经验5）IPC: 精通 内核 Binder IPC 机制，掌握 Binder 核心组件的工作原理，具备Binder驱动优化能力2.具备扎实的 C 语言编程能力，能独立开发或修改内核调度、调频、IPC 相关模块代码3.掌握 perf、ftrace、perfdog、perfetto 等性能分析工具，能快速定位性能瓶颈4.有芯片平台（高通 / 联发科 ）CPU 性能调优经验者优先5. 具备针对安卓典型场景（游戏满帧、后台待机、视频录制）定制子系统优化策略的经验6.熟悉业界内核常用优化方案：帧调频，锁优先级传递、感知调度、soc一体化调频等方案
