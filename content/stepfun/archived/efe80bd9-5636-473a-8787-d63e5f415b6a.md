**工作职责**

* 负责 **Web 及跨端容器（移动 / 桌面）** 中的前端开发，参与 **AI 大模型应用** 的工程化落地，持续保障代码质量与用户体验
* 与产品、设计、后端紧密协作，将 AI 能力与业务需求转化为稳定、可扩展的前端解决方案
* 围绕 **质量、效率、体验** 的持续提升，进行组件封装、文档沉淀、工程工具建设与系统搭建
* 参与前端架构与工程体系建设，持续优化研发流程、代码规范与开发体验
* 在必要场景下补充基础后端能力，支持业务快速验证与功能落地

**岗位要求**

* 2 年及以上前端开发经验，具备良好的工程意识与代码规范习惯
* 熟练掌握 **JavaScript / HTML / CSS**，理解 Web 标准与浏览器运行机制，熟悉常见布局方案（如 **Flex / Grid**），理解表现层与数据层的合理分离
* 至少熟练使用一款主流前端框架：**React / Vue / Angular** 等，并理解其核心设计思想
* 熟悉 **HTTP / HTTPS** 等网络基础，能够使用常见调试与性能分析工具进行问题定位
* 对前后端协作边界有清晰认知，理解接口设计、数据结构与异常处理对前端体验的影响

**加分项**

* 有 **跨端技术融合实践经验**，包括但不仅限于 **React Native、Hybrid Web、Electron** 等
* 了解并实践过 **大模型相关技术**（如 GPT），对其工程化应用、Agent 方案（如 **LangChain**）有理解或落地经验
* 具备 **全栈经验**：以前端为主，能够使用 **Node.js / Python** 等补充基础后端能力
