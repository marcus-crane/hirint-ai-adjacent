At Two Six Technologies, we build, deploy, and implement innovative products that solve the world’s most complex challenges today. Through unrivaled collaboration and unwavering trust, we push the boundaries of what’s possible to empower our team and support our customers in building a safer global future.

Two Six Technologies, is building our team and looking for an experienced Automation Tester to support a new opportunity. This role will be primarily responsible for developing and maintaining automated end-to-end, integration and performance tests.

**Responsibilities Include:**

* Performs software product testing and investigates possible causes of problems
* Performs manual testing as stop-gap for automated testing
* Develops functional, end-to-end, regression, and performance tests in automated test environments
* Detects and writes software bug reports and feature requests
* Understands, reproduces, troubleshoots product defects and works with developers to determine the best resolution
* Assists in the planning and implementation of testing new features

**What you will need:**

* 2-5 years of professional software testing experience
* Strong proficiency in automated testing frameworks and methodologies
* Proven experience designing and implementing integration test suites
* Experience with REST API testing and validation
* Proficiency with Cypress or similar end-to-end testing frameworks
* Experience with CI/CD pipeline integration for automated tests
* Knowledge of test design patterns and best practices
* Experience with Git version control and collaborative workflows
* Understanding of software development lifecycle and agile methodologies
* Strong analytical and debugging skills
* Excellent written communication for test documentation and defect reporting
* Experience working collaboratively with development teams

**Nice to haves:**

* Strong programming skills in JavaScript/TypeScript
* Experience testing microservice architectures
* Familiarity with additional testing frameworks (Playwright, Selenium, Jest, Mocha)
* Experience with API testing tools (Postman, REST, Assured, Supertest)
* Knowledge of performance and load testing tools (k6, JMeter, Gatling)
* Experience with contract testing (Pact, or similar)
* Familiarity with testing in containerized environments (Docker, Kubernetes)
* Experience with test data management and mocking strategies
* Knowledge of accessibility testing (WCAG compliance, axe-core)
* Experience with visual regression tools
* Familiarity with backend technologies (go, Node.js, Python)
* Understanding of security testing principles
* Experience with test reporting and metrics dashboards
* Knowledge of behavior-driven development (BDD) frameworks (Cucumber, SpecFlow)
* Experience mentoring junior test engineers
* Familiarity with cloud test platforms (AWS/C2S)

**Security Clearance:**

* Active TS/SCI required. Active TS/SCI w. Polygraph preferred.
