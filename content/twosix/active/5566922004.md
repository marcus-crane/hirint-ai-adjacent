Two Six Technologies is currently looking for a Mid-Senior Level Test Engineer to join our team in Chantilly, VA.

**What you will do:**

* Create test cases
* Perform Manual tests
* Creation of automated tests
* Documenting work within task tracking system
* Provide feedback to developers

**What you will need:**

* Experience in manual testing of custom web applications
* Ability and willingness to work as part of a dynamic team that builds applications across multiple networks
* Good communication skills required
* Ability to work within a team of developers to create tests and provide testing feedback is a must

**Nice to Haves:**

* Test automation with Selenium

**Clearance Requirement:**

* Active TS/SCI with Polygraph
