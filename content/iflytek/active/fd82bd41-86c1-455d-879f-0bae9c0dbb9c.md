## 职位描述

1、负责 JAVA 技术方向的产品和项目研发工作；

2、负责系统架构演进的设计和落地，保障系统在千万级用户规模下能够保持性能、稳定性、以及可观测方面的良性运转；

3、参与项目重点、难点的技术攻坚；

4、参与相关系统文档的撰写和维护；

5、制定项目研发计划，把控项目研发进度，控制项目质量、进度。

## 任职要求

1、5年以上java开发工作经验，主导过多个中大型互联网项目的经验，熟悉项目从需求分析到上线的全流程，有成功推动项目落地并取得良好业务成果的案例。

2、具备团队研发负责人经验优先，有人机交互系统研发经验案例优先，有运营商通话产品研发经验优先；

3、熟练掌握 Java 语言，理解面向对象编程思想，具备常见开发框架的熟练应用经验，熟悉容器化实施经验；

4、熟练使用redis、rocketmq、zk等相关中间件并理解其基本原理；

5、具备复杂问题处理的经验，并有一套问题分析排查的方案论；

6、了解AI技术的发展进程，并具备AI/大模型项目开发经验/实践。
