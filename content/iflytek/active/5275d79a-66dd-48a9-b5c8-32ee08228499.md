## 职位描述

对外：伙伴连接与深度管理

- 建立并维护头部生态伙伴的核心干系人网络，从对接人一路摸到决策层，建立自己的关系地图与推进节奏

- 深入理解伙伴的业务目标、产品规划、商业模式、当前痛点、稀缺资源分布，输出并持续更新伙伴的「利益相关方分析」

- 主导与伙伴的战略/产品级沟通，敢于在对齐目标后做出承诺，也敢于对不合理诉求说不

- 借助 PR、联合品牌、行业活动等方式放大合作影响力，持续构建讯飞在伙伴侧的「优先合作」心智

对内：产品策划与融合设计

- 站在用户体验的角度，设计伙伴能力在车载场景下的最佳形态——不是 SDK 接入，而是融合了讯飞语音 / 大模型 / 音效 / Agent 编排的差异化体验

- 输出产品方案与 PRD，与产线、方案、设计、研发团队协同推进方案落地

- 主导「伙伴 × 讯飞」联合方案的打包设计，使其成为面向车企客户可销售、可复制的产品形态

## 任职要求

- 3 年以上产品经理 / 商务拓展 / 生态合作相关经验，至少 2 年在互联网、汽车、智能硬件或 AI 行业深度参与过从 0 到 1 的合作项目

- 具备扎实的产品能力：能独立完成用户洞察 → 方案设计 → PRD → 体验定义的完整闭环，能与研发团队高效对齐

- 具备扎实的商务能力：善于建立关系网、识别干系人诉求、推动跨公司合作达成共识，在面向高层的场合能独立代表讯飞

- 结构化思考能力强，能把复杂模糊的合作机会拆解成清晰的目标、策略、任务、里程碑

- 对 AI、大模型、智能座舱有基本判断力，对行业趋势有自己的观点
