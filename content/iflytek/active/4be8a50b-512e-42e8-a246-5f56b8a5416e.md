## 职位描述

1. 对接各类实际业务，开发AI Agent 系统，并进行持续迭代和优化。

2. 设计与开发能够自主完成复杂任务的 Agent 系统，完成实际业务场景落地，负责 Prompt 编排、Tool 定义与调用、多步骤任务链设计、上下文管理与状态控制等。

3. 根据业务需求构建和优化 RAG 系统，包括数据处理与切分、Embedding 选型、检索优化。

4. 实现 Agent Teams 多智能体协作，包括任务拆解、多 Agent 分工、协调器设计与 Agent 间通信。

5. 建立 Agent 评测体系，通过日志分析和 A/B 测试系统性优化 Agent 效果，降低幻觉率、提升工具调用成功率和复杂任务完成率。

6. 与产品、后端及业务团队紧密协作，能够清晰沟通技术方案与实现边界，推动 Agent 功能从需求到落地的全流程交付

## 任职要求

任职要求

1. 对AI开发工具及框架有深刻的理解，能清晰描述自己的 AI 驱动开发工作流，对 Harness Engineering 有深刻理解。

2. 2 年以上软件开发经验，至少 1 个实际落地的 LLM/Agent 项目经验，熟练掌握 Python，具备良好工程规范意识。

3. 熟悉 RAG 系统构建与调优、Tool Use/Function Calling、Agent Teams 多智能体协作设计。

4. 具备良好的团队协作与沟通能力，能主动推进跨职能协作，适应快速迭代的开发节奏。

5. 对模型微调有经验者优先。

6. 对 AI 技术有强烈好奇心，能快速跟进前沿模型更新并应用到工作中。
