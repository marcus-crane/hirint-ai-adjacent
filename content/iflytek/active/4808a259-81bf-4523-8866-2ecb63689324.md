## 职位描述

我们相信：Model + Harness = Agent

模型决定智能上限，Harness 决定模型能否稳定、可靠地完成真实任务。你将参与下一代 Agent 运行与编排基础设施的研发，包括任务循环、上下文与记忆、工具调用、评测、可观测性和安全控制，帮助模型从“会回答问题”进化为“能够完成复杂任务”。这不是一个只调用模型 API、编写 Prompt 或搭建简单 RAG 应用的岗位。你将有机会深入 Agent 的运行机制，参与解决规划执行、工具使用、上下文管理、错误恢复和结果验证等真实工程问题。如果你具备扎实的工程基础，对 Agent 有强烈兴趣，并通过项目展现了自己的学习能力和动手能力，我们非常欢迎你申请。

1、参与 Agent Runtime / Harness 核心模块研发，包括 Agent Loop、任务规划与执行、工具调度、状态管理、流式输出、错误恢复以及任务暂停与恢复。

2、参与上下文、记忆和工具扩展体系建设，探索上下文压缩、Token 预算、长期记忆、Skills、MCP、权限控制及沙箱执行。

3、参与 Agent 评测与可观测性建设，通过任务成功率、工具调用正确率、执行轨迹、延迟和成本等指标，定位并改善 Agent 失败问题。

4、与模型研发、训练、产品及工程团队协作，基于真实任务、运行轨迹和用户反馈，推动模型能力与 Harness 的共同优化。

5、深度使用团队开发的 Agent 产品，持续跟进前沿技术和行业优秀产品，从开发者与用户的双重视角提出改进方案，并快速完成原型验证。

## 任职要求

1、具备扎实的计算机基础和软件工程能力，能够完成模块设计、编码、测试和问题排查。

2、熟练掌握至少一种编程语言，如 Rust、Python、TypeScript、Go、Java 或 C++。

3、有真实的工程项目经历，课程项目、竞赛项目、个人项目、科研项目或开源项目均可，并能够清楚说明设计思路、技术取舍和问题解决过程。

4、对 LLM 和 Agent 有浓厚兴趣，了解 Agent Loop、Tool Use、Context、Memory、Planning 等基本概念，或具备快速学习相关技术的能力。

5、使用过一种或多种 AI Coding 工具、代码类 Agent 或通用 Agent 产品，能够借助 AI 快速进入陌生技术栈，并通过测试、代码审查和运行验证保证交付质量。

6、具备良好的沟通协作能力、技术好奇心和产品意识，愿意深入分析异常分支、失败场景和边界条件。

加分项

1、开发或深度改造过 Agent、MCP Server、AI Coding 工具、Agent Framework 或多智能体系统。

2、有上下文管理、记忆系统、工具调用、模型评测、数据构造或 KV Cache 相关实践。

3、有异步任务、分布式系统、沙箱执行、权限控制、安全隔离或可观测性相关经验。

4、有高质量个人项目、开源贡献、论文、竞赛成绩或其他能够体现技术能力的代表性成果。

5、能够阅读英文技术资料，并参与开源社区讨论或与海外开发者交流。

6、申请时欢迎提供以下内容任选一至两项即可，不要求全部具备：

-开发或深度参与过的 Agent、MCP、开发者工具或开源项目。

-一次真实的 Agent 失败分析，包括执行过程、根因判断、修复方案和验证结果。

-一个借助 AI 在陌生技术栈中完成高质量交付的案例。

-你对现有 Agent 产品在上下文管理、工具调用、多 Agent 协作或开发者体验方面的具体判断。
