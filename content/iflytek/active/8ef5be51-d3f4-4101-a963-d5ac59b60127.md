## 职位描述

1. 负责 语音 / CV 相关 AI 产品的需求分析、功能规划与版本迭代，输出 PRD 及交互原型

2. 基于对模型能力的深度理解，将算法能力转化为真实可用的产品功能

3. 制定产品评测标准与体验指标，推动模型效果与用户体验的持续提升

4. 深入用户反馈，挖掘语音 / 视觉类场景的核心需求与痛点

5. 参与语音或视觉模块的效果验证，搭建 Demo 或内部工具快速推进方向验证

6. 协调算法、工程、业务多方资源，推动产品按时高质量交付

## 任职要求

1. 本科及以上学历，计算机、人工智能、信号处理等相关专业优先

2. 具备以下任一方向的大模型开发经验（1~3 年）：

语音方向： TTS / 声音克隆 / 歌唱合成 / ASR / 语音大模型

CV 方向： 目标检测 / 图像生成 / 多模态大模型 / AIGC

3. 熟悉开发语言，能独立完成数据处理、模型调用及简单工程开发

4. 对产品工作有强烈热情，逻辑表达能力强，能将复杂技术方案转化为清晰的产品语言
