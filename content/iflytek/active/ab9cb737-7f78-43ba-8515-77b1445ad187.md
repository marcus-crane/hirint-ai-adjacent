## 职位描述

1.负责Android/Linux平台项目的应用研发和设计工作

2.精通linux C/C++，熟悉CMake，熟悉linux编译及调试过程

3.根据客户需求，协助客户完成项目各模块的版本集成，并在项目周期内对客户的常见问题进行沟通与解答；

4.项目里程碑节点需要出差联调保障项目顺利交付；

5.对Linux有深入了解，了解主流linux发行版及其优缺点

## 任职要求

1.大学本科及以上学历，具备 2 年及以上相关开发工作经验

2.精通 C++ 开发，熟练掌握多线程、跨平台开发技术；具备复杂问题分析与解决能力，在线程同步、内存泄漏 / 越界排查、系统性能优化等领域积累了丰富的实践经验与成熟解决方案

3.熟练掌握 Linux 环境下的编译、调试流程，精通 CMake 构建工具；深入理解 Linux 系统原理，熟悉主流 Linux 发行版特性，并能准确分析各版本的优势与适用场景

4.精通 Linux 内核原理及源码逻辑，具备扎实的 Linux 驱动开发与应用层开发经验

5.熟悉嵌入式系统开发流程，精通 ARM、MIPS 或 X86 等主流硬件架构

6.具备扎实的计算机体系结构、数据结构理论基础，熟悉各类总线协议原理及应用

7.具备车载信息娱乐系统应用开发经验者优先；熟悉 AI 技术原理或大模型工具链者优先
