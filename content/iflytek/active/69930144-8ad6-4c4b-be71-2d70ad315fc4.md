## 职位描述

【岗位职责】：

1. 负责机器人 Agent 应用链路设计与开发，打通意图理解、工具调用、任务编排和结果反馈。

2. 负责 Prompt、RAG、工具调用、工作流编排和 API 封装等工程能力落地。

3. 与端侧应用、云服务和交互团队协同，推动机器人交互能力快速产品化。

4. 参与关键能力评测、异常处理和线上问题复盘，持续优化 Agent 可控性和稳定性。

## 任职要求

【任职要求】：

1. 具备 3 年以上 AI 应用工程化经验，有 LLM、RAG 或 Agent 落地案例。

2. 熟悉主流大模型接入、Prompt 设计、工具调用和服务编排方案。

3. 具备较强工程实现能力，能把实验能力推进到稳定可交付的产品链路。

加分项：

1. 熟悉 AI 编程工具和自动化研发流程者优先。

2. 有机器人、智能终端或多模态交互产品经验者优先。
